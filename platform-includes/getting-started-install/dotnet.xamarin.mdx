Install the **NuGet** package:

```shell
# For Xamarin.Forms
Install-Package Sentry.Xamarin.Forms -Version {{@inject packages.version('sentry.dotnet.xamarin-forms') }}

# If you are not using Xamarin.Forms, but only Xamarin:
Install-Package Sentry.Xamarin -Version {{@inject packages.version('sentry.dotnet.xamarin') }}
```
