If you use a third-party SDK to evaluate feature flags, you can enable a Sentry SDK integration to track those evaluations. Integrations are provider specific. Documentation for supported SDKs is listed below.

- <PlatformLink to="/configuration/integrations/launchdarkly/">LaunchDarkly</PlatformLink>
- <PlatformLink to="/configuration/integrations/openfeature/">OpenFeature</PlatformLink> (multiple providers supported)
- <PlatformLink to="/configuration/integrations/statsig/">Statsig</PlatformLink>
- <PlatformLink to="/configuration/integrations/unleash/">Unleash</PlatformLink>

### Generic Integration

The generic `featureFlagsIntegration` allows you to manually track feature flag evaluations. Read the <PlatformLink to="/configuration/integrations/featureflags/">documentation</PlatformLink> to learn more.
