```csharp {tabTitle:C#}
SentrySdk.ConfigureScope(scope =>
{
    scope.User = new SentryUser
    {
        Id = Guid.NewGuid().ToString(),
        IpAddress = "{{auto}}", // Tells Sentry to infer the IP address
        Email = "jane.doe@example.com",
    };
});
```
