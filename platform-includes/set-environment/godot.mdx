```gdscript
extends SentryConfiguration
## Tip: Assign configuration script in the Project Settings.

func _configure(options: SentryOptions):
    options.environment = "production"
```

The SDK automatically detects the following Godot-specific environments:
- [code]headless_server[/code] when running in headless mode
- [code]export_release[/code] for release exports
- [code]export_debug[/code] for debug exports
- [code]editor_dev[/code] when running inside the Godot editor
- [code]editor_dev_run[/code] when running a project from the Godot editor
