<SplitLayout>
<SplitSection>
<SplitSectionText>

Opt in by adding `spanStreamingIntegration()` to your list of integrations when initializing the SDK in your client hook file:

</SplitSectionText>
<SplitSectionCode>

```javascript {filename: src/hooks.client.(js|ts)}
import * as Sentry from "@sentry/sveltekit";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  tracesSampleRate: 1.0,
  integrations: [
    // enables stream mode
    Sentry.spanStreamingIntegration(),
  ],
});
```

</SplitSectionCode>
</SplitSection>

<SplitSection>
<SplitSectionText>

Then set the `traceLifecycle` option to `'stream'` when initializing the SDK in your server instrumentation file:

<Alert>Requires SvelteKit `2.31.0` or higher.</Alert>

</SplitSectionText>
<SplitSectionCode>

```javascript {filename: src/instrumentation.server.(js|ts)}
import * as Sentry from "@sentry/sveltekit";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  tracesSampleRate: 1.0,
  // enables stream mode
  traceLifecycle: "stream",
});
```

</SplitSectionCode>
</SplitSection>
</SplitLayout>

To revert to transaction mode, remove `traceLifecycle` and `spanStreamingIntegration()` from these files.
