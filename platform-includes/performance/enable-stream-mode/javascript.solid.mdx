<SplitSectionText>

Opt in by adding `spanStreamingIntegration()` to your list of integrations when initializing the SDK:

</SplitSectionText>
<SplitSectionCode>

```javascript {filename: index.jsx}
import * as Sentry from "@sentry/solid";
import { solidRouterBrowserTracingIntegration } from "@sentry/solid/solidrouter";
import { render } from "solid-js/web";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  tracesSampleRate: 1.0,
  integrations: [
    solidRouterBrowserTracingIntegration(),
    // enables stream mode
    Sentry.spanStreamingIntegration(),
  ],
});
```

</SplitSectionCode>
</SplitSection>
</SplitLayout>

To revert to transaction mode, remove `spanStreamingIntegration()` from your integrations.
