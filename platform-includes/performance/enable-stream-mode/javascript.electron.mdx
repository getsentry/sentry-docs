<SplitSection>
<SplitSectionText>

Opt in by adding `spanStreamingIntegration()` to your list of integrations when initializing the SDK in your renderer:

</SplitSectionText>
<SplitSectionCode>

```javascript
import * as Sentry from "@sentry/electron/renderer";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  tracesSampleRate: 1.0,
  integrations: [
    Sentry.browserTracingIntegration(),
    // enables stream mode
    Sentry.spanStreamingIntegration(),
  ],
});
```

</SplitSectionCode>
</SplitSection>

<SplitSection>
<SplitSectionText>

Then set the `traceLifecycle` option to `'stream'` when initializing the SDK in your main process:

</SplitSectionText>
<SplitSectionCode>

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  tracesSampleRate: 1.0,
  // enables stream mode
  traceLifecycle: "stream",
  integrations: [Sentry.startupTracingIntegration()],
});
```

</SplitSectionCode>
</SplitSection>
</SplitLayout>

To revert to transaction mode, remove `traceLifecycle` and `spanStreamingIntegration()` from these files.
