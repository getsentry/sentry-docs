`routingInstrumentation` is avaliable for the Angular Router. For more details, see the [Angular router docs](/platforms/javascript/guides/angular/#configure).

```JavaScript
Sentry.init({
  dsn: '___PUBLIC_DSN___',
  tracesSampleRate: 1.0,
  integrations: [
    new Sentry.BrowserTracing({
      routingInstrumentation: Sentry.routingInstrumentation,
    }),
  ],
});
```
