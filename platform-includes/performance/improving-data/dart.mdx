## Improving Data on Transactions and Spans

### Adding Data Attributes to Transactions

You can add data attributes to your transactions. This data is visible in the trace explorer in Sentry. Data attributes can be of type `String`, `int`, `double` or `bool`, as well as (non-mixed) arrays of these types:

```dart
final transaction = Sentry.startTransaction('my-transaction', 'http.server');
transaction.setData('data_attribute_1', 42);
transaction.setData('data_attribute_2', true);
transaction.setData('data_attribute_3', ['value1', 'value2']);
transaction.setData('data_attribute_4', [true, false]);
transaction.setData('data_attribute_4', [42, 43]);
```

### Adding Data Attributes to Spans

You can add data attributes to your spans. This data is visible in the trace explorer in Sentry. Data attributes can be of type `String`, `int`, `double` or `bool`, as well as (non-mixed) arrays of these types:

```dart
final span = transaction.startChild('http.client');
span.setData('data_attribute_1', 42);
span.setData('data_attribute_2', true);
span.setData('data_attribute_3', ['value1', 'value2']);
span.setData('data_attribute_4', [true, false]);
span.setData('data_attribute_4', [42, 43]);
```
