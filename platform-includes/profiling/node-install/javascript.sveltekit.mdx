<Note>

Node profiling is available starting in `@sentry/profiling-node` version `0.3.0`.
You have to have the `@sentry/sveltekit` package installed.

</Note>

```bash {tabTitle:npm}
npm install @sentry/sveltekit @sentry/profiling-node --save
```

```bash {tabTitle:yarn}
yarn add @sentry/sveltekit @sentry/profiling-node
```

```bash {tabTitle:pnpm}
pnpm add @sentry/sveltekit @sentry/profiling-node
```
