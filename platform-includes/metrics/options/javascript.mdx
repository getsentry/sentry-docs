### beforeSendMetric

Filter or modify metrics before sending. Return `null` to drop a metric.

</SplitSectionText>
<SplitSectionCode>

```javascript
Sentry.init({
  dsn: "___PUBLIC_DSN___",
  beforeSendMetric: (metric) => {
    // Drop specific metrics
    if (metric.name === "debug_metric") {
      return null;
    }

    // Add attributes
    metric.attributes = {
      ...metric.attributes,
      processed: true,
    };

    return metric;
  },
});
```

</SplitSectionCode>

### Disable Metrics

Set `enableMetrics: false` to disable metrics collection entirely.

### Flush Metrics

Metrics are buffered and sent periodically. Use this snippet to flush immediately:

</SplitSectionText>
<SplitSectionCode>

```javascript
// Disable metrics
Sentry.init({
  dsn: "___PUBLIC_DSN___",
  enableMetrics: false,
});

// Flush all pending metrics
await Sentry.flush();
```
