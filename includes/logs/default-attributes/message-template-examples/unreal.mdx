For example, with the following log:

```cpp
SentrySubsystem->AddLog(FString::Printf(TEXT("Player %s has %d health"), *PlayerName, PlayerHealth), ESentryLevel::Info, TEXT("GameFlow"));
```

Sentry will add the following attributes:

- `message.template`: "Player %s has %d health"
- `message.parameter.0`: PlayerName value
- `message.parameter.1`: PlayerHealth value
