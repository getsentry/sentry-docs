To filter out all SDK-generated logs and only keep your application logs, use the following snippet:

```js
Sentry.init({
  dsn: "___PUBLIC_DSN___",
  enableLogs: true,
  beforeSendLog: (log) => {
    if (log.attributes?.["sentry.origin"] !== undefined) {
      return null;
    }
    return log;
  },
});
```
