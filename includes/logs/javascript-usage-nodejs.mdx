You can also avoid using the `fmt` function by formatting the log message via a `printf`-like format string and arguments.

This follows the same structure as the [`util.format` Node.js standard library function](https://nodejs.org/api/util.html#utilformatformat-args).

```js
Sentry.logger.info("User %s logged in successfully", ["John Doe"]);
Sentry.logger.warn("Failed to load user %s data", ["John Doe"], {
  errorCode: 404,
});
Sentry.logger.error(
  "Failed to process payment. Order: %s. Amount: %d. Details: %o",
  ["order_123", 99.99, paymentDetailsObject]
);
```
