  You can now use Jira Server and GitHub integrations to automatically create issues for your Sentry Alerts. You can simply add an action to your alert rules to create these issues.

Read our docs for [Jira Server](https://docs.sentry.io/product/integrations/issue-tracking/jira/#issue-management) and [GitHub](https://docs.sentry.io/product/integrations/source-code-mgmt/github/#issue-management) for more information
