  We published [the core repositories that drive Codecov](https://github.com/codecov). These repositories are available under the same [BUSL license](https://github.com/codecov/self-hosted/blob/main/LICENSE) as Sentry’s core repos. The code is available now for you to run for yourself, study, modify, and redistribute in original and modified form.\
\
[Learn more](https://about.codecov.io/blog/codecov-is-now-open-source/).
