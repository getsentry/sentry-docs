  The Sentry SDK team is happy to announce the immediate availability of Sentry Laravel SDK v3.7.0.

### Features TL;DR

* Tracing without Performance

  The SDK will now continue a trace from incoming HTTP requests, even if performance is not enabled.

* GraphQL and HTTP client improvements

* Less sensitive data send by default

To read more about these changes and others, see our full [changelog](https://github.com/getsentry/sentry-laravel/releases/tag/3.7.0).
