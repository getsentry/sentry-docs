  Sentry will comment on recent pull requests suspected of causing issues. This will help developers track the impact of their code changes. You need to have the GitHub integration and [Suspect Commits ](https://docs.sentry.io/product/issues/suspect-commits/)set up to use this feature.

Join our [GitHub discussion](https://github.com/getsentry/sentry/discussions/49996) for more details
