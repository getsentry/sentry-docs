  View request/response bodies and API headers, for the URLs you care about in [Session Replay](https://sentry.io/for/session-replay/). Identify the root cause of arcane errors, while mirroring the browser dev tools experience.\
\
In order to prevent sensitive information from leaving the browser, we've made this feature opt-in. Explicitly choose which URLs you'd like additional data on. Learn more about how to set up this feature in our [docs](https://docs.sentry.io/platforms/javascript/session-replay/configuration/#network-details).

![](https://ik.imagekit.io/launchnotes/production/39w21zrnri33ysznp8tzt0pgr25e#t=0.1)

These are the prerequisites to use the feature:\
1\. Your org needs to have[ sent replay events](https://docs.sentry.io/platforms/javascript/session-replay/)\
2\. JavaScript SDK (or related framework) version of [7.50.0](https://github.com/getsentry/sentry-javascript/releases/tag/7.50.0) or higher
