  Our Java SDK now supports Sentry Cron Monitoring. You can now start monitoring your recurring jobs in minutes through the SDK. Update your SDK to [v6.30.0](https://github.com/getsentry/sentry-java/releases/tag/6.30.0), and read our [setup guide](https://docs.sentry.io/platforms/java/crons/) to get started.
