  First class support of GraphQL client side errors for Mobile has been shipped in beta!

See our [Android SDK docs](https://docs.sentry.io/platforms/android/configuration/integrations/apollo3/#graphql-client-errors) for more info on how to get it working with our Apollo3 integration.

Server side support in [Java](https://github.com/getsentry/sentry-java/issues/2790), [PHP](https://github.com/getsentry/sentry-laravel/pull/720), and [Python](https://github.com/getsentry/sentry-python/issues/2198) all coming soon!
