## Install and Configure

<Note>

Sentry owner, manager, or admin permissions are required to install this integration.

</Note>

1. Navigate to **Settings > Integrations > SIGNL4**

2. Follow the full [SIGNL4 installation instructions](https://www.signl4.com/blog/portfolio_item/sentry/).
