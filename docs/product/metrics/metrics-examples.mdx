If you're not sure where to start, here are a few examples of how you can use custom metrics.

## Frontend Examples
| **Use Case**           | **Metric Type** | **Example** |
| ----------------------------- | --------------------- |
| United States of America (US) | Iowa, USA             |
| European Union (EU)           | Frankfurt, Germany    |

## Backend Examples

| **Use Case**                                | **Metric Type** | Example |
| ------------------------------------------- | --------------- |
| Count of API calls to one of your endpoints | counter         | sentry_sdk.metrics.incr(key=”endpoint_call_counter”, value=1} |
| Count of API calls of a 3rd party API       | counter         | sentry_sdk.metrics.incr(key=”external_api_calls”, value=1, tags={””: “”, “”: ””} |
