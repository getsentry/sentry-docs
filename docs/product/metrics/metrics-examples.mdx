If you're not sure where to start, here are a few examples of how you can use custom metrics.

## Frontend Examples
| **Use Case**                                 | **Metric Type**    | **Example**                                                   |
| -------------------------------------------- | ------------------ | ------------------------------------------------------------- |
| Count button clicks on subpages              | counter            | Sentry.metrics.increment("button_click", 1, {  tags: { browser: "Firefox", app_version: "1.0.0", subpage: “signup”}}); |
| Number of user signups                       | counter            | Sentry.metrics.increment(”signups”, 1, tags: {plan_type: “freemium”} |
| Measure UX improvement to signup form        | distribution       | Sentry.metrics.distribution("login.time_to_finish_form", 14.3 {tags: { subpage: “signup”}, unit: “second”}); |
| Track the loading times for a component      | distribution       | Sentry.metrics.distribution("component_load_time", 15.0, {tags: { type: "important" },unit: "millisecond",}); |
| Track API request payloads                   | distribution       | Sentry.metrics.distribution("api_request_payload", 136.3, {  tags: { api_name: “maps_provider”, subpage: “signup”}, unit: “kilobyte”}); |
| Count concurrent users in application        | gauge              | Sentry.metrics.gauge("concurrent_users", 10, {  tags: { browser: "Firefox", app_version: "1.0.0"}}); |
| Track the number of items in a shopping cart | gauge              | Sentry.metrics.gauge("shopping_cart_item_count", itemCount, {tags: { platform: "web" }  |
| Count the unique number of users on a page   | set                | Sentry.metrics.set("user_view", "jane" tags={"page": "/home"}); |
| Count unique number of user sessions         | set                |  Sentry.metrics.set("unique_active_sessions", sessionId, {tags: { platform: "mobile"}}); |


## Backend Examples

| **Use Case**                                       | **Metric Type**    | **Example**                                                   |
| -------------------------------------------------- | -------------------| ------------------------------------------------------------- |
| Count of API calls of a 3rd party API              | counter            | sentry_sdk.metrics.incr(key="api_calls", value=1, tags={"api": "third_party"}) |
| Count of jobs executed per worker                  | counter            | sentry_sdk.metrics.incr(key=”external_api_calls”, value=1,<br> tags={”worker_id”: “485ea324-003d-4f30-a0f9-beed9792c4a1”, “worker_type”: ”standard”}) |
| Processing time for a task	                       | distribution       | sentry_sdk.metrics.distribution(key=”processing_time”, value=0.002, unit=”second”, tags={”task”: “example_task”}) |
| Track ML model confidence during inference         | distribution       | sentry_sdk.metrics.distribution(key=”model_confidence”, value=0.72, unit=”ratio”, tags={”model_id”: “a294c108”}) |
| Track CPU usage over time                          | gauge              | sentry_sdk.metrics.gauge(key=”cpu_usage”, value=94, unit=”percent”) |
| Track cache hit ratio                              | gauge              | sentry_sdk.metrics.gauge(key=”cache_hit_ratio”, value=85, unit=”percent”) |
| Count unique server ids running a software version | set                | sentry_sdk.metrics.set(key=”server_ids_running”, value=“server_id”, tags={"software_version": software_version}) |
| Track number of unique DB queries executed         | set                | sentry_sdk.metrics.set(key=”db_queries”, value=query, tags={"query_type": "read"}) |
