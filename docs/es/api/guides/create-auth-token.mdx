Para usar las API de Sentry, necesitas un token de autenticación. Este tutorial te guía para crear un token de autenticación de la organización mediante una integración interna. Sentry recomienda usar tokens de autenticación de organización siempre que sea posible, ya que no están vinculados a cuentas de usuario específicas.

Consulta nuestra documentación sobre [autenticación](/es/api/auth/) para obtener más información sobre los diferentes tipos de tokens de autenticación disponibles.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una cuenta de Sentry con un rol a nivel de organización de Manager o Admin.

<div id="create-an-internal-integration">
  ## Crear una integración interna
</div>

Las [integraciones internas](/es/organization/integrations/integration-platform/internal-integration/) se usan para crear integraciones personalizadas de Sentry para tu organización. También se pueden usar para crear y gestionar los tokens de tu organización.

1. Abre [sentry.io](https://sentry.io/)

2. Haz clic en &quot;Settings&quot; en el menú izquierdo para abrir la página de **Organization Settings**.

3. Haz clic en &quot;Custom Integrations&quot; en el panel lateral izquierdo para crear una nueva integración interna y un token de autenticación a nivel de organización.

4. Pulsa el Button &quot;Create New Integration&quot;.

5. Asegúrate de que &quot;Internal Integration&quot; esté seleccionado en el modal y pulsa &quot;Next&quot;.

6. Escribe un nombre para tu integración.

![Recorrido de UX de Sentry.io para crear una integración interna.](./img/create-internal-integration.gif)

<div id="create-a-api-authentication-token">
  ## Crear un token de autenticación de API
</div>

1. En &quot;Permissions&quot;, selecciona los permisos necesarios para las API que deseas usar.

   Cada página de documentación de un endpoint de API enumera los permisos requeridos en la sección &quot;Scopes&quot;. Por ejemplo, el endpoint [Create a New Project](/es/api/projects/create-a-new-project/) requiere permisos de project:write o superiores.

2. Haz clic en &quot;Save Changes&quot;.

3. Desplázate hasta el final de la página y copia el token generado en &quot;Tokens&quot;.

![Recorrido de la UX de Sentry.io para crear un token y configurar sus permisos.](./img/set-token-permissions.gif)

Mantén tu token de autenticación a mano en el portapapeles o en una variable de entorno para usarlo en llamadas a la API.
