La API web de Sentry se usa para acceder a la plataforma de Sentry de forma programática. Puedes usarla para gestionar recursos a nivel de cuenta, como organizaciones y equipos, así como para administrar y exportar datos.

Si buscas información sobre la superficie de la API de los SDK de Sentry, consulta la documentación de [SDK Development](https://develop.sentry.dev/sdk/overview/).

<div id="versioning">
  ## Versionado
</div>

La versión actual de la API web de Sentry se considera **v0**. Nuestros endpoints públicos suelen ser estables, pero los endpoints en beta están sujetos a cambios.

<div id="choosing-the-right-api-base-domain">
  ## Elegir el dominio base de la API adecuado
</div>

Aunque muchas de nuestras API usan `sentry.io` como host para los endpoints de la API, si quieres indicar una [ubicación de almacenamiento de datos](/es/organization/data-storage-location/#what-types-of-data-are-stored-where) específica, debes usar dominios específicos por región.

* La región de EE. UU. se hospeda en `us.sentry.io`
* La región de DE se hospeda en `de.sentry.io`.

Para saber qué recursos de la API están disponibles en dominios por región, consulta [qué tipos de datos se almacenan y dónde](/es/organization/data-storage-location/#what-types-of-data-are-stored-where) para más información.
