Los widgets de Insights ofrecen visualizaciones que puedes ajustar para que se adapten mejor a tu flujo de trabajo. Proporcionan vistas específicas de rendimiento que te permiten ver y actuar sobre datos de transacciones en toda tu organización.
Según el [Dominio de Insights](http://localhost:3000/product/insights/) que hayas seleccionado, habrá disponibles distintos widgets de rendimiento preseleccionados.

<Alert>
  El valor que aparece en la esquina superior derecha de algunos widgets es el **valor global para el período de tiempo seleccionado**, no el promedio de los datos del gráfico. En conjuntos de datos con bajo volumen de eventos, el valor global puede ser mayor o menor de lo esperado porque métricas como [Apdex](/es/product/insights/overview/metrics/#apdex) y [User Misery](/es/product/insights/overview/metrics/#user-misery) no son lineales.
</Alert>

<div id="average-ttids">
  ## TTIDs promedio
</div>

Muestra la duración promedio del TTID para las pantallas principales de proyectos móviles. Haz clic en el nombre de la pantalla para ver más información sobre ella en la página [Screen Loads](/es/product/insights/mobile/mobile-vitals/screen-loads/).

Este widget está disponible en [Mobile Overview](/es/product/insights/mobile/).

![Widgets de rendimiento - TTIDs promedio](./img/widgets_average_ttids.png)

<div id="most-slow-frames">
  ## Más fotogramas lentos
</div>

Muestra las transacciones con más fotogramas lentos en proyectos móviles. Más información en [Mobile Vitals](/es/product/insights/mobile/mobile-vitals).

Este widget está disponible en [Mobile Overview](/es/product/insights/mobile/).

![Performance Widgets - Más fotogramas lentos](./img/widgets_most_slow_frames.png)

<div id="most-frozen-frames">
  ## Más fotogramas congelados
</div>

Muestra las transacciones con más fotogramas congelados en proyectos móviles. Más información en [Mobile Vitals](/es/product/insights/mobile/mobile-vitals).

Este widget está disponible en la pestaña [Mobile Overview](/es/product/insights/mobile).

![Performance Widgets - Most Frozen Frames](./img/widgets_most_frozen_frames.png)

<div id="most-time-consuming-queries">
  ## Consultas que consumen más tiempo
</div>

Muestra las consultas a la base de datos que más tiempo tardan en tu aplicación. Estas consultas suelen ser lentas, muy frecuentes o ambas cosas. Haz clic en una consulta para abrir su [**página de resumen de la consulta**](/es/product/insights/queries#query-summary-page) o haz clic en “Ver todas” para abrir la [**página de consultas**](/es/product/insights/queries#queries-page).

Este widget está disponible en la [vista general de Backend](/es/product/insights/backend/).

![Widgets de rendimiento: Consultas con más tiempo invertido](./img/widgets_most_time_spent_queries.png)

<div id="most-time-consuming-assets">
  ## Recursos que consumen más tiempo
</div>

Muestra los recursos que más tiempo requieren en tu aplicación. Suelen ser lentos, muy frecuentes o ambas cosas. Haz clic en un recurso para abrir su [**página de resumen de recurso**](/es/product/insights/frontend/assets#asset-summary-page) o haz clic en &quot;Ver todo&quot; para abrir la [**página de recursos**](/es/product/insights/frontend/assets#assets-page).

Este widget está disponible en la [vista general de Frontend](/es/product/insights/frontend/).

![Widgets de rendimiento: recursos con mayor tiempo invertido](./img/widgets_most_time_spent_assets.png)

<div id="most-time-consuming-domains">
  ## Dominios que más tiempo consumen
</div>

Muestra los dominios en los que la aplicación pasa más tiempo esperando, ordenados por tiempo total por dominio. La aplicación realiza muchas llamadas a estos dominios o estos tardan mucho en responder a las solicitudes. Haz clic en un dominio para abrir su página de [**Resumen del dominio**](/es/product/insights/requests#domain-summary-page) o en “Ver todos” para abrir la página de [**Solicitudes**](/es/product/insights/requests#requests-page).

Este widget está disponible en las pestañas [Resumen de backend](/es/product/insights/backend/), [Resumen de frontend](/es/product/insights/frontend/) y [Resumen de móvil](/es/product/insights/mobile/).

![Widgets de rendimiento - Dominios que más tiempo consumen](./img/widgets_most_time_consuming_domains.png)

<div id="best-page-opportunities">
  ## Mejores oportunidades de página
</div>

Muestra las páginas web de tu aplicación con mayor [oportunidad](/es/product/insights/web-vitals/#opportunity) de mejora. Estas páginas suelen tener puntuaciones bajas en [Web Vitals](/es/product/insights/web-vitals/), un alto volumen de tráfico o ambos. Haz clic en una página para abrir su [**Resumen de página**](/es/product/insights/web-vitals/#page-overview) o haz clic en «Ver todo» para abrir la página de [**Web Vitals**](/es/product/insights/web-vitals/#web-vitals-page).

Este widget está disponible en el [Resumen de frontend](/es/product/insights/frontend/).

![Performance Widgets - Mejores oportunidades de página](./img/widgets_best_page_opportunities.png)

<div id="slow-http-ops">
  ## Operaciones HTTP lentas
</div>

Muestra transacciones con spans HTTP cuyo tiempo acumulado es lento. Las operaciones HTTP lentas no se muestran para todos los proyectos ni todas las transacciones, ya que estas deben contener spans HTTP. Consulta la documentación del SDK del proyecto correspondiente para obtener más información.

Este widget está disponible en las pestañas [Frontend Overview](/es/product/insights/frontend/) y [Backend Overview](/es/product/insights/backend/).

![Performance Widgets - Slow HTTP Ops](./img/widgets_slow_http_ops.png)

<div id="slow-db-ops">
  ## Operaciones de BD lentas
</div>

Muestra transacciones con spans de BD de duración acumulada lenta. Las operaciones de BD lentas no se muestran para todos los proyectos ni transacciones, ya que las transacciones deben contener spans de BD. Consulta la documentación del SDK del proyecto en cuestión para obtener más información.

Este widget está disponible en la [Descripción general de Backend](/es/product/insights/backend/).

![Widgets de rendimiento: Operaciones de BD lentas](./img/widgets_slow_db_ops.png)

<div id="slow-resource-ops">
  ## Operaciones de recursos lentas
</div>

Muestra transacciones con spans de recursos acumulados lentos. Presenta transacciones del frontend con spans de recursos de bajo rendimiento y el tiempo total dedicado a cargar scripts, CSS y más.

Este widget está disponible en la [Vista general de frontend](/es/product/insights/frontend/).

![Performance Widgets - Slow Resource Ops](./img/widgets_slow_resource_ops.png)
