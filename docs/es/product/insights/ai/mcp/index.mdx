Las herramientas de monitoreo de MCP (Model Context Protocol) de Sentry te ayudan a entender qué ocurre en tus implementaciones de MCP. Recopilan automáticamente información sobre conexiones con servidores MCP, acceso a recursos, ejecuciones de herramientas y errores en toda tu pipeline de MCP, desde las solicitudes del cliente hasta las respuestas del servidor.

<div id="example-mcp-monitoring-use-cases">
  ## Ejemplos de casos de uso de monitoreo de MCP
</div>

- Tu servidor MCP no responde a las llamadas de herramientas y quieres rastrear el flujo completo de la solicitud para identificar dónde se interrumpe la conexión.
- Los clientes informan que tus recursos de MCP están devolviendo datos obsoletos o con formato incorrecto, y necesitas depurar el contexto completo de las solicitudes de recursos y las respuestas del servidor.
- Tus implementaciones de MCP están experimentando problemas de rendimiento y quieres identificar qué componentes (inicio del servidor, obtención de recursos o ejecución de herramientas) están generando cuellos de botella.
