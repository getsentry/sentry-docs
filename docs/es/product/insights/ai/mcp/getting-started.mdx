Sentry MCP Observability te ayuda a rastrear y depurar implementaciones del Model Context Protocol (MCP) mediante nuestros SDK e integraciones compatibles. Supervisa tus flujos de trabajo MCP de principio a fin, desde las conexiones del cliente hasta las respuestas del servidor, incluidas las ejecuciones de herramientas, el acceso a recursos y las comunicaciones del protocolo.

Para empezar a enviar datos de MCP a Sentry, asegúrate de haber creado un proyecto de Sentry para tu repositorio con MCP habilitado y sigue la guía a continuación:

<div id="supported-sdks">
  ## SDKs compatibles
</div>

<div id="javascript-mcp-server">
  ### JavaScript - Servidor MCP
</div>

<VersionRequirement
  product="MCP Observability"
  sdk="Node SDK"
  minVersion="9.46.0"
/>

El SDK de JavaScript de Sentry admite la observabilidad de MCP al envolver el servidor MCP del paquete [@modelcontextprotocol/sdk](https://www.npmjs.com/package/@modelcontextprotocol/sdk). Este contenedor captura automáticamente spans de los flujos de trabajo de tu servidor MCP, incluidas las ejecuciones de herramientas, el acceso a recursos y las conexiones de clientes.

<div id="quick-start-with-mcp-server">
  #### Guía rápida de inicio con el servidor MCP
</div>

```javascript
import * as Sentry from "@sentry/node";
import { McpServer } from "@modelcontextprotocol/sdk";

// La inicialización de Sentry debe ir antes que todo lo demás
Sentry.init({
  dsn: "___PUBLIC_DSN___",
  tracesSampleRate: 1.0,
  // Opcional: Habilítalo para capturar en Sentry los argumentos y resultados de las llamadas a herramientas, que podrían incluir PII
  sendDefaultPii: true,
});

// Tu servidor MCP
const server = Sentry.wrapMcpServerWithSentry(new McpServer({
    name: "my-mcp-server",
    version: "1.0.0",
}));

...
```


<div id="python-mcp-server">
  ### Python - Servidor MCP
</div>

<VersionRequirement
  product="MCP Observability"
  sdk="Python SDK"
  minVersion="2.43.0"
/>

El SDK de Python de Sentry es compatible con la observabilidad de MCP para [FastMCP](https://gofastmcp.com/getting-started/welcome). La integración captura automáticamente spans de los flujos de trabajo de tu servidor MCP, incluidas las ejecuciones de herramientas, el acceso a recursos y el manejo de prompts.

<div id="quick-start-with-fastmcp">
  #### Guía rápida de FastMCP
</div>

```python
import sentry_sdk
from mcp.server.fastmcp import FastMCP

# La inicialización de Sentry debe ir antes que todo lo demás
sentry_sdk.init(
    dsn="___PUBLIC_DSN___",
    traces_sample_rate=1.0,
    # Opcional: Habilítalo para capturar en Sentry los argumentos y resultados de las llamadas a herramientas, lo que puede incluir PII
    send_default_pii=True,
)

# Crea el servidor MCP
mcp = FastMCP("Example MCP Server")

# Define una herramienta
@mcp.tool()
async def calculate_sum(a: int, b: int) -> int:
    """Suma dos números."""
    return a + b

# Ejecuta el servidor
mcp.run()
```
