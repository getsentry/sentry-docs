Una vez que hayas [configurado el SDK de Sentry](/es/product/insights/ai/mcp/getting-started/) para tu proyecto MCP, empezarás a recibir datos en el panel [MCP Insights](https://sentry.io/orgredirect/organizations/:orgslug/insights/ai/mcp/) de Sentry.

El panel principal ofrece una vista completa de toda la actividad de tu servidor MCP, métricas de rendimiento y ejecuciones recientes de herramientas.

![Resumen de monitoreo de MCP](./img/overview.png)

El panel muestra widgets clave como:

* **Tráfico**: Muestra las solicitudes de MCP a lo largo del tiempo, las tasas de error y las versiones, para seguir la actividad y el estado general del servidor
* **Tráfico por cliente**: Indica qué clientes MCP se conectan a tu servidor (cursor-vscode, CustomMCPClient, etc.)
* **Distribución de transporte**: Muestra la distribución de los protocolos de transporte utilizados (http, sse, transportes personalizados)
* **Herramientas/Recursos/Prompts más usados**: Muestra qué herramientas/recursos/prompts de MCP son invocados con mayor frecuencia por los clientes
* **Herramientas/Recursos/Prompts más lentos**: Identifica herramientas/recursos/prompts con los tiempos de respuesta más altos para optimizar el rendimiento
* **Herramientas/Recursos/Prompts con más fallos**: Destaca las herramientas/recursos/prompts con las tasas de error más altas que requieren atención

Debajo de estos widgets hay tablas que te permiten ver los datos con más detalle:

* **Herramientas**: Métricas de rendimiento para cada herramienta MCP, incluyendo el número de solicitudes, la tasa de error, la duración promedio y la latencia P95
* **Recursos**: Patrones de acceso y rendimiento para los recursos MCP por URI, mostrando solicitudes, tasa de error, duración promedio y latencia P95
* **Prompts**: Estadísticas de uso para las plantillas de prompts de MCP por nombre, incluyendo solicitudes, tasa de error, duración promedio y latencia P95
