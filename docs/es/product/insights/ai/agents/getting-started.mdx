Sentry AI Agent Monitoring te ayuda a realizar el seguimiento y depurar aplicaciones de agentes de IA usando nuestros SDK e integraciones compatibles. Supervisa los flujos de trabajo completos de tus agentes, desde la interacción del usuario hasta la respuesta final, incluidas las llamadas a herramientas, las interacciones con modelos y la lógica personalizada.

Para empezar a enviar datos de agentes de IA a Sentry, asegúrate de haber creado un proyecto de Sentry para tu repositorio con funciones de IA y sigue una de las guías a continuación:

<div id="quick-start-with-vercel-ai-sdk">
  #### Guía rápida con Vercel AI SDK
</div>

```javascript
import * as Sentry from "@sentry/node";

// La inicialización de Sentry debe ir antes que todo lo demás
Sentry.init({
  tracesSampleRate: 1.0,
  integrations: [Sentry.vercelAIIntegration()],
});

import { generateText } from "ai";
import { openai } from "@ai-sdk/openai";

// Tu función de agente de IA
async function aiAgent(userQuery) {
  const result = await generateText({
    model: openai("gpt-4o"),
    prompt: userQuery,
    experimental_telemetry: {
      isEnabled: true,
      functionId: "ai-agent-main",
      recordInputs: true,
      recordOutputs: true,
    },
  });

  return result.text;
}
```

  Iremos incorporando integraciones de agentes de IA de forma continua. Vota en [GitHub](https://github.com/getsentry/sentry-javascript/issues/16960) cuál te gustaría ver después. </Alert>

<div id="quick-start-with-openai-agents">
  #### Guía rápida de OpenAI Agents
</div>

```python
import sentry_sdk
import agents
from pydantic import BaseModel

sentry_sdk.init(
    dsn="YOUR_DSN",
    traces_sample_rate=1.0,
    send_default_pii=True,  # Incluye entradas/salidas del LLM
)

# Crea tu agente de IA
my_agent = agents.Agent(
    name="Mi Agente",
    instructions="Eres un asistente útil.",
    model="gpt-4o-mini",
)

# Tu función del agente de IA
result = await agents.Runner.run(
    my_agent,
    input=user_query,
)

```

  También puedes instrumentar agentes de IA manualmente siguiendo nuestras [guías de instrumentación manual](/es/platforms/python/tracing/instrumentation/custom-instrumentation/ai-agents-module).
