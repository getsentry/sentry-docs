AI Performance de Sentry te permite ver métricas clave de tus flujos de LLM, como el uso de tokens y la duración. Además de contar con un espacio dedicado para monitorear el rendimiento de IA, también puedes consultar los Insights de Frontend, Backend y Mobile de Sentry para comprender el rendimiento de las aplicaciones que ejecutan tus flujos.
