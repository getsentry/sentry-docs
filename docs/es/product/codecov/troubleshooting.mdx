**¿Por qué mis stack traces no tienen datos de cobertura de pruebas?**

- Es posible que no tengas un informe de cobertura para el archivo que te interesa. Consulta [Codecov](https://about.codecov.io/) para asegurarte de que exista un informe de cobertura.

- Es posible que no tengas habilitada la opción [Source Code Coverage](https://about.codecov.io/product/feature/source-code-coverage/).
