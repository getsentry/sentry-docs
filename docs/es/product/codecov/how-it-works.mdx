Sentry usa asignaciones de código para determinar el repositorio y la rama predeterminada del frame de la traza de pila y consultar el endpoint del informe de cobertura de Codecov. El endpoint le proporciona a Sentry el estado de cobertura de cada línea para que pueda aplicarse a la traza de pila en la interfaz. Esto permite que los usuarios vean el código no probado que está causando errores, incluido si está parcial o totalmente cubierto, directamente en la traza de pila. Esto te ayuda a evitar errores similares en el futuro.
