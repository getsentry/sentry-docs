Para configurar Sentry Crons, usa los enlaces a continuación para los SDK compatibles o la CLI de Sentry. Una vez configurado, podrás recibir alertas si tu tarea no se ejecuta a la hora programada o no se completa correctamente.

* [Node](/es/platforms/javascript/guides/node/crons/)

* [Go](/es/platforms/go/crons/)
