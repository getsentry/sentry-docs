<div id="installation">
  ## Instalación
</div>

<Alert>
  Aunque los fragmentos a continuación pueden usarse para descargar y utilizar la CLI directamente, se recomienda encarecidamente verificar la firma y el SHASUM para garantizar la integridad de la CLI antes de usarla. Consulta [Comprobación de integridad del Uploader](/es/product/test-analytics/sentry-prevent-cli/#integrity-checking-the-cli) a continuación para obtener más información.
</Alert>

Para `Python`, instala Sentry Prevent CLI con `pip install sentry-prevent-cli`. Debes tener `Python 3` como predeterminado.

```bash
pip install sentry-prevent-cli
```

Para las `binary platforms`, usa los siguientes comandos para descargar la versión más reciente de Sentry Prevent CLI.

```bash {tabTitle:macos}
curl -L https://github.com/getsentry/prevent-cli/releases/latest/download/sentry-prevent-cli-macos -o sentry-prevent-cli_macos
chmod +x sentry-prevent-cli_macos
```

```bash {tabTitle:linux x64_64}
curl -L https://github.com/getsentry/prevent-cli/releases/latest/download/sentry-prevent-cli-linux -o sentry-prevent-cli_linux_x64_64
chmod +x sentry-prevent-cli_linux_x64_64
```

```bash {tabTitle:linux arm64}
curl -L https://github.com/getsentry/prevent-cli/releases/latest/download/sentry-prevent-cli-linux-arm64 -o sentry-prevent-cli_linux_arm64
chmod +x sentry-prevent-cli_linux_arm64
```

```bash {tabTitle:alpine linux x64_64}
curl -L https://github.com/getsentry/prevent-cli/releases/latest/download/sentry-prevent-cli-alpine -o sentry-prevent-cli_alpine_x64_64
chmod +x sentry-prevent-cli_alpine_x64_64
```

```bash {tabTitle:alpine linux arm64}
curl -L https://github.com/getsentry/prevent-cli/releases/latest/download/sentry-prevent-cli-alpine-arm64 -o sentry-prevent-cli_alpine_arm64
chmod +x sentry-prevent-cli_alpine_arm64
```

```bash {tabTitle:windows}
curl -L https://github.com/getsentry/prevent-cli/releases/latest/download/sentry-prevent-cli-windows.exe -o sentry-prevent-cli_windows.exe
```

<div id="uploading-test-results">
  ## Carga de resultados de pruebas
</div>

Para cargar los resultados de las pruebas, usa el siguiente comando:

```bash
sentry-prevent-cli upload --report-type test-results --token <PREVENT_TOKEN>
```

<div id="integrity-checking-the-cli">
  ## Verificación de integridad de la CLI
</div>

Las publicaciones de GitHub también incluyen el SHA de cada recurso. Puedes ejecutar

```bash
shasum -a 256 <nombre_de_archivo>
```

o

```bash
sha256sum <archivo>
```

y compara la suma de verificación generada con la de las publicaciones de GitHub.
