description: "Aprende cómo Sentry puede actuar como árbitro para aprobar o rechazar canary releases en tus pipelines de implementación."
---

<Alert level="warning" title="Aviso de deprecación">
  Esta integración está obsoleta y se eliminará en el futuro también para las organizaciones existentes.
</Alert>

Las [Sentry GitHub Deployment Gates](https://github.com/apps/sentry-deployment-gate) permiten que Sentry interactúe con las versiones de GitHub mediante reglas de protección de implementaciones. Después de enviar la información de la versión a Sentry, puedes monitorear una versión en busca de errores y rechazar un canary release antes de que se implemente por completo.

Para más detalles sobre Sentry Deployment Gate, consulta la [aplicación de GitHub](https://github.com/apps/sentry-deployment-gate).

<Alert level="warning">
  Si haces cambios en el slug de tu organización, tendrás que actualizar la configuración de esta integración. Obtén más información en nuestra [guía de solución de problemas](/es/organization/integrations/troubleshooting).
</Alert>
