El [Sentry Release Pipe](https://bitbucket.org/sentryio/sentry-new-release) automatiza la gestión de versiones de Sentry en Bitbucket Pipelines con un solo paso. Tras enviar la información de la versión de Sentry, podrás asociar commits a versiones. También podrás aplicar source maps para ver el código original en Sentry.

Para más detalles sobre los conceptos de gestión de versiones de Sentry, consulta la documentación completa sobre [releases](/es/product/releases/).
