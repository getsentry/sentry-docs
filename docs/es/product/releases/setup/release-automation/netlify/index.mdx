El plugin de compilación de Sentry para Netlify automatiza la gestión de versiones de Sentry en Netlify. Una vez configurado, envía automáticamente información de versiones y commits a Sentry y carga los mapas de origen (source maps), lo que te permite ver el código original en tus stack traces.

La documentación completa está disponible en el [README del plugin](https://github.com/getsentry/sentry-netlify-build-plugin).

Para más detalles sobre los conceptos de gestión de versiones de Sentry, consulta nuestra documentación sobre [releases](/es/product/releases/).

<Alert>
  Nota: Este plugin de compilación es independiente de `@netlify/sentry`, que es un plugin de monitoreo creado por Netlify. La documentación de ese plugin se encuentra [aquí](https://docs.netlify.com/netlify-labs/experimental-features/sentry-integration/). Para más información, consulta [`@sentry/netlify-build-plugin` vs. `@netlify/sentry`](https://github.com/getsentry/sentry-netlify-build-plugin#sentrynetlify-build-plugin-vs-netlifysentry).
</Alert>
