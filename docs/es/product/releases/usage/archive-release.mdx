Dado que un evento recibido de una versión eliminada la vuelve a crear automáticamente, Sentry ofrece la opción de ocultar o archivar versiones. Archivar también puede ser útil si ya no te interesa ver una versión o si creaste accidentalmente una versión que ya no quieres seguir.

<div id="using-sentryio">
  ## Uso de Sentry.io
</div>

Archivar en [sentry.io](https://sentry.io) elimina la versión de la lista de versiones que puedes seleccionar para ver:

![archive release ui](./img/release-archive.png)

Las versiones archivadas no aparecerán en **Releases** a menos que cambies el filtro para mostrarlas:

![archived releases ui](./img/releases-filter-archived.png)

<div id="using-the-cli">
  ## Uso de la CLI
</div>

En la CLI, ejecuta este comando para archivar una versión:

```bash
sentry-cli releases archive VERSION
```
