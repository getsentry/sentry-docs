Puedes restaurar versiones archivadas desde [sentry.io](https://sentry.io) o con Sentry CLI.

<div id="using-sentryio">
  ## Uso de Sentry.io
</div>

En [sentry.io](https://sentry.io), puedes restaurar un lanzamiento haciendo clic en el menú contextual en la parte superior de la página y seleccionando "Restore":

![restore release ui](./img/release-restore.png)

<div id="using-the-cli">
  ## Uso de la CLI
</div>

En la CLI, restaura una versión ejecutando este comando:

```bash
sentry-cli releases restore VERSION
```
