Este tutorial paso a paso te guía para configurar Sentry tanto en una aplicación de ejemplo de frontend (React) como de backend (Express). Al finalizar, podrás trazar el origen y el impacto de tus incidencias a través de proyectos y plataformas. Como parte de este tutorial, vas a:

* Configurar el SDK de Sentry en una app de ejemplo con React y Node/Express.
* Provocar un error de frontend de ejemplo que, a su vez, provoque un error de backend de ejemplo.
* Ir a la página [Traces](https://sentry.io/orgredirect/organizations/:orgslug/traces/) en Sentry y ver una vista detallada de toda tu traza.

Este tutorial utiliza una [aplicación de React de ejemplo](https://github.com/getsentry/tracing-tutorial-frontend) en el frontend, así como una [API de Express](https://github.com/getsentry/tracing-tutorial-backend) de ejemplo en el backend. Tener cierta familiaridad con JavaScript te ayudará a seguirlo. Si lo prefieres, también puedes aplicar los mismos pasos a tu propio proyecto, suponiendo que estás ejecutando React en el frontend y Express en el backend.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una cuenta de [Sentry](https://sentry.io), o [regístrate](https://sentry.io/signup/) para obtenerla
* Una cuenta de [GitHub](https://github.com/), o [regístrate](https://github.com/join) para obtenerla
* [Node 18+](https://nodejs.org/en/download/)
