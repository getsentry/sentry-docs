Este tutorial paso a paso te guía por el proceso de configurar y ajustar tu cuenta de Sentry para supervisar errores en tu aplicación de backend en Python. Como parte de este tutorial, vas a:

* Añadir configuraciones básicas al SDK de Sentry
* Supervisar varios tipos de eventos y errores en Sentry
* Implementar el rastreo de errores entre tus aplicaciones frontend y backend

Este tutorial utiliza una [aplicación de ejemplo en Django](https://github.com/sentry-tutorials/backend-monitoring). Esto nos permite simplificar los pasos necesarios para obtener el máximo valor al integrar Sentry en tu flujo de trabajo de desarrollo.

De forma alternativa, puedes seguir esta guía como referencia y aplicar los cambios pertinentes directamente en tu código fuente.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una cuenta de [Sentry](https://sentry.io), o [regístrate](https://sentry.io/signup/) para crear una
* Una cuenta de [GitHub](https://github.com/), o [regístrate](https://github.com/join) para crear una
