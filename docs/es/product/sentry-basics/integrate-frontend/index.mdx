Este tutorial paso a paso te guía para configurar Sentry en un proyecto de React y monitorear errores en una aplicación frontend. Al completar este tutorial, podrás:

* Recibir alertas cuando se produzcan errores en tu código
* Ver una traza legible con contexto del código fuente
* Ver commits sospechosos y asignaciones sugeridas basadas en información de commits y trazas

Este tutorial utiliza una [aplicación de React de ejemplo](https://github.com/getsentry/frontend-tutorial). Puedes seguirlo con el código de ejemplo proporcionado o aplicar los mismos pasos a tu propio proyecto.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una cuenta de [Sentry](https://sentry.io) o [regístrate](https://sentry.io/signup/) para crear una
* Una cuenta de [GitHub](https://github.com/) o [regístrate](https://github.com/join) para crear una
* [Node 18+](https://nodejs.org/en/download/)
