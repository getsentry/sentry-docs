---

Sentry ofrece Reproducción de sesiones para Web y dispositivos móviles. Esto incluye aplicaciones basadas en el navegador y ciertas plataformas móviles nativas, como Android, iOS y React Native. Ambas versiones de Replay (Web y Móvil) están disponibles de forma general y son estables.

En el caso de las aplicaciones basadas en el navegador, esto incluye sitios web estáticos, aplicaciones de una sola página y también aplicaciones renderizadas en el servidor; por ejemplo, plataformas como Electron, Next.js y Remix.

Para obtener más información sobre qué SDK admitimos, visita nuestra documentación de [Web](web/#supported-sdks) y [Móvil](mobile/#supported-sdks).
