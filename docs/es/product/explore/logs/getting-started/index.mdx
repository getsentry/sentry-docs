<div id="log-drains">
  ## Drains de logs
</div>

Sentry admite la conexión a drains o sinks de logs para ingerir registros. Estamos trabajando activamente para agregar compatibilidad con más plataformas, lo cual estamos siguiendo en [este issue de GitHub](https://github.com/getsentry/sentry/issues/91726).

<div id="endpoints">
  ### Endpoints
</div>

Puedes enviar registros a Sentry mediante el [endpoint de logs de OpenTelemetry (OTLP) de Sentry](/es/concepts/otlp/#opentelemetry-logs). Esto se puede usar con cualquier SDK de OpenTelemetry o con el [OpenTelemetry Collector](https://opentelemetry.io/docs/collector/) para enviar registros a Sentry.
