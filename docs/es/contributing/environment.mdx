Esta configuración supone que tienes [Yarn][Yarn], [Volta][volta] y [pre-commit][pre-commit] instalados.

Si prefieres no usar Volta, asegúrate de estar usando Node 20.

Primero, abre una terminal y clona el repositorio `sentry-docs`:

```bash
# Clona el repositorio de documentación
git clone git@github.com:getsentry/sentry-docs.git
```

A continuación, ve al repositorio clonado y ejecuta lo siguiente para instalar las dependencias:

```bash
# Instalar o actualizar las dependencias de la aplicación
make
```

Ahora, inicia el servidor web de desarrollo (según la documentación que quieras ejecutar):

```bash
# Iniciar el servidor de desarrollo para la documentación de usuario
yarn dev

# Iniciar el servidor de desarrollo para la documentación para desarrolladores
yarn dev:developer-docs
```

Ahora podrás acceder a la documentación en http://localhost:3000.

[Next.js]: https://nextjs.org/

[volta]: https://volta.sh/

[pre-commit]: https://pre-commit.com/

[Yarn]: https://yarnpkg.com/


<div id="linting">
  ## Linting
</div>

Hay algunos linters disponibles. Ejecuta `yarn lint` para ejecutarlos todos.

Algunas comprobaciones se pueden corregir automáticamente con ESLint

```bash
yarn lint:eslint:fix
```

Además, usamos Prettier para formatear nuestro código y Markdown. Ejecuta Prettier si aparecen errores de lint en CI:

```bash
yarn lint:prettier:fix
```
