La sección de Uso explica cómo capturar manualmente eventos y mensajes. Un nuevo framework puede adoptar el contenido común, que se encuentra en `/docs/platforms/<SDK>/common/usage`. No dudes en consultar los archivos de ese directorio para ayudarte a resolver dudas.

<Alert >

No cambies el contenido común. Ese cambio se propagará a todas las guías que dependen de él. En su lugar, abre un issue en GitHub con tus preguntas o sugerencias.

</Alert >

La mayoría de estas páginas son bastante intuitivas, por lo que solo se incluye información básica. Aun así, si te surge alguna duda, abre un issue en GitHub.
