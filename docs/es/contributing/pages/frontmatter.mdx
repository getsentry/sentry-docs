El frontmatter es un bloque en formato YAML definido en la parte superior de cada archivo Markdown o MDX:

```markdown
---
title: Título del Documento
---

Contenido Markdown o MDX
```

El frontmatter estándar se aplicará en casi todas las páginas:

`title`

Título del documento: se usa en `<title>` y también en elementos como los títulos de búsqueda.

`noindex` (false)

Establécelo en true para deshabilitar la indexación (robots, Algolia) de este contenido.

`notoc` (false)

Establécelo en true para deshabilitar el renderizado del índice de contenidos a nivel de página.

`draft` (false)

Establécelo en true para marcar esta página como borrador y ocultarla de varios otros componentes (como PageGrid).

`keywords` ([])

Una lista de palabras clave para la indexación con la búsqueda.

`description`

Una descripción para usar en la etiqueta `<meta>`, así como en cuadrículas de páginas generadas automáticamente.

`sidebar_order` (10)

El orden de esta página en barras laterales y cuadrículas generadas automáticamente.

Gran parte de la demás funcionalidad de las páginas también se controla mediante el frontmatter, como:

* [Redirects](../redirects/)
* [Buscar](../search/)

`nextPage` (`{ path: 'path/to/page', title: 'Page Title' }`)

Anula la siguiente página mostrada en la paginación inferior.

`previousPage` (`{ path: 'path/to/page', title: 'Page Title' }`)

Anula la página anterior mostrada en la paginación inferior.
