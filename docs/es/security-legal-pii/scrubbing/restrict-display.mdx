Es común que, para cumplir con la normativa en tu empresa, los datos solo puedan transmitirse mediante SSL y almacenarse de forma segura. Un ámbito en el que esto suele surgir es el de las notificaciones por correo electrónico.

<div id="confidential-information-in-alerts">
  ## Información confidencial en las alertas
</div>

De forma predeterminada, Sentry enviará una gran cantidad de datos como parte de la notificación de un issue. En algunos casos, estos datos pueden incluir código fuente u otros datos de usuarios.

Si necesitas controlar esto, puedes habilitar la configuración de **Privacidad mejorada** para tu organización. Para hacerlo, visita el panel de tu organización, haz clic en **Settings** y marca la opción para habilitar la privacidad mejorada.

Una vez que hayas habilitado esta configuración, partes del sistema, principalmente las notificaciones por correo electrónico, reflejarán el cambio de inmediato y comenzarán a restringir los datos a solo atributos básicos, como el título y la descripción del issue.
