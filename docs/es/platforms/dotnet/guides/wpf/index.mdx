El SDK de .NET de Sentry funciona con aplicaciones de Windows Presentation Foundation mediante el [paquete NuGet de Sentry](https://www.nuget.org/packages/Sentry). Es compatible con aplicaciones WPF que se ejecutan en .NET Framework 4.6.2, .NET Core 3.0 o versiones posteriores.

<div id="features">
  ## Funcionalidades
</div>

Además de capturar errores, puedes supervisar las interacciones entre varios servicios o aplicaciones si [habilitas el tracing](/es/concepts/key-terms/tracing/).

Selecciona qué funcionalidades de Sentry quieres instalar, además de la monitorización de errores, para obtener a continuación las instrucciones correspondientes de instalación y configuración.

<div id="configure">
  ## Configuración
</div>

Además de [inicializar el SDK con `SentrySdk.Init`](/es/platforms/dotnet/), debes configurar el controlador de errores específico de WPF.

<Alert>
  El SDK debe inicializarse en el constructor de la clase de tu aplicación (normalmente `App.xaml.cs`). No inicialices el SDK en el evento `OnStartup()` de la aplicación o el `Hub` no se inicializará correctamente.
</Alert>

El SDK maneja automáticamente `AppDomain.UnhandledException`. En WPF, en aplicaciones de producción (cuando no hay un depurador conectado), WPF captura la excepción para mostrar un cuadro de diálogo al usuario. También puedes configurar tu aplicación para capturar esas excepciones antes de que aparezca el cuadro de diálogo:

```csharp
using System.Windows;

public partial class App : Application
{
    public App()
    {
        SentrySdk.Init(options =>
        {
            // Indica a qué proyecto de Sentry enviar los eventos:
            options.Dsn = "___PUBLIC_DSN___";

            // Al configurar por primera vez, para ver qué está haciendo el SDK:
            options.Debug = true;

            // Agrega URL y encabezados de solicitud, IP y nombre de usuarios, etc.
            options.SendDefaultPii = true;

            // ___PRODUCT_OPTION_START___ performance
            // Establece traces_sample_rate en 1.0 para capturar el 100% de las transacciones para rastreo.
            // Te recomendamos ajustar este valor en producción.
            options.TracesSampleRate = 1.0;

            // ___PRODUCT_OPTION_END___ performance
            // Habilita el Modo Global ya que esta es una aplicación cliente
            options.IsGlobalModeEnabled = true;

            //TODO: cualquier otra opción que necesites va aquí
        });
        DispatcherUnhandledException += App_DispatcherUnhandledException;
    }

    void App_DispatcherUnhandledException(object sender, DispatcherUnhandledExceptionEventArgs e)
    {
        SentrySdk.CaptureException(e.Exception);

        // Si quieres evitar que la aplicación se cierre inesperadamente:
        e.Handled = true;
    }
}
```
