---
title: Cómo ignorar excepciones
description: "Aprende a ignorar excepciones por tipo al inicializar el SDK."
---

Puedes ignorar excepciones por tipo al inicializar el SDK:

```csharp
// Agrega esto al callback de inicialización del SDK
options.AddExceptionFilterForType<OperationCanceledException>();
```

```fsharp
// Agrega esto al callback de inicialización del SDK
options.AddExceptionFilterForType<OperationCanceledException>()
```

Funciona en toda la cadena de herencia.
El ejemplo anterior también excluirá `TaskCanceledException` porque hereda de `OperationCanceledException`.
