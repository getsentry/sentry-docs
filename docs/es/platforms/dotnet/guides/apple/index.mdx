Sentry admite [.NET para iOS, macOS y Mac Catalyst](https://learn.microsoft.com/dotnet/ios/) directamente mediante el [paquete NuGet de Sentry](https://www.nuget.org/packages/Sentry).

<div id="overview-of-the-features">
  ## Descripción general de las características
</div>

* Todas las características de nuestro [.NET SDK](/es/platforms/dotnet) para tu código administrado
* Informes nativos de fallos para plataformas de Apple, usando nuestro [Cocoa SDK](/es/platforms/apple), que incluye:
  * [Adjuntar automáticamente capturas de pantalla a excepciones nativas](/es/platforms/apple/guides/ios/configuration/options/#attach-screenshot)
  * Detectar [bloqueos de la app](/es/platforms/apple/configuration/app-hangs/)
  * [Trazado automático del rendimiento](/es/platforms/apple/performance/)
  * [Breadcrumbs automáticos](/es/platforms/apple/enriching-events/breadcrumbs/#automatic-breadcrumbs)
  * [Swizzling](/es/platforms/apple/configuration/swizzling/)
* Seguimiento automático de sesiones y [estado de la versión](/es/product/releases/)

<div id="install">
  ## Instalación
</div>

Añade la dependencia de Sentry a tu aplicación .NET para iOS, macOS y/o Mac Catalyst:

```shell {tabTitle:.NET Core CLI}
dotnet add package Sentry -v {{@inject packages.version('sentry.dotnet') }}
```

```powershell {tabTitle:Package Manager}
Install-Package Sentry -Version {{@inject packages.version('sentry.dotnet') }}
```

<div id="configure">
  ## Configurar
</div>

En tu archivo `AppDelegate.cs`, llama a `SentrySdk.Init` en el evento `FinishedLaunching` e incluye cualquier opción que quieras configurar. El `Dsn` es el único parámetro obligatorio.

```csharp
[Register("AppDelegate")]
public class AppDelegate : UIApplicationDelegate
{
    public override bool FinishedLaunching(UIApplication application, NSDictionary launchOptions)
    {
        // Inicializa el SDK de Sentry
        SentrySdk.Init(options =>
        {
            options.Dsn = "___PUBLIC_DSN___";
            options.SendDefaultPii = true; // añade automáticamente la dirección IP del usuario
#if DEBUG
            // Registra información de depuración sobre el SDK de Sentry
            options.Debug = true;
#endif

            // Todas las opciones nativas del SDK de iOS están disponibles a continuación
            // https://docs.sentry.io/platforms/apple/guides/ios/configuration/
            // Habilita la detección de bloqueos de aplicación del SDK nativo de iOS
            options.Native.EnableAppHangTracking = true;

            options.SetBeforeSend(evt =>
            {
                if (evt.Exception?.Message.Contains("Something you don't care want logged?") ?? false)
                {
                    return null; // devuelve null para filtrar el evento
                }
                // o añade datos adicionales
                evt.SetTag("dotnet-iOS-Native-Before", "Hello World");
                return evt;
            });

            options.OnCrashedLastRun = e =>
            {
                Console.WriteLine(e);
            };
        });
    }
}
```

<div id="options">
  ### Opciones
</div>

La integración de .NET para iOS, macOS y Mac Catalyst forma parte de [Sentry](/es/platforms/dotnet/). Consulta la documentación de ese paquete para obtener información sobre opciones independientes de la plataforma.
