---
title: Funcionalidades
description: Conoce las funcionalidades de este SDK de Sentry.
sidebar_order: 2
---

- Integración de una sola línea con ASP.NET Core: `UseSentry`
- Captura excepciones no controladas y con [controladores de excepciones personalizados](https://learn.microsoft.com/en-us/aspnet/core/fundamentals/error-handling?view=aspnetcore-7.0#exception-handler-page)
- Captura `LogError` o `LogCritical`
- Cualquier evento enviado incluirá mensajes de registro relevantes de la aplicación
- Captura automáticamente información de la solicitud, detalles del navegador, sistema operativo del servidor y del entorno de ejecución, información de la versión, etc.
- Captura encabezados de la solicitud y, si así lo eliges, el cuerpo de la solicitud
- Compatibilidad con EventProcessors y ExceptionProcessors (registrados mediante inyección de dependencias)
- Configuración de proxy HTTP
- Protección contra saturación de eventos (429 retry-after y cola interna limitada)
- Probado en Windows, macOS, Linux, .NET Core, .NET Framework y Mono
