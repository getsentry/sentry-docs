<div id="alerts">
  ## Alertas
</div>

Cuando tu tarea recurrente no hace check-in (omitida), excede el tiempo máximo de ejecución configurado (fallida) o informa manualmente un fallo, Sentry creará un evento de error con una etiqueta para tu monitor.

Para recibir alertas sobre estos eventos:

1. Ve a **Alertas** en la barra lateral.
2. Crea una nueva alerta y selecciona &quot;Issues&quot; en &quot;Errors&quot; como tipo de alerta.
3. Configura tu alerta y define un filtro para usar: `The event's tags match {key} {match} {value}`.

Ejemplo: `The event's tags match monitor.slug equals my-monitor-slug-here`
