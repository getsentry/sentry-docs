---
title: Nombre de la transacción
description: "Aprende a establecer o cambiar el nombre de la transacción para capturar al usuario y obtener información clave que ayude a construir una identidad única en Sentry."
---

El nombre de la transacción actual se usa para agrupar transacciones en nuestro
producto [Insights](/es/product/insights/), además de anotar los eventos de error
con su punto de falla.

El nombre de la transacción puede hacer referencia a la ruta actual de la aplicación web o a la tarea
que se está ejecutando. Por ejemplo:

* `GET /api/{version}/users/`
* `UserListView`
* `myapp.tasks.renew_all_subscriptions`

Idealmente, el nombre de la transacción no contiene valores variables como IDs de usuario,
sino que tiene una cardinalidad baja y aun así identifica de forma única una parte del
código que te importa.

Muchas de nuestras integraciones de frameworks ya establecen un nombre de transacción, aunque puedes establecer uno tú mismo.
