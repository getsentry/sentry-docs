Sentry usa *breadcrumbs* para crear un rastro de eventos que sucedieron antes de un problema. Estos eventos son muy similares a los registros tradicionales, pero pueden capturar datos estructurados más ricos.

Esta página ofrece una descripción general del registro manual de breadcrumbs y su personalización. Obtén más información sobre los datos que se muestran en la página **Detalles del problema** y cómo filtrar los breadcrumbs para resolver problemas rápidamente en [Uso de breadcrumbs](/es/product/error-monitoring/breadcrumbs).

<Alert title="Más información sobre el uso del SDK">
  Los desarrolladores que quieran modificar la interfaz de breadcrumbs pueden obtener más información en nuestra [documentación para desarrolladores sobre la interfaz de Breadcrumbs](https://develop.sentry.dev/sdk/data-model/event-payloads/breadcrumbs/).
</Alert>

<div id="manual-breadcrumbs">
  ## Migas de pan manuales
</div>

Puedes añadir migas de pan manualmente cuando pase algo interesante. Por ejemplo, podrías registrar una miga de pan si el usuario se autentica o si se produce otro cambio de estado.

Registrar una miga de pan manualmente:

<PlatformContent includePath="enriching-events/breadcrumbs/breadcrumbs-example" />

Las claves de migas de pan disponibles son `type`, `category`, `message`, `level`, `timestamp` (que por defecto usa la hora del reloj del sistema) y `data`, que es donde puedes incluir cualquier información adicional que quieras que lleve la miga de pan. Usar claves distintas de estas seis no generará un error, pero hará que los datos se descarten cuando Sentry procese el evento.

  ## Personalizar los breadcrumbs
</div>

Los SDK permiten personalizar los breadcrumbs mediante el hook <PlatformIdentifier name="before-breadcrumb" />.

Este hook recibe un breadcrumb ya ensamblado y <PlatformLink to="/configuration/filtering/#using-hints">un objeto `hint`</PlatformLink> que contiene metadatos adicionales. La función puede modificar el breadcrumb o decidir descartarlo por completo devolviendo `None`:

<PlatformContent includePath="enriching-events/breadcrumbs/before-breadcrumb" />
