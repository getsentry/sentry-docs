La integración de Starlite agrega compatibilidad con el [framework Starlite](https://docs.litestar.dev/1/).

<div id="install">
  ## Instalar
</div>

Instala `sentry-sdk` desde PyPI con el extra `starlite`:

```bash {tabTitle:pip}
pip install "sentry-sdk[starlite]" "uvicorn"
```

```bash {tabTitle:uv}
uv add "sentry-sdk[starlite]" "uvicorn"
```

<div id="configure">
  ## Configuración
</div>

Si tienes el paquete `starlite` en tus dependencias, la integración de Starlite se habilitará automáticamente cuando inicialices el SDK de Sentry.

<PlatformContent includePath="getting-started-config" />

<div id="verify">
  ## Verifica
</div>

```python
from starlite import Starlite, get

sentry_sdk.init(...)  # igual que arriba

@get("/hello")
async def hello_world() -> str:
    1 / 0
    return "¡Hola!"

app = Starlite(route_handlers=[hello_world])
```

Guarda el archivo de arriba como `app.py` y inicia el servidor de desarrollo con:

```bash
uvicorn app:app
```

Cuando visites [http://localhost:8000/hello](http://localhost:8000/hello) en tu navegador, se creará una transacción en la sección Performance de [sentry.io](https://sentry.io). Además, el `ZeroDivisionError` que hemos introducido en nuestro controlador `hello_world` se enviará a [sentry.io](https://sentry.io) y quedará vinculado a la transacción.

Los datos tardan unos instantes en aparecer en [sentry.io](https://sentry.io).
