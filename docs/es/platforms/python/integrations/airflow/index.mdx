[Apache Airflow](https://airflow.apache.org/) **1.10.6** y versiones posteriores se puede configurar para enviar errores a Sentry.

<div id="installation">
  ## Instalación
</div>

Instala el paquete `apache-airflow` con el requisito `sentry`.

```bash {tabTitle:pip}
pip install "apache-airflow[sentry]"
```

```bash {tabTitle:uv}
uv add "apache-airflow[sentry]"
```

Luego, añade tu DSN de Sentry a tu archivo de configuración (p. ej., `airflow.cfg`) en la sección `[sentry]`.

```ini {filename:airflow.cfg}
[sentry]
sentry_dsn = ___PUBLIC_DSN___
```

Ahora, Airflow debería enviar los errores a Sentry automáticamente. Airflow también generará etiquetas personalizadas y breadcrumbs en función del Directed Acyclic Graph (DAG) actual y de las tareas en el momento del error.

Consulta la [documentación oficial de Apache Airflow](https://airflow.apache.org/docs/stable/errors.html) para más detalles.


<div id="configuration">
  ### Configuración
</div>

Consulta la [documentación oficial de Apache Airflow](https://airflow.apache.org/docs/stable/configurations-ref.html#sentry) para ver la lista completa de opciones de configuración disponibles.
