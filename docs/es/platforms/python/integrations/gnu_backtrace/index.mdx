---
title: Retrotraza de GNU
description: "Conoce la integración de retrotraza de GNU y cómo añadirla a tu lista de integraciones."
---

La integración de retrotraza de GNU analiza la traza de pila nativa producida por [`backtrace_symbols`](https://linux.die.net/man/3/backtrace_symbols) a partir de mensajes de error y la concatena con el traceback de Python.

Actualmente se ha comprobado que funciona con las excepciones del servidor generadas por [`clickhouse-driver`](https://github.com/mymarilyn/clickhouse-driver).

<div id="install">
  ## Instalar
</div>

Instala `sentry-sdk` desde PyPI.

```bash {tabTitle:pip}
pip install "sentry-sdk"
```

```bash {tabTitle:uv}
uv add "sentry-sdk"
```


<div id="configure">
  ## Configuración
</div>

Añade `GnuBacktraceIntegration()` a tu lista de `integrations`:

```python
import sentry_sdk
from sentry_sdk.integrations.gnu_backtrace import GnuBacktraceIntegration

sentry_sdk.init(
    dsn="___PUBLIC_DSN___",
    # Agrega datos como encabezados de solicitud e IP de usuarios, si corresponde;
    # consulta https://docs.sentry.io/platforms/python/data-management/data-collected/ para más información
    send_default_pii=True,
    integrations=[
        GnuBacktraceIntegration(),
    ],
)
```


<div id="verify">
  ## Verificar
</div>

La integración de GNU backtrace se prueba con excepciones del servidor generadas por [`clickhouse-driver`](https://github.com/mymarilyn/clickhouse-driver).

Otras bibliotecas deben generar excepciones con valores compatibles con backtrace para que la integración de GNU backtrace pueda analizarlas.
