La integración de [GQL](https://gql.readthedocs.io/en/stable/) te permite monitorear todas las solicitudes de GraphQL que tu aplicación de Python realiza con la biblioteca `gql`. En concreto, esta integración reporta cada [`TransportQueryError`](https://gql.readthedocs.io/en/latest/modules/transport_exceptions.html#gql.transport.exceptions.TransportQueryError) a Sentry y proporciona contexto adicional, incluido:

* La URL a la que se realizó la solicitud
* La consulta de GraphQL que provocó el error
* La información de `errors` que devuelve el servidor de GraphQL

<div id="install">
  ## Instalación
</div>

Asegúrate de tener instalada la última versión del SDK de Sentry:

```bash {tabTitle:pip}
pip install "sentry-sdk"
```

```bash {tabTitle:uv}
uv add "sentry-sdk"
```

<div id="configure">
  ## Configuración
</div>

Si tienes el paquete `gql` en tus dependencias, la integración de GQL se habilitará automáticamente al inicializar el SDK de Sentry.

<div id="data-privacy-and-the-gql-integration">
  ## Privacidad de datos e integración de GQL
</div>

Para garantizar la privacidad de los datos, el SDK evita enviar, de forma predeterminada, cualquier información que pueda contener información de identificación personal (PII). Consulta la documentación sobre el indicador [`send_default_pii`](/es/platforms/python/configuration/options/#send-default-pii) para obtener más información.

En la integración de GQL, esta política implica que el SDK **no envía** las consultas de GraphQL ni la información de `errors` que devuelve el servidor de GraphQL, a menos que sobrescribas la configuración predeterminada mediante el indicador `send_default_pii`.

<Alert level="warning">
  Antes de habilitar el indicador `send_default_pii`, asegúrate de haber configurado las herramientas de [limpieza de datos sensibles](/es/platforms/python/data-management/sensitive-data/) de Sentry para suprimir toda la PII.
</Alert>

Para que Sentry registre las consultas de GraphQL y la información de `errors` devuelta por el servidor de GraphQL, agrega lo siguiente a la inicialización de tu SDK:

```python
sentry_sdk.init(
    # ...
    # Agrega datos como encabezados de solicitud e IP de usuarios, si corresponde;
    # consulta https://docs.sentry.io/platforms/python/data-management/data-collected/ para más información
    send_default_pii=True,
)
```
