<div id="configuration">
  ## Configuración
</div>

<div id="enabling-integrations">
  ### Habilitar integraciones
</div>

Puedes añadir integraciones usando la opción de configuración [`integrations`](/es/platforms/python/configuration/options/#integrations).

Las integraciones marcadas como “auto-enabled” en la tabla anterior se activarán automáticamente, a menos que establezcas [`auto_enabling_integrations`](/es/platforms/python/configuration/options/#auto-enabling-integrations) en `False`. Si quieres configurar los parámetros de una integración específica (por ejemplo, cambiar el `transaction_style` predeterminado de Flask), añádela a tu lista de `integrations` como lo harías con una integración que no se habilita automáticamente y pasa las opciones deseadas.

```python
import sentry_sdk
from sentry_sdk.integrations.asyncio import AsyncioIntegration
from sentry_sdk.integrations.flask import FlaskIntegration

sentry_sdk.init(
    integrations=[
        # La integración de Flask se habilita automáticamente, pero queremos cambiar
        # transaction_style del valor predeterminado "endpoint" a "url"
        FlaskIntegration(transaction_style="url"),
        # La integración de asyncio no se habilita automáticamente
        # y debe añadirse manualmente.
        AsyncioIntegration(),
    ],
)
```

<div id="disabling-integrations">
  ### Deshabilitar integraciones
</div>

Para deshabilitar una integración, usa la opción de configuración [`disabled_integrations`](/es/platforms/python/configuration/options/#disabled-integrations):

```python
import sentry_sdk
from sentry_sdk.integrations.flask import FlaskIntegration

sentry_sdk.init(
    # No utilices la integración de Flask aunque Flask esté instalado.
    disabled_integrations=[
        FlaskIntegration(),
    ],
)
```

También es posible desactivar todas las integraciones agregadas automáticamente. Hay dos tipos:

* **Integraciones activadas automáticamente** como `FlaskIntegration` se agregan
  si el SDK detecta que tienes instalado un paquete correspondiente (como Flask).
  Esto sucede cuando la opción [`auto_enabling_integrations`](/es/platforms/python/configuration/options/#auto-enabling-integrations) está en
  `True` (valor predeterminado).
* **Integraciones predeterminadas** como `logging` o `excepthook` siempre están habilitadas,
  independientemente de los paquetes que tengas instalados, siempre que la
  opción [`default_integrations`](/es/platforms/python/configuration/options/#default-integrations) sea `True` (valor predeterminado). Proporcionan
  funcionalidad esencial del SDK, como la deduplicación de errores o el vaciado de eventos al cerrar el intérprete.

Para desactivar todas las integraciones activadas automáticamente, puedes usar la opción [`auto_enabling_integrations`](/es/platforms/python/configuration/options/#auto-enabling-integrations):

```python
import sentry_sdk
from sentry_sdk.integrations.flask import FlaskIntegration

sentry_sdk.init(
    # Desactiva todas las integraciones que se habilitan automáticamente, excepto Flask
    auto_enabling_integrations=False,
    integrations=[
        FlaskIntegration(),
    ],
)
```

Para desactivar todas las [integraciones predeterminadas](default-integrations/),
configura [`default_integrations`](/es/platforms/python/configuration/options/#default-integrations) en `False`. Ten en cuenta que esto desactiva *todas* las integraciones añadidas automáticamente, tanto las predeterminadas como las que se habilitan automáticamente. Cualquier integración que quieras usar
deberá especificarse manualmente mediante la opción de configuración [`integrations`](/es/platforms/python/configuration/options/#integrations).

```python
import sentry_sdk
from sentry_sdk.integrations.atexit import AtexitIntegration
from sentry_sdk.integrations.argv import ArgvIntegration
from sentry_sdk.integrations.dedupe import DedupeIntegration
from sentry_sdk.integrations.excepthook import ExcepthookIntegration
from sentry_sdk.integrations.stdlib import StdlibIntegration
from sentry_sdk.integrations.modules import ModulesIntegration
from sentry_sdk.integrations.threading import ThreadingIntegration

sentry_sdk.init(
    # Desactiva la integración de logging predeterminada, pero conserva las demás.
    default_integrations=False,
    integrations=[
        AtexitIntegration(),
        ArgvIntegration(),
        DedupeIntegration(),
        ExcepthookIntegration(),
        StdlibIntegration(),
        ModulesIntegration(),
        ThreadingIntegration(),
    ],
)
```
