Esta integración utiliza [`pure_eval`](https://github.com/alexmojaki/pure_eval) para evaluar de forma segura expresiones adicionales en el código fuente y mostrar sus valores junto con otras variables locales.

<div id="install">
  ## Instalación
</div>

Instala `sentry-sdk` desde PyPI con el extra `pure_eval`, o ejecuta `pip install pure_eval executing asttokens`.

```bash {tabTitle:pip}
pip install "sentry-sdk[pure_eval]"
```

```bash {tabTitle:uv}
uv add "sentry-sdk[pure_eval]"
```


<div id="configure">
  ## Configuración
</div>

Añade `PureEvalIntegration()` a tu lista de `integrations`:

```python
import sentry_sdk
from sentry_sdk.integrations.pure_eval import PureEvalIntegration

sentry_sdk.init(
    dns="___PUBLIC_DSN___",
    # Agrega datos como encabezados de solicitud e IP de usuarios, si aplica;
    # consulta https://docs.sentry.io/platforms/python/data-management/data-collected/ para más información
    send_default_pii=True,
    integrations=[
        PureEvalIntegration(),
    ],
)
```


<div id="verify">
  ## Verifica
</div>

```python
from types import SimpleNamespace

def main():
    sentry_sdk.init(...)  # mismo que arriba

    namespace = SimpleNamespace()
    namespace.d = {1: 2}
    print(namespace.d[1] / 0)

main()
```

Cuando ejecutes este script de ejemplo, se enviará un error a Sentry. Gracias a `PureEvalIntegration`, la traza de la pila del error incluirá los valores de `namespace.d` y `namespace.d[1]`, que no se mostrarían sin `PureEvalIntegration`.
