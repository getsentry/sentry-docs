---

La integración de [LaunchDarkly](https://launchdarkly.com/) rastrea las evaluaciones de flags de funciones (feature flags) producidas por el SDK de LaunchDarkly. Estas evaluaciones se mantienen en memoria y se envían a Sentry cuando ocurren eventos de error y transacción. **Por ahora, solo admitimos evaluaciones de flags booleanos.**

<div id="install">
  ## Instalar
</div>

Instala `sentry-sdk` desde PyPI con el extra `launchdarkly`.

```bash {tabTitle:pip}
pip install "sentry-sdk[launchdarkly]"
```

```bash {tabTitle:uv}
uv add "sentry-sdk[launchdarkly]"
```


<div id="configure">
  ## Configurar
</div>

Agrega `LaunchDarklyIntegration()` a tu lista de `integrations`:

```python
import sentry_sdk
from sentry_sdk.integrations.launchdarkly import LaunchDarklyIntegration

sentry_sdk.init(
    dsn="___PUBLIC_DSN___",
    # Agrega datos como encabezados de solicitud e IP de usuarios, si aplica;
    # consulta https://docs.sentry.io/platforms/python/data-management/data-collected/ para más información
    send_default_pii=True,
    integrations=[
        LaunchDarklyIntegration(),
    ],
)
```


<div id="verify">
  ## Verificar
</div>

La integración se verifica evaluando una bandera/flag de funcionalidad con el SDK de LaunchDarkly antes de capturar una excepción.

```python
import ldclient
import sentry_sdk

client = ldclient.get()
client.variation("hello", Context.create("test-context"), False)

sentry_sdk.capture_exception(Exception("¡Algo salió mal!"))
```

Visita el sitio web de Sentry y confirma que tu evento de error ha registrado la feature flag &quot;hello&quot; y su valor &quot;false&quot;.


<div id="supported-versions">
  ## Versiones compatibles
</div>

- launchdarkly-server-sdk >= 9.8.0
- sentry-sdk >= 2.19.2
- python >= 3.8
