La integración de [HTTPX](https://www.python-httpx.org/) instrumenta las solicitudes HTTP salientes mediante los clientes de HTTPX, tanto síncrono como asíncrono.

Usa esta integración para crear spans de solicitudes salientes y garantizar que los rastreos se propaguen correctamente a los servicios posteriores.

<div id="install">
  ## Instalación
</div>

Instala `sentry-sdk` desde PyPI con el extra `httpx`.

```bash {tabTitle:pip}
pip install "sentry-sdk[httpx]"
```

```bash {tabTitle:uv}
uv add "sentry-sdk[httpx]"
```


<div id="configure">
  ## Configuración
</div>

La integración de HTTPX se activa automáticamente si tienes instalado el paquete `httpx`.

<div id="verify">
  ## Verifica
</div>

```python
import httpx

def main():
    sentry_init(...)  # igual que antes
    with sentry_sdk.start_transaction(name="testing_sentry"):
        r1 = httpx.get("https://sentry.io/")
        r2 = httpx.post("http://httpbin.org/post")

main()
```

Esto creará una transacción llamada `testing_sentry` en la sección Performance de [sentry.io](https://sentry.io) y generará spans para las solicitudes HTTP salientes.

Los datos pueden tardar unos instantes en aparecer en [sentry.io](https://sentry.io).
