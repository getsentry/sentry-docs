<Alert>
  Para capturar transacciones y spans personalizados según las necesidades de tu organización, primero debes <PlatformLink to="/tracing/">configurar el tracing</PlatformLink>.
</Alert>

Hay dos maneras de aplicar la instrumentación a tu aplicación:

<div id="automatic-instrumentation">
  ## Instrumentación automática
</div>

Muchas integraciones con frameworks populares capturan automáticamente transacciones que se pueden enviar a Sentry. Obtén más información sobre la instrumentación automática [aquí](/es/platforms/python/tracing/instrumentation/automatic-instrumentation/).

<div id="custom-instrumentation">
  ## Instrumentación personalizada
</div>

Para añadir datos de rendimiento personalizados a tu aplicación, debes incorporar instrumentación personalizada en forma de [spans](/es/concepts/key-terms/tracing/distributed-tracing/#traces-transactions-and-spans). Los spans son una forma de medir el tiempo que tarda en producirse una acción específica. Por ejemplo, puedes crear un span para medir cuánto tarda en ejecutarse una función. Obtén más información sobre el ciclo de vida de los spans [aquí](/es/platforms/python/tracing/span-lifecycle/).
