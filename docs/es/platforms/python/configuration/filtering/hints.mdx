title: Pistas

description: "Conoce los diferentes tipos de `hints` de Event y Breadcrumb."
---

Los `hints` de Event y Breadcrumb son objetos que contienen información diversa usada para armar un evento o un breadcrumb. Estas pistas se pasan como el parámetro `hint` a `before_send` y `before_breadcrumb` (así como a los procesadores de eventos) como un diccionario. Se puede proporcionar más de una pista, aunque esto es poco común.

`exc_info`

Si estableces esta pista, será una tupla de información de excepción con la forma `(exc_type, exc_value, tb)`. Puede usarse para extraer información adicional del objeto de error original.

`log_record`

Esta pista se pasa a los breadcrumbs y contiene el registro de log que lo creó. Puede usarse para extraer información adicional del registro de log de `logging` original que no se extrae por defecto. Asimismo, puede ser útil para descartar breadcrumbs poco relevantes.

`httplib_request`

Un objeto de solicitud de `httplib` para breadcrumbs creados a partir de solicitudes HTTP.
