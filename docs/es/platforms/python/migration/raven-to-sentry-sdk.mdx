Esta guía describe los patrones comunes implicados en la migración de `raven-python` al SDK de `sentry-python`.

<div id="installation">
  ## Instalación
</div>

La instalación ahora es la misma sin importar el framework o la biblioteca con la que la integres. No hay rutinas de inicialización alternativas aparte de `sentry_sdk.init`. Para obtener instrucciones específicas de la integración, consulta [nuestra lista de guías del nuevo SDK](/es/platforms/python/).

**Anterior**:

```python
import raven
client = raven.Client("___PUBLIC_DSN___", release="1.3.0")
```

**Novedad**:

```python
import sentry_sdk
sentry_sdk.init(
    dsn="___PUBLIC_DSN___",
    release="1.3.0",
)
```

<div id="set-a-global-tag">
  ## Establecer una etiqueta global
</div>

**Anterior**:

```python
client.tags_context({'key': 'value'})
```

**Novedades**:

```python
sentry_sdk.set_tag('key', 'value')
```

<div id="capture-custom-exception">
  ## Capturar excepción personalizada
</div>

**Anterior**:

```python
try:
    throwing_function()
except Exception:
    client.captureException(extra={'debug': False})
```

**Novedad**:

```python
with sentry_sdk.push_scope() as scope:
    scope.set_extra('debug', False)

    try:
        throwing_function()
    except Exception as e:
        sentry_sdk.capture_exception(e)
```

<div id="capture-a-message">
  ## Capturar un mensaje
</div>

**Anterior**:

```python
client.captureMessage('test', level='info', extra={'debug': False})
```

**Novedades**:

```python
with sentry_sdk.push_scope() as scope:
    scope.set_extra('debug', False)
    sentry_sdk.capture_message('test', 'info')
```

<div id="breadcrumbs">
  ## Breadcrumbs
</div>

**Anterior**:

```python
from raven import breadcrumbs

breadcrumbs.record(
    message='Artículo añadido al carrito',
    category='action',
    data={
        'isbn': '978-1617290541',
        'cartSize': '3'
    }
)
```

**Novedad**:

```python
sentry_sdk.add_breadcrumb(
  message='Artículo añadido al carrito',
  category='action',
  data={
    'isbn': '978-1617290541',
    'cartSize': '3'
  }
)
```

<div id="filtering-sensitive-data">
  ## Filtrado de datos sensibles
</div>

Raven solía tener algunas heurísticas integradas para detectar campos de contraseña y
números de tarjetas de crédito. Como esas heurísticas eran las mismas para todo el mundo, era
imposible mejorarlas para un caso de uso sin afectar el caso de uso de otra
persona. No hay una solución sencilla del tipo buscar y reemplazar en el nuevo
SDK.

Te recomendamos considerar las opciones alternativas descritas en [*Filtrado de eventos*](/es/platforms/python/configuration/filtering/).
