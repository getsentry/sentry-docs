Cuando un usuario se encuentra con un error, Sentry permite recopilar comentarios adicionales. Puedes recopilar comentarios según el método que admita el SDK.

<div id="crash-report-modal">
  ## Modal de Crash-Report
</div>

Nuestro modal de Crash-Report, integrable y basado en JavaScript, es útil cuando normalmente mostrarías una página de error simple (la clásica `500.html`) en tu sitio web.

Para recopilar comentarios, el modal de Crash-Report solicita y recoge el nombre del usuario, su dirección de correo electrónico y una descripción de lo ocurrido. Cuando se envían comentarios, Sentry los vincula con el evento original, brindándote información adicional sobre los problemas.
