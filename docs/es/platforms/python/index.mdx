<div id="install">
  ## Instalación
</div>

Instala el SDK de Sentry con [`pip`](https://pip.pypa.io/en/stable/):

```bash {tabTitle:pip}
pip install "sentry-sdk"
```

```bash {tabTitle:uv}
uv add "sentry-sdk"
```

<div id="configure">
  ## Configurar
</div>

La configuración debe hacerse **lo antes posible** en el ciclo de vida de tu aplicación.

```python
import sentry_sdk

sentry_sdk.init(
    dsn="___PUBLIC_DSN___",
    # Agregar encabezados de solicitud e IP para usuarios,
    # consulta https://docs.sentry.io/platforms/python/data-management/data-collected/ para más información
    send_default_pii=True,
    # ___PRODUCT_OPTION_START___ performance
    # Establece traces_sample_rate en 1.0 para capturar el 100%
    # de las transacciones para el rastreo.
    traces_sample_rate=1.0,
    # ___PRODUCT_OPTION_END___ performance
    # ___PRODUCT_OPTION_START___ profiling
    # Para recopilar perfiles de todas las sesiones de perfil,
    # establece `profile_session_sample_rate` en 1.0.
    profile_session_sample_rate=1.0,
    # Los perfiles se recopilarán automáticamente mientras
    # haya un span activo.
    profile_lifecycle="trace",
    # ___PRODUCT_OPTION_END___ profiling
    # ___PRODUCT_OPTION_START___ logs

    # Habilita el envío de logs a Sentry
    enable_logs=True,
    # ___PRODUCT_OPTION_END___ logs
)
```

En programas asíncronos, se recomienda llamar a `sentry_sdk.init()` dentro de una función `async` para garantizar que el código asíncrono se instrumente correctamente. Si es posible, recomendamos llamar a `sentry_sdk.init()` al inicio de la primera función `async` que llames.

```python
import asyncio
import sentry_sdk

async def main():
    sentry_sdk.init(
        ...  # igual que arriba
    )

asyncio.run(main())
```

<Alert>
  Obtén más información sobre cómo capturar manualmente un error o un mensaje en nuestra <PlatformLink to="/usage/">documentación de uso</PlatformLink>.

Para ver y resolver el error registrado, inicia sesión en [sentry.io](https://sentry.io) y selecciona tu proyecto. Al hacer clic en el título del error se abrirá una página donde podrás ver información detallada y marcarlo como resuelto.
