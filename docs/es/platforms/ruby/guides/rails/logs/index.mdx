<div id="troubleshooting">
  ## Solución de problemas
</div>

Si los registros no aparecen en Sentry:

**Verifica la configuración:**

```ruby
Sentry.init do |config|
  config.enable_logs = true  # Debe configurarse
end
```

**Prueba tu configuración:**

```ruby
Sentry.logger.info("Mensaje de prueba de registro")
Sentry.get_current_client.flush
```

Luego, verifica si el log aparece en Sentry.

**Activa el modo de depuración:**

```ruby
config.debug = true  # Ver salida de diagnóstico del SDK
```
