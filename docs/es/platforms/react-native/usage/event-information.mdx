El SDK de Sentry para React Native agregará automáticamente información a tu evento para facilitar la depuración y comprensión de tu app.

<div id="event-origin-and-environment-tags">
  ## Etiquetas de origen y de entorno del evento
</div>

La etiqueta de origen del evento indica desde qué capa de tu app se generó el evento.

Los valores posibles para `event.origin` son:

- `javascript`
- `android`
- `ios`

La etiqueta de entorno del evento indica en qué entorno se generó el evento.

Los valores posibles para `event.environment` son:

- `javascript`
- `java`
- `native`

Por ejemplo, si se produce un bloqueo nativo cuando tu app se está ejecutando en iOS, el evento podría etiquetarse con `event.origin: ios` y `event.environment: native`.
