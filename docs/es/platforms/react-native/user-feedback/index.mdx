La función de comentarios de los usuarios te permite recopilar opiniones desde cualquier lugar de tu aplicación, en cualquier momento, sin necesidad de que ocurra antes un evento de error.

Ten en cuenta que, si usas una instancia autogestionada de Sentry, deberás estar en la versión 24.4.2 o superior para usar todas las capacidades de la función de comentarios de los usuarios. Las versiones anteriores pueden tener funcionalidades limitadas.

<div id="user-feedback-widget">
  ## Widget de comentarios de usuario
</div>

El widget de comentarios de usuario permite a los usuarios enviar comentarios desde cualquier lugar de tu aplicación.

Para recopilar comentarios de usuario desde tu aplicación, usa el método `showFeedbackWidget`.

```javascript
import * as Sentry from "@sentry/react-native";

Sentry.wrap(RootComponent);

Sentry.showFeedbackWidget();
```

También puedes usar `showFeedbackButton` y `hideFeedbackButton` para mostrar y ocultar un botón que abre el widget de comentarios.

```javascript
import * as Sentry from "@sentry/react-native";

Sentry.wrap(RootComponent);

Sentry.showFeedbackWidget();
Sentry.hideFeedbackButton();
```

Ten en cuenta que [el componente raíz de la aplicación debe envolverse con `Sentry.wrap`](/es/platforms/react-native/#wrap-your-app) para que los métodos `showFeedbackWidget` y `showFeedbackButton` funcionen. Estos métodos dependen de la implementación de `Modal` de React Native. Está totalmente admitida en la arquitectura heredada. Para la nueva arquitectura (renderizador Fabric), requiere React Native `0.71` o superior.

Para configurar el widget o el Button, puedes usar `Sentry.feedbackIntegration({})` o añadirlo a la inicialización de Sentry.

```javascript
import * as Sentry from "@sentry/react-native";

Sentry.init({
  dsn: "___PUBLIC_DSN___",

  integrations: [
    Sentry.feedbackIntegration({
        // La configuración adicional del SDK va aquí; por ejemplo:
        styles:{
            submitButton: {
                backgroundColor: '#6a1b9a',
            },
        },
        namePlaceholder: 'Nombre completo',
    }),
  ],
});
```

Hay muchas opciones que puedes pasar al constructor de la integración. Consulta la [documentación de configuración](/es/platforms/react-native/user-feedback/configuration/) para obtener más detalles.

<div id="feedback-widget-component">
  ### Componente FeedbackWidget
</div>

También puedes integrar el componente `FeedbackWidget` manualmente en tu aplicación.

```javascript
import { FeedbackWidget } from "@sentry/react-native";

<FeedbackWidget/>
```

Para conocer todas las opciones que puedes pasar al componente `FeedbackWidget`, consulta la [documentación de configuración](/es/platforms/react-native/user-feedback/configuration/).

<div id="session-replay">
  ### Reproducción de sesiones
</div>

El widget de comentarios de usuario se integra sin problemas con Reproducción de sesiones. Cuando se abre el widget, el SDK almacena hasta 30 segundos de la sesión del usuario. Si se envían comentarios, esa reproducción se envía junto con ellos, lo que te permite ver tanto los comentarios como las acciones del usuario previas al envío.

<div id="user-feedback-api">
  ## API de comentarios de usuario
</div>

La API de comentarios de usuario te permite recopilar opiniones mientras utilizas tu propia interfaz. Puedes usar el mismo lenguaje de programación que emplea tu app para enviar comentarios de usuario. En este caso, el SDK crea la solicitud HTTP para que no tengas que encargarte de enviar los datos por HTTP.

Sentry vincula los comentarios con el evento original, lo que te brinda información adicional sobre los problemas. Sentry necesita el `eventId` para poder asociar los comentarios de usuario con el evento correspondiente. Por ejemplo, para obtener el `eventId`, puedes usar <PlatformLink to="/configuration/options/#before-send"><PlatformIdentifier name="before-send" /></PlatformLink> o el valor de retorno del método que captura un evento.
