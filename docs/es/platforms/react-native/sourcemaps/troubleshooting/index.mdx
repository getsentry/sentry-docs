Si no se reconocen los mapas de código fuente, verifica si aparecen advertencias como:

![Advertencia de fuentes faltantes](./img/001-manual-sourcemaps.png)

El nombre de archivo del bundle debe coincidir con el nombre de archivo del evento (y el que se muestra en la advertencia) para aplicar correctamente los mapas de código fuente.

Encuentra más casos de solución de problemas en nuestra [documentación de Troubleshooting](/es/platforms/react-native/troubleshooting/#source-maps).
