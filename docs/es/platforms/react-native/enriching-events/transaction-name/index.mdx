---
title: Nombre de la transacción
description: "Aprende a establecer o reemplazar el nombre de la transacción para capturar al usuario y obtener información clave que construya una identidad única en Sentry."
---

El nombre de la transacción actual se usa para agrupar transacciones en nuestro producto
[Insights](/es/product/insights/), además de anotar los eventos de error
con su punto de fallo.

El nombre de la transacción puede hacer referencia a la ruta actual de la aplicación web o a la tarea
que se está ejecutando. Por ejemplo:

* `GET /api/{version}/users/`
* `UserListView`
* `myapp.tasks.renew_all_subscriptions`

Idealmente, el nombre de la transacción no contiene valores variables, como IDs de usuario,
sino que mantiene una baja cardinalidad y aun así identifica de forma única una parte del código
que te interesa.

Muchas de nuestras integraciones con frameworks ya establecen un nombre de transacción, aunque puedes definir uno tú mismo.

Primero tendrás que importar el SDK, como de costumbre:

```javascript
import * as Sentry from "@sentry/react-native";
```

Para reemplazar el nombre de la transacción que se está ejecutando:

```javascript
Sentry.getCurrentScope().setTransactionName("UserListView");
```

Consulta [la documentación de tracing](../../tracing/) para saber cómo iniciar y finalizar transacciones.
