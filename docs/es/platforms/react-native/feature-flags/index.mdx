<div id="prerequisites">
  ## Requisitos previos
</div>

* [SDK de Sentry](/es/platforms/react-native/#configure) versión `7.0.0`.

<div id="usage">
  ## Uso
</div>

La integración de Feature Flags te permite realizar un seguimiento manual de las evaluaciones de flags mediante una API. Estas evaluaciones se mantienen en memoria y se envían a Sentry en eventos de error y de transacción. **Por el momento, solo admitimos evaluaciones de flags booleanos.**

```javascript
import * as Sentry from "@sentry/react-native";

const flagsIntegration = Sentry.featureFlagsIntegration();
flagsIntegration.addFeatureFlag('feature_flag_a', true);

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [flagsIntegration],
});
```

Si llamas a esta función varias veces con el mismo nombre de bandera, se sobrescribirá el valor anterior.
