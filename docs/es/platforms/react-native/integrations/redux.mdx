title: Integración de Redux
description: "Conoce la integración de Redux de Sentry."
sidebar_order: 50
---

_(Disponible a partir de la versión 1.8.0)_

La compatibilidad con Redux está incluida en el paquete `@sentry/react-native` desde la versión `1.8.0`. Para integrar Sentry con Redux, usa `Sentry.createReduxEnhancer` en el mismo lugar donde inicializas tu store de Redux.
