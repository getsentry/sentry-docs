<div id="automatically-captured-transactions">
  ## Transacciones capturadas automáticamente
</div>

El SDK de Unity crea automáticamente transacciones para el inicio del juego y la carga de escenas.

Si quieres que el SDK también genere spans para todas las llamadas a `Awake` durante el inicio y las operaciones de escena, puedes habilitar las siguientes opciones desde la ventana de configuración del Editor o <Link to="/platforms/unity/configuration/options/#programmatic-configuration">de forma programática</Link>.

![Selecciona las transacciones generadas automáticamente.](./auto-transaction-config.png)

```csharp
public override void Configure(SentryUnityOptions options)
{
    options.AutoSceneLoadTraces = true;
    options.AutoStartupTraces = true;
}
```

Después de configurar la instrumentación, el SDK de Unity generará trazas como se muestra en la imagen a continuación.

![Ejemplo de transacción de inicio.](./startup-transaction.png)
