---
title: Cómo solucionar problemas de CORS
sidebar_order: 80
---

Si tu frontend y backend están alojados en dominios distintos (por ejemplo, el frontend en `https://example.com` y el backend en `https://api.example.com`), y el frontend realiza solicitudes XHR/fetch al backend, tendrás que configurar los encabezados [CORS](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Access-Control-Allow-Headers) en el backend para evitar que se bloqueen las solicitudes.

Configura el CORS del backend para permitir los encabezados `sentry-trace` y `baggage`.

La configuración del encabezado de respuesta del servidor podría ser: `"Access-Control-Allow-Headers: sentry-trace, baggage"`. La configuración específica dependerá de tu implementación.
