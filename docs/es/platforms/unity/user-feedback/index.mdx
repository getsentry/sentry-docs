<Alert level="info">

El prefab de comentarios de los usuarios está disponible en la versión `4.0.0` del SDK de Sentry para Unity, que actualmente está en beta.

</Alert>

Cuando un usuario encuentra un error, Sentry permite recopilar comentarios adicionales. El SDK de Unity ofrece tanto una solución sencilla basada en prefab como una API programática para recopilar comentarios de los usuarios.

<div id="use-the-user-feedback-prefab">
  ## Usa el prefab de comentarios de usuario
</div>

El SDK de Unity incluye un prefab listo para usar que ofrece una solución completa para comentarios de usuario. Este prefab incluye:

- Un botón de comentarios (icono de megáfono) que abre el formulario de comentarios
- Un formulario de comentarios con campos para:
  - Nombre (opcional)
  - Correo electrónico (opcional)
  - Mensaje/comentarios (obligatorio)
  - Captura de pantalla (opcional, se toma automáticamente)

<div id="adding-the-prefab-to-your-scene">
  ### Agregar el prefab a tu escena
</div>

Para agregarlo, arrastra y suelta el prefab User Feedback desde el paquete del Unity SDK en cualquier escena. El prefab está listo para usarse de inmediato sin configuración adicional e incluye un canvas escalable que funciona en diferentes tamaños y resoluciones de pantalla.

<div id="default-appearance">
  ### Apariencia predeterminada
</div>

El prefab de comentarios de usuario incluye un botón con un megáfono que abre el formulario de comentarios:

![Botón de megáfono](./img/Megahpone.png)

Al hacer clic, se muestra el formulario de comentarios con campos para que el usuario ingrese información:

![Formulario de comentarios](./img/FeedbackForm.png)

<div id="customization">
  ### Personalización
</div>

Puedes personalizar el formulario de comentarios de las siguientes maneras:

- Crear una variante del prefab para heredar del original y seguir recibiendo actualizaciones
- Eliminar los campos opcionales (nombre, correo electrónico) quitándolos del VerticalLayoutGroup
- Modificar la apariencia y el estilo de los elementos del formulario
- El prefab incluye un inspector personalizado que resalta referencias que faltan o están rotas:
  - Los campos opcionales que faltan se resaltan en amarillo
  - Los campos obligatorios que faltan se resaltan en rojo

<div id="use-the-net-sdk-api">
  ## Usa la API del SDK de .NET
</div>

Si prefieres crear tu propio formulario de comentarios o necesitas control por código, puedes usar la API del SDK para enviar comentarios a Sentry:

```csharp {tabTitle:C#}
var eventId = SentrySdk.CaptureMessage("Un evento que recibirá comentarios de usuarios.");

SentrySdk.CaptureFeedback("Se estropeó.", "user@example.com", "El usuario", associatedEventId: eventId);
```
