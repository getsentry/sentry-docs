Sentry es compatible con [PlayStation](/es/platforms/playstation/), [Xbox](/es/platforms/xbox/) y [Nintendo Switch](/es/platforms/nintendo-switch/) mediante el SDK de Unity. Una vez que completes el proceso de verificación del middleware, nos pondremos en contacto contigo y te enviaremos una invitación a nuestros repositorios privados de GitHub con código específico para consolas.
Esto permite que tu configuración y los datos personalizados definidos en C# aparezcan en las excepciones de C#, así como en los volcados de fallos en estas plataformas de consola.

<Alert>
  El acceso a los informes de errores y fallos de Sentry para consolas es una función de pago. Revisaremos los precios y los detalles contigo durante el proceso de verificación.
</Alert>
