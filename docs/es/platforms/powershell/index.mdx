En esta página te ayudamos a empezar a usar el SDK de PowerShell de Sentry. El SDK de Sentry para PowerShell se ha probado y funciona con:

* PowerShell 7.4+ en Windows, macOS y Linux
* Windows PowerShell 5.1

El SDK de PowerShell se basa en el SDK de Sentry para .NET. Puedes consultar la [documentación del SDK de .NET](/es/platforms/dotnet/configuration) para obtener información sobre funciones avanzadas que no encuentres aquí.
