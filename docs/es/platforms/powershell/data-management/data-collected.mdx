---
title: Datos recopilados
description: "Consulta qué datos recopila el SDK de Sentry para PowerShell."
sidebar_order: 1
---

El SDK de Sentry para PowerShell es un envoltorio del SDK de Sentry para .NET. Los datos que recopila el SDK de Sentry para PowerShell son los mismos que recopila el SDK de Sentry para .NET. Para más detalles, consulta [Datos recopilados por el SDK de Sentry para .NET](/es/platforms/dotnet/data-management/data-collected/).
