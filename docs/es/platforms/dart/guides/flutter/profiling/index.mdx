La función de generación de perfiles captura datos a través de varias capas de lenguaje, incluidos los lenguajes nativos (como Swift y Objective-C), así como Dart.

<Alert>

La generación de perfiles de Flutter está actualmente en alfa y disponible solo para **iOS** y **macOS**, y requiere la versión 7.12.0 o posterior del SDK.

</Alert>

<div id="enable-tracing">
  ## Habilitar el tracing
</div>

El perfilado depende de que el producto Tracing de Sentry esté habilitado previamente. Para habilitar el tracing en el SDK:

```dart {diff}
SentryFlutter.init(
  (options) => {
    options.dsn = '___PUBLIC_DSN___';
+   // Recomendamos ajustar este valor en producción:
+   options.tracesSampleRate = 1.0;
  },
  appRunner: () => runApp(MyApp()),
);
```

Consulta la <PlatformLink to="/tracing/">documentación de configuración de tracing</PlatformLink> para obtener más detalles sobre cómo configurar el muestreo. Establecer la tasa de muestreo en 1.0 significa que se capturarán todas las transacciones.


<div id="enable-profiling">
  ## Habilitar el perfilado
</div>

Para habilitar el perfilado, establece `profilesSampleRate`:

```dart {diff}
SentryFlutter.init(
  (options) => {
    options.dsn = '___PUBLIC_DSN___';
    // Recomendamos ajustar este valor en producción:
    options.tracesSampleRate = 1.0;
+   // La tasa de muestreo para perfilado es relativa a tracesSampleRate
+   // Al configurarlo en 1.0 se perfilará el 100% de las transacciones muestreadas:
+   options.profilesSampleRate = 1.0;
  },
  appRunner: () => runApp(MyApp()),
);
```
