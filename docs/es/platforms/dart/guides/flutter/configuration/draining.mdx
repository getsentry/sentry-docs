---
title: Apagado y vaciado
sidebar_order: 80
description: "Conoce el comportamiento predeterminado de nuestro SDK si la aplicación se cierra inesperadamente."
---

El comportamiento predeterminado de la mayoría de los SDK es enviar eventos por la red de forma asíncrona en segundo plano. Esto significa que algunos eventos podrían perderse si la aplicación se cierra inesperadamente. Los SDK ofrecen mecanismos para gestionar esta situación.
