Los símbolos de depuración son esenciales para comprender las trazas de pila en tu aplicación de Dart cuando se producen errores. Sin símbolos de depuración, las trazas de pila de código minificado u ofuscado pueden resultar difíciles o imposibles de interpretar.

<div id="what-are-debug-symbols">
  ## ¿Qué son los símbolos de depuración?
</div>

Los símbolos de depuración proporcionan la información necesaria para convertir direcciones de programa en nombres de funciones, nombres de archivos fuente y números de línea. Cuando compilas una aplicación de Dart, especialmente con la ofuscación u optimización activadas, esta información suele eliminarse del paquete principal de la aplicación para reducir el tamaño.

<div id="when-to-upload-debug-symbols">
  ## Cuándo cargar símbolos de depuración
</div>

Con la configuración predeterminada, obtienes trazas de pila completas en los errores de Dart de forma inmediata, a menos que uses las opciones [`split-debug-info`](https://flutter.dev/docs/perf/app-size#reducing-app-size) y [`obfuscate`](https://flutter.dev/docs/deployment/obfuscate). En esos casos, debes cargar los archivos de información de depuración generados por la compilación para que Sentry pueda mostrar trazas de pila correctas.

<div id="upload-methods">
  ## Métodos de carga
</div>

Sentry ofrece varios métodos para subir símbolos de depuración en aplicaciones Dart:

1. **[Complemento de Sentry para Dart (recomendado)](/es/platforms/dart/guides/flutter/debug-symbols/dart-plugin)** - La forma más sencilla de subir automáticamente símbolos de depuración para iOS, Android y Web.

2. **[Opciones de carga manual](/es/platforms/dart/guides/flutter/debug-symbols/manual-upload):**
   * [iOS y macOS](/es/platforms/apple/dsym/) - Sube archivos dSYM para la simbolización de fallos en plataformas de Apple.
   * [Android](/es/cli/dif/#uploading-files) - Sube archivos de mapeo manualmente usando la CLI de Sentry.
   * [Android NDK](/es/cli/dif/#uploading-files) - Sube archivos de depuración ELF DWARF para fallos de código nativo.
   * [Web](/es/cli/releases/#upload-source-maps) - Sube mapas de origen (source maps) para aplicaciones web de Flutter.

Elige el método que mejor se adapte a tu flujo de trabajo y a los requisitos de tu plataforma. Para la mayoría de los usuarios, el complemento de Sentry para Dart es la solución más sencilla en todas las plataformas.
