El [complemento de Sentry para Dart](https://github.com/getsentry/sentry-dart-plugin) es la forma recomendada de cargar símbolos de depuración para aplicaciones en Dart. Automatiza el proceso de carga para Android, iOS, macOS y la Web, lo que simplifica proporcionar a Sentry los archivos necesarios para la simbolización.
