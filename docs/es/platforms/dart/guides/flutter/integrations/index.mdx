El SDK de Sentry usa integraciones para conectarse con bibliotecas populares, instrumentar tu aplicación automáticamente y ofrecerte los mejores datos desde el primer momento.

Las integraciones pueden agregar automáticamente instrumentación de errores, instrumentación de rendimiento y/o información de contexto adicional a tu aplicación. Flutter se basa en el SDK de Dart, por lo que todas las integraciones de Dart también están disponibles en Flutter.
