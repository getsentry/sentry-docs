---
title: Cómo resolver problemas de CORS
sidebar_order: 80
---

Si tu frontend y backend están en dominios distintos (por ejemplo, el frontend en `https://example.com` y el backend en `https://api.example.com`), y el frontend realiza solicitudes XHR/fetch al backend, deberás configurar los encabezados [CORS](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Access-Control-Allow-Headers) en tu backend para evitar que se bloqueen las solicitudes.

Configura el CORS de tu backend para permitir los encabezados `sentry-trace` y `baggage`.

La configuración de los encabezados de respuesta de tu servidor podría verse así: `"Access-Control-Allow-Headers: sentry-trace, baggage"`. La configuración exacta dependerá de tu entorno.
