El SDK de Dart de Sentry permite el reporte automático de errores y excepciones e identifica problemas de rendimiento en tu aplicación. A continuación, encontrarás una lista de funciones disponibles como parte de este SDK.

**Funciones:**

- Breadcrumbs capturados automáticamente:
  - <PlatformLink to="/enriching-events/breadcrumbs/#automatic-breadcrumbs">Dart SDK</PlatformLink>.
  - <PlatformLink to="/integrations/logging">Integración de logging</PlatformLink>.
  - <PlatformLink to="/integrations/dio">Integración de Dio</PlatformLink>.
- Los <PlatformLink to="/enriching-events/attachments/">adjuntos</PlatformLink> enriquecen tus eventos al almacenar archivos adicionales, como archivos de configuración o de registro.
- <PlatformLink to="/user-feedback/">Comentarios de usuario</PlatformLink> permite recopilar información del usuario cuando ocurre un evento.
- <Link to="/product/insights/overview">Tracing</Link> crea transacciones para:
  - <PlatformLink to="/integrations/http-integration/#performance-monitoring-for-http-requests">Solicitudes HTTP</PlatformLink>.
  - <PlatformLink to="/integrations/dio/#performance-monitoring-for-http-requests">Biblioteca HTTP Dio</PlatformLink>.
  - <PlatformLink to="/integrations/file">Integración de E/S de archivos</PlatformLink>.
