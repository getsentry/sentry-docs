Cuando un usuario se encuentra con un error, Sentry permite recopilar comentarios adicionales. Puedes recopilar comentarios según el método que admita el SDK.

<div id="user-feedback-api">
  ## API de comentarios de usuario
</div>

La API de comentarios de usuario te permite recopilar comentarios utilizando tu propia interfaz. Puedes usar el mismo lenguaje de programación que empleas en tu app para enviar comentarios de usuario. En este caso, el SDK crea la solicitud HTTP para que no tengas que encargarte de enviar datos por HTTP.

Sentry asocia los comentarios con el evento original, brindándote información adicional sobre los problemas. Sentry necesita el `eventId` para poder asociar los comentarios de usuario con el evento correspondiente. Hay varias formas de obtener el `eventId`:

- usa {<PlatformLink to="/configuration/options/#before-send"><PlatformIdentifier name="before-send" /></PlatformLink>}
- usa el valor de retorno de cualquier método que capture un evento.
- usa `Sentry.lastEventId` para obtener el ID del último evento enviado.
