El [plugin de Sentry para Dart](https://github.com/getsentry/sentry-dart-plugin) es la forma recomendada de subir símbolos de depuración para aplicaciones Dart. Automatiza la subida para Android, iOS, macOS y web, lo que permite proporcionar a Sentry sin complicaciones los archivos necesarios para la simbolización.
