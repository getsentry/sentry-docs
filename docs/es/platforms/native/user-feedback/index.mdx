Sentry permite recopilar comentarios adicionales cuando un usuario se encuentra con un error. Puedes recopilar comentarios según el método que admita el SDK de tu preferencia.

<div id="user-feedback-api">
  ## API de comentarios de usuario
</div>

La API de comentarios de usuario te permite recopilar comentarios usando tu propia interfaz. Puedes usar el mismo lenguaje de programación que ya utilizas en tu aplicación para enviar los comentarios. En este caso, el SDK crea la solicitud HTTP para que no tengas que encargarte de enviar datos por HTTP.

Sentry asocia los comentarios con el evento original, ofreciéndote información adicional sobre los problemas. Para hacerlo, Sentry necesita el `eventId`. Por ejemplo, para obtener el `eventId`, puedes usar <PlatformLink to="/configuration/options/#before-send"><PlatformIdentifier name="before-send" /></PlatformLink> o el valor de retorno del método que captura un evento.
