<div id="custom-timestamps">
  ## Marcas de tiempo personalizadas
</div>

A veces necesitas medir tiempos en código que no puede invocar directamente funciones del SDK nativo (como shaders de GPU). Para estos casos, puedes usar las siguientes funciones con temporización explícita:

- `sentry_transaction_start_ts`
- `sentry_transaction_finish_ts`
- `sentry_transaction_start_child_ts`
- `sentry_transaction_start_child_ts_n`
- `sentry_span_start_child_ts`
- `sentry_span_start_child_ts_n`
- `sentry_span_finish_ts`

La única diferencia con sus equivalentes sin marca de tiempo es que aceptan un parámetro adicional de marca de tiempo, que es el desplazamiento respecto a la época de Unix en microsegundos.

Al usar estas funciones, asegúrate de que las marcas de tiempo proporcionadas sean consistentes. Se recomienda usar la interfaz normal con marcado de tiempo automático y recurrir a la interfaz con temporización explícita solo cuando sea estrictamente necesario.
