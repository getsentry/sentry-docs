Para permitir que Sentry procese por completo los fallos nativos y te proporcione
trazas de pila simbolizadas, debes subir _archivos de información de depuración_
(a veces también llamados _símbolos de depuración_ o simplemente _símbolos_). Recomendamos
subir la información de depuración durante el proceso de compilación o de lanzamiento.

Para todas las bibliotecas para las que quieras recibir simbolización, **necesitas
proporcionar información de depuración**. Esto incluye dependencias y bibliotecas del sistema operativo.
Si no estás seguro de qué archivos se requieren, ve a _Project
Settings > Processing Issues_, donde verás una lista de todos los archivos necesarios y
las instrucciones para obtenerlos.

Para obtener más información sobre cómo subir información de depuración y sus formatos compatibles,
consulta [Archivos de información de depuración](../data-management/debug-files/).
