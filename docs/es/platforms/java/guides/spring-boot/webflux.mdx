title: Integración con WebFlux
sidebar_order: 6
description: "Obtén más información sobre cómo usar Spring WebFlux con Sentry."
---

Al igual que Spring MVC, Sentry ofrece compatibilidad con Spring WebFlux. Al desarrollar una aplicación web reactiva, Sentry Spring Boot Starter crea la infraestructura necesaria para capturar los errores que se produzcan durante el procesamiento de solicitudes HTTP.

La versión mínima de Spring Boot compatible con la integración de Sentry WebFlux es `v2.2.5.RELEASE`.
