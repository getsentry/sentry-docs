Las opciones de configuración comunes a todas las integraciones de Java, como `environment` y `release`, pueden definirse mediante variables de entorno, propiedades del sistema o el archivo `sentry.properties`. [Consulta la página de configuración](../options) para una explicación detallada de cada propiedad.

Las opciones de configuración específicas de la integración con `java.util.logging` pueden definirse en el archivo `logging.properties`:

```properties
io.sentry.jul.SentryHandler.level=CONFIG
io.sentry.jul.SentryHandler.minimumEventLevel=INFO
io.sentry.jul.SentryHandler.minimumBreadcrumbLevel=CONFIG
io.sentry.jul.SentryHandler.printfStyle=true
```
