Al usar `sentry-opentelemetry-agent` puedes elegir si el agente debe llamar a `Sentry.init`.
De forma predeterminada, el agente inicializará Sentry por ti. Si prefieres inicializar Sentry manualmente o que otra integración haga la inicialización, puedes desactivar este comportamiento.
