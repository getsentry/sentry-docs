También puedes desactivar la inicialización automática de Sentry en `sentry-opentelemetry-agent` estableciendo `SENTRY_AUTO_INIT=false` como variable de entorno. Al hacerlo, tendrás que usar otra integración de Sentry que realice la inicialización (por ejemplo, Spring Boot) o inicializar Sentry manualmente.
