Sentry usa *breadcrumbs* para crear un rastro de eventos que ocurrieron antes de un problema. Estos eventos son muy similares a los registros tradicionales, pero pueden capturar datos estructurados más ricos.

Esta página ofrece una descripción general del registro manual de breadcrumbs y su personalización. Obtén más información sobre lo que se muestra en la página **Detalles del problema** y cómo puedes filtrar los breadcrumbs para resolver incidencias más rápido en [Uso de Breadcrumbs](/es/product/error-monitoring/breadcrumbs).

<Alert title="Más información sobre el uso del SDK">
  Los desarrolladores que quieran modificar la interfaz de breadcrumbs pueden consultar nuestra [documentación para desarrolladores sobre la interfaz de Breadcrumbs](https://develop.sentry.dev/sdk/data-model/event-payloads/breadcrumbs/).
</Alert>
