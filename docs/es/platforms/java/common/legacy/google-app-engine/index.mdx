La biblioteca `sentry-appengine` ofrece compatibilidad con [Google App Engine](https://cloud.google.com/appengine/) para Sentry mediante la [Task Queue API](https://cloud.google.com/appengine/docs/java/taskqueue/).

El código fuente se encuentra [en GitHub](https://github.com/getsentry/sentry-java/tree/master/sentry-appengine).

<div id="installation">
  ### Instalación
</div>

```xml {tabTitle:Maven}
<dependency>
    <groupId>io.sentry</groupId>
    <artifactId>sentry-appengine</artifactId>
    <version>1.7.30</version>
</dependency>
```

```groovy {tabTitle:Gradle}
implementation 'io.sentry:sentry-appengine:1.7.30'
```

```scala {tabTitle: SBT}
libraryDependencies += "io.sentry" % "sentry-appengine" % "1.7.30"
```

Para otros gestores de dependencias, consulta el [repositorio central de Maven](https://search.maven.org/artifact/io.sentry/sentry-appengine).

<div id="usage">
  ### Uso
</div>

Este módulo proporciona una nueva implementación de `SentryClientFactory` que reemplaza el sistema asíncrono predeterminado por uno compatible con Google App Engine. Deberás configurar Sentry para que use `io.sentry.appengine.AppEngineSentryClientFactory` como su factoría.

El tamaño de la cola y las opciones de hilos no se usarán, ya que son específicas del sistema de hilos predeterminado de Java.

<div id="queue-name">
  ### Nombre de la cola
</div>

De forma predeterminada se utilizará la cola de tareas por defecto, pero es posible especificar cuál se usará con la opción `sentry.async.gae.queuename`:

```
{DSN}?async.gae.queuename=MyQueueName
```

<div id="connection-name">
  ### Nombre de la conexión
</div>

Dado que las tareas en cola se envían a distintas instancias de la aplicación, es importante poder identificar qué conexión debe usarse al procesar el evento. Para ello, el módulo de GAE identificará cada conexión mediante un identificador generado automáticamente o definido por el usuario. Para establecer manualmente el identificador de la conexión (se usa solo internamente), utiliza la opción `sentry.async.gae.connectionid`:

```
{DSN}?async.gae.connectionid=MyConnection
```
