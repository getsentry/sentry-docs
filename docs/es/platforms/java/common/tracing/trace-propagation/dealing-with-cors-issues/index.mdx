---
title: Cómo resolver problemas de CORS
sidebar_order: 80
---

Si tu frontend y backend están alojados en dominios diferentes (por ejemplo, tu frontend está en `https://example.com` y tu backend en `https://api.example.com`), y el frontend realiza solicitudes XHR/fetch a tu backend, tendrás que configurar los encabezados [CORS](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Access-Control-Allow-Headers) de tu backend para asegurarte de que las solicitudes no se bloqueen.

Configura CORS en tu backend para permitir los encabezados `sentry-trace` y `baggage`.

La configuración del encabezado de respuesta de tu servidor podría ser: `"Access-Control-Allow-Headers: sentry-trace, baggage"`. Tu configuración dependerá de tu entorno.
