title: Apagado y vaciado
sidebar_order: 70
description: "Obtén más información sobre el comportamiento predeterminado de nuestro SDK si la aplicación se cierra inesperadamente."
---

De forma predeterminada, el SDK envía eventos por la red en un hilo en segundo plano. Esto significa que algunos eventos podrían perderse si la aplicación se cierra inesperadamente. El SDK ofrece mecanismos para manejar esta situación.
