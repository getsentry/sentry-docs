---
title: Instalación manual 
description: "Integra Sentry en tu proyecto de Xcode usando nuestros frameworks precompilados."
---

Para integrar Sentry en tu proyecto de Xcode, sigue estos pasos:

1. Descarga la versión más reciente del SDK desde la [página de lanzamientos](https://github.com/getsentry/sentry-cocoa/releases) de Sentry Cocoa.
2. Cada versión incluye las siguientes cuatro opciones de framework:
    - Sentry-Dynamic.xcframework.zip: framework dinámico
	- Sentry.xcframework.zip: framework estático
	- SentrySwiftUI.xcframework.zip: framework estático con compatibilidad con SwiftUI
	- Sentry-WithoutUIKitOrAppKit.xcframework.zip: framework estático sin vinculación a UIKit o AppKit ni funciones relacionadas
3. Importa el framework elegido en el target de tu proyecto de Xcode.

<div id="usage-guidelines">
  ### Pautas de uso:
</div>

•	Usa `Sentry-Dynamic`, `Sentry` o `Sentry-WithoutUIKitOrAppKit` de forma independiente. Solo uno de estos debe incluirse en tu proyecto a la vez.
	•	Si usas `SentrySwiftUI`, debe combinarse con `Sentry-Dynamic`.
