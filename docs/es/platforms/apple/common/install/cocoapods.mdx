---
title: CocoaPods
description: "Aprende a instalar el SDK de Sentry con CocoaPods."
---

Para integrar Sentry en tu proyecto de Xcode con CocoaPods, especifícalo en tu *Podfile*:

```ruby
platform :ios, '11.0'
use_frameworks! # Esto es importante

target 'YourApp' do
  pod 'Sentry', :git => 'https://github.com/getsentry/sentry-cocoa.git', :tag => '{{@inject packages.version('sentry.cocoa') }}'
end
```

Después, ejecuta `pod install`.

Si usas Xcode 14 o posterior, es posible que tu proyecto no se compile por `error: Sandbox: rsync.samba(42924)`. Para solucionarlo, desactiva la opción &quot;Enable User Script Sandbox&quot; en la configuración del target estableciéndola en `NO`. Consulta <PlatformLink to="/troubleshooting#sandbox-error-using-cocoapods">Solución de problemas</PlatformLink> para más información.
