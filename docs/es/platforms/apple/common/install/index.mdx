Recomendamos instalar el SDK con nuestro [Sentry Wizard](https://github.com/getsentry/sentry-wizard) ejecutando el siguiente comando en el directorio de tu proyecto:

```bash
brew install getsentry/tools/sentry-wizard && sentry-wizard -i ios
```

Esto aplicará un parche a tu proyecto y configurará el SDK. Solo tendrás que hacerlo una vez; después podrás agregar los archivos parcheados a tu sistema de control de versiones.
Si lo prefieres, puedes elegir alguno de los métodos alternativos a continuación.
