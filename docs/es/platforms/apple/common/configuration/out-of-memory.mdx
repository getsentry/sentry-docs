---
title: Falta de memoria
sidebar_order: 10
description: "Aprende a desactivar el seguimiento de falta de memoria"
---

Renombramos esta integración a <PlatformLink to="/configuration/watchdog-terminations/">terminaciones de watchdog</PlatformLink> en la versión 8.0.0. La integración de falta de memoria y la de terminaciones de watchdog son lo mismo. La única diferencia es el nombre.

Si deseas desactivar esta función, puedes hacerlo mediante opciones:

```swift {tabTitle:Swift}
import Sentry

SentrySDK.start { options in
    options.dsn = "___PUBLIC_DSN___"
    options.enableOutOfMemoryTracking = false
}
```

```objc {tabTitle:Objective-C}
@import Sentry;

[SentrySDK startWithConfigureOptions:^(SentryOptions *options) {
    options.dsn = @"___PUBLIC_DSN___";
    options.enableOutOfMemoryTracking = NO;
}];
```
