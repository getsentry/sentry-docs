Todos los SDK de Sentry incluyen integraciones que amplían la funcionalidad del SDK.

Las integraciones del sistema están habilitadas de forma predeterminada para integrarse con la biblioteca estándar o con el propio intérprete. Están documentadas para que sepas qué hacen y puedas deshabilitarlas si causan problemas.

<div id="sentrycrashintegration">
  ### SentryCrashIntegration
</div>

Esta integración es el núcleo del SDK. Se integra con todos los controladores de señales y excepciones para capturar errores o bloqueos no detectados. También es responsable de añadir la mayor parte de la información del dispositivo a los eventos. Si está desactivada, no recibirás informes de bloqueos y los eventos contendrán pocos datos del dispositivo.

<Alert>
  La detección de bloqueos no está disponible en watchOS y las excepciones Mach no se informan en tvOS.
</Alert>

<Alert>
  No se pueden instalar los controladores de señales y excepciones si hay un depurador adjunto. Para probar bloqueos localmente, asegúrate de desmarcar la opción “Debug executable” en la acción Run del esquema de Xcode de tu app.
</Alert>

<div id="sentryautobreadcrumbtrackingintegration">
  ### SentryAutoBreadcrumbTrackingIntegration
</div>

Esta integración hará swizzle de algunos métodos para crear breadcrumbs. Por ejemplo, para `UIApplicationDidReceiveMemoryWarningNotification`, `sendAction:to:from:forEvent:` (interacciones de la UI) o `viewDidAppear:`, esos breadcrumbs se adjuntarán a tus eventos.

<div id="sentryautosessiontrackingintegration">
  ### SentryAutoSessionTrackingIntegration
</div>

Esta integración se registra para notificaciones del ciclo de vida y comienza y finaliza automáticamente las sesiones necesarias para [Release health](/es/product/releases/health/#session).

<div id="sentrywatchdogterminationstrackingintegration">
  ### SentryWatchdogTerminationsTrackingIntegration
</div>

Esta integración registra las terminaciones del watchdog basadas en heurísticas. Esta función está disponible para iOS, tvOS y Mac Catalyst y solo funciona si la aplicación estaba en primer plano. Consulta la <PlatformLink to="/configuration/watchdog-terminations/">información de configuración</PlatformLink> para más detalles.

<div id="disable-an-integration">
  ## Desactivar una integración
</div>

Este ejemplo quita la integración activada de forma predeterminada que agrega breadcrumbs al evento:

```swift {tabTitle:Swift}
import Sentry

SentrySDK.start { options in
    options.integrations = Sentry.Options.defaultIntegrations().filter { name in
        name != "SentryAutoBreadcrumbTrackingIntegration"
    }
}
```

```objc {tabTitle:Objective-C}
@import Sentry;

NSMutableArray *integrations = [SentryOptions defaultIntegrations].mutableCopy;
[integrations removeObject:@"SentryAutoBreadcrumbTrackingIntegration"];
[SentrySDK startWithConfigureOptions:^(SentryOptions *options) {
    options.integrations = integrations;
}];
```
