El SDK de Apple de Sentry para watchOS permite el reporte básico de errores y excepciones, con compatibilidad limitada para la simbolización y sin detección de bloqueos ni cuelgues de la app. A continuación se muestra una lista de las funcionalidades disponibles como parte de este SDK.

**Funcionalidades:**

* Compatibilidad limitada para la simbolización y sin manejo de bloqueos
* Eventos [enriquecidos](/es/platforms/apple/enriching-events/context/) con datos del dispositivo
* Cacheo sin conexión cuando un dispositivo no puede conectarse; enviamos un informe una vez que recibimos otro evento
* Los [Attachments](/es/platforms/apple/enriching-events/attachments/) enriquecen el evento al almacenar archivos adicionales, como archivos de configuración o de registro
* [User Feedback](/es/platforms/apple/user-feedback/) permite recopilar información del usuario cuando ocurre un evento
