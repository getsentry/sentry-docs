El SDK de Apple de Sentry para macOS permite el reporte automático de errores y excepciones, e identifica problemas de rendimiento en tu aplicación. La versión mínima de macOS compatible es la 10.10. A continuación, se muestra una lista de funcionalidades disponibles como parte de este SDK.

<Alert>
  En macOS, se requiere una sencilla [configuración manual](/es/platforms/apple/guides/macos/usage/#capturing-uncaught-exceptions) para que el SDK informe las excepciones no capturadas.
</Alert>

<Alert>
  El SDK crea un informe de bloqueo que persiste en disco. Aunque intenta enviarlo justo después del bloqueo, puede que no siempre sea posible porque el entorno puede estar inestable en ese momento. Si sucede, el informe se enviará al iniciar la aplicación.
</Alert>

**Funcionalidades:**

* Se capturan varios tipos de errores, incluidos:
  * Excepciones Mach
  * Señales fatales
  * Excepciones no controladas
  * Excepciones de C++
  * Excepciones de Objective-C
  * Mensajes de error de fatalError, assert y precondition
  * Bloqueos durante el inicio. La inicialización del SDK espera de forma sincrónica hasta 5 segundos para enviar los eventos si la app se bloquea dentro de los 2 segundos posteriores a la inicialización.
* Eventos [enriquecidos](/es/platforms/apple/enriching-events/context/) con datos del dispositivo
* Caché sin conexión cuando un dispositivo no puede conectarse; enviamos un informe cuando recibimos otro evento
* [Breadcrumbs automáticos](/es/platforms/apple/enriching-events/breadcrumbs/#automatic-breadcrumbs) capturados para
  * Eventos del ciclo de vida de la aplicación (`didBecomeActive`, `didEnterBackground`)
  * Eventos táctiles
  * Eventos del sistema (cambio en nivel o estado de batería, advertencias de memoria, cambio de orientación del dispositivo, teclado mostrado y ocultado, captura de pantalla)
  * Solicitudes HTTP salientes
  {/* <!-- When adding an auto breadcrumb item here please also update /platform-includes/enriching-events/breadcrumbs/automatic-breadcrumbs/apple.mdx --> */}
* [Salud de la versión](/es/product/releases/health/) hace un seguimiento de usuarios y sesiones sin bloqueos
* [Archivos adjuntos](/es/platforms/apple/enriching-events/attachments/) enriquecen tu evento almacenando archivos adicionales, como archivos de configuración o registros
* [Comentarios de usuarios](/es/platforms/apple/user-feedback/) permiten recopilar información del usuario cuando ocurre un evento
* [Contexto de código fuente](/es/platforms/apple/data-management/debug-files/source-context/) muestra fragmentos de código alrededor de la ubicación de los marcos de la pila
