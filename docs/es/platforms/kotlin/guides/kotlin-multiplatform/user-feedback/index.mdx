Cuando un usuario experimenta un error, Sentry permite recopilar comentarios adicionales. Puedes recopilar comentarios según el método compatible con el SDK.

<div id="user-feedback-api">
  ## API de comentarios de usuario
</div>

La API de comentarios de usuario te permite recopilar opiniones mientras usas tu propia interfaz. Puedes emplear el mismo lenguaje de programación de tu app para enviar los comentarios. En este caso, el SDK crea la solicitud HTTP para que no tengas que encargarte de enviar los datos por HTTP.
