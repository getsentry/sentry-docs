Para simbolicar tus trazas de pila, debes proporcionar información de depuración a Sentry.
El proceso de simbolicación recompone las trazas de pila para mostrar la función, los nombres de archivo y los números de línea del error.

<div id="apple">
  ## Apple
</div>

Para aplicaciones de Apple, sigue la documentación de iOS sobre [carga de símbolos de depuración](/es/platforms/apple/guides/ios/dsym) para configurar la subida de símbolos de depuración.

<div id="android">
  ## Android
</div>

<div id="gradle">
  ### Gradle
</div>

Para cargar archivos de mapeo de ProGuard o símbolos de depuración nativos con Gradle, debes instalar el [Sentry Android Gradle Plugin](/es/platforms/android/configuration/gradle) en tu módulo `androidApp`:

```kotlin {filename:androidApp/build.gradle.kts}
plugins {
    id("com.android.application")
    id("io.sentry.android.gradle") version "{{@inject packages.version('sentry.java.android.gradle-plugin', '3.0.0') }}"
}

sentry {
  // Evita que las dependencias de Sentry se incluyan en la app de Android a través del AGP.
  autoInstallation {
    enabled.set(false)
  }
  
  // El slug de la organización de Sentry que se usará para cargar los mapeos de ProGuard y los contextos del código fuente.
  org.set("___ORG_SLUG___")

  // El slug del proyecto de Sentry que se usará para cargar los mapeos de ProGuard y los contextos del código fuente.
  projectName.set("___PROJECT_SLUG___")

  // El token de autenticación que se usará para cargar los mapeos de ProGuard y los contextos del código fuente.
  // ADVERTENCIA: No expongas este token en tus archivos build.gradle; en su lugar, define
  // una variable de entorno y léela en esta propiedad.
  authToken.set(System.getenv("SENTRY_AUTH_TOKEN"))
}
```

Después de instalar el plugin, debes configurar tu token de autenticación como una variable de entorno:

<OrgAuthTokenNote />

```bash
export SENTRY_AUTH_TOKEN=___ORG_AUTH_TOKEN___
```

Una vez configurado, el complemento subirá automáticamente a Sentry los símbolos de depuración necesarios cuando se cree una versión de lanzamiento.
