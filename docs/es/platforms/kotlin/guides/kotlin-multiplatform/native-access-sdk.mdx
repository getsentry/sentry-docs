title: Acceso nativo al SDK
description: "Cómo acceder directamente al SDK sin el módulo compartido."

---

Si necesitas acceder directamente al SDK en tu plataforma nativa sin el módulo compartido, primero debes cambiar la configuración de la dependencia de `implementation(...)` a `api(...)`:

```kotlin {filename:shared/build.gradle.kts}
val commonMain by getting {
  dependencies {
    api("io.sentry:sentry-kotlin-multiplatform:{{@inject packages.version('sentry.kotlin.kmp', '0.0.1-alpha.2') }}")
  }
}
```


<div id="export-the-framework">
  ## Exportar el framework
</div>

Si tienes objetivos de Apple, también necesitas exportar el framework.

<div id="cocoapods">
  ### CocoaPods
</div>

```kotlin {filename:shared/build.gradle.kts}
cocoapods {
  summary = "Descripción del Módulo Compartido"
  homepage = "Enlace a la página de inicio del Módulo Compartido"
  ios.deploymentTarget = "14.1"
  podfile = project.file("../iosApp/Podfile")
  // Asegúrate de usar la versión correcta según nuestra Tabla de Compatibilidad de Versiones del SDK de Cocoa.
  pod("Sentry") {
    version = "~> 8.25"
    linkOnly = true
    extraOpts += listOf("-compiler-option", "-fmodules")
  }
  framework {
    baseName = "shared"
    export("io.sentry:sentry-kotlin-multiplatform:{{@inject packages.version('sentry.kotlin.kmp', '0.0.1-alpha.2') }}")
  }
}
```


<div id="swift-package-manager">
  ### Swift Package Manager
</div>

```kotlin {filename:shared/build.gradle.kts}
listOf(
  iosX64(),
  iosArm64(),
  iosSimulatorArm64()
).forEach {
  it.binaries.framework {
      baseName = "shared"
      isStatic = true
      export("io.sentry:sentry-kotlin-multiplatform:{{@inject packages.version('sentry.kotlin.kmp', '0.0.1-alpha.2') }}")
  }
}
```


<div id="usage">
  ## Uso
</div>

Ahora puedes acceder al SDK directamente desde tu plataforma nativa:

<div id="android">
  ### Android
</div>

```kotlin
import io.sentry.kotlin.multiplatform.Sentry

Sentry.captureMessage("Mi mensaje")
```


<div id="ios">
  ### iOS
</div>

```swift
import shared

Sentry.shared.captureMessage(message: "Mi mensaje")
```
