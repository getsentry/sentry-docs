El SDK Kotlin Multiplatform de Sentry permite el envío automático de errores y excepciones.
Como el SDK proporciona una API unificada para todas las plataformas compatibles y aprovecha directamente los SDK nativos de cada una, también admite las mismas funcionalidades que los SDK nativos. Haz clic en un enlace a continuación para ver las funcionalidades relevantes para tu plataforma:

* [Funcionalidades de Apple](/es/platforms/apple/features)
* [Funcionalidades de Android](/es/platforms/android/features)
* [Funcionalidades de Java](/es/platforms/java)

<Alert>
  Es posible que algunas funcionalidades aún no estén disponibles en la capa compartida de Kotlin Multiplatform. Puede que tengas que usar directamente los SDK nativos o escribir tu propio código específico de la plataforma.

  Para acceder a las opciones de inicialización del SDK nativo, usa la [inicialización de Native Platform Options](/es/platforms/kotlin/guides/kotlin-multiplatform/initialization-strategies/#native-platform-options).
</Alert>

<div id="supported-platforms">
  ## Plataformas compatibles
</div>

La siguiente tabla muestra las plataformas admitidas y sus presets correspondientes.

| Plataforma objetivo | Preset objetivo                                                                                            |
| ------------------- | ---------------------------------------------------------------------------------------------------------- |
| Android             | <ul><li>`androidTarget`</li></ul>                                                                          |
| Kotlin/JVM          | <ul><li>`jvm`</li></ul>                                                                                    |
| iOS                 | <ul><li>`iosArm64`</li><li>`iosX64`</li><li>`iosSimulatorArm64`</li></ul>                                  |
| macOS               | <ul><li>`macosArm64`</li><li>`macosX64`</li></ul>                                                          |
| watchOS             | <ul><li>`watchosArm32`</li><li>`watchosArm64`</li><li>`watchosX64`</li><li>`watchosSimulatorArm64`</li></ul> |
| tvOS                | <ul><li>`tvosArm64`</li><li>`tvosX64`</li><li>`tvosSimulatorArm64`</li></ul>                               |

<div id="stubbed-platforms-no-op-implementations">
  ## Plataformas con stubs (implementaciones no-op)
</div>

Puedes añadir estos targets a tu proyecto; compilarán y cumplirán con la API, pero **no harán nada en tiempo de ejecución**.

| Plataforma de destino | Preajuste de destino                           |
|-----------------------|-------------------------------------------------|
| JS                    | <ul><li>`js`</li></ul>                          |
| Wasm JS               | <ul><li>`wasmJs`</li></ul>                      |
| Linux                 | <ul><li>`linuxx64`</li><li>`linuxarm64`</li></ul> |
| Windows               | <ul><li>`mingwx64`</li></ul>                    |
