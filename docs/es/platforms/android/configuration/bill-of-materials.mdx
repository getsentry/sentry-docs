title: Uso de un BOM
description: "Obtén más información sobre cómo usar una lista de materiales con varias dependencias de Sentry."
sidebar_order: 1000
---

Cuando uses varias dependencias de Sentry, puedes evitar especificar la versión de cada una con un *BOM* o *Bill of Materials*.

Con Gradle 5.0 o posterior, puedes agregar lo siguiente a la sección `dependencies` de tu `build.gradle`:

```groovy {filename:build.gradle}
implementation platform('io.sentry:sentry-bom:{{@inject packages.version('sentry.java.bom', '5.1.0') }}') //importar bom
implementation('io.sentry:sentry-android') //sin versión especificada
implementation('io.sentry:sentry-android-fragment') //sin versión especificada
```
