---
title: Aplicaciones multiproceso
sidebar_order: 6
description: "Aprende a inicializar automáticamente el SDK en procesos que no son el principal."
---

El SDK de Sentry para Android no se inicializa automáticamente en procesos que no son el principal debido a `ContentProvider`. Si quieres habilitar la inicialización automática (`auto-init`) del SDK de Sentry para Android en otros procesos, agrega lo siguiente a `AndroidManifest.xml`:

```xml
<provider
    android:name="io.sentry.android.core.SentryInitProvider"
    android:authorities="${applicationId}.SentryInitProvider"
    android:exported="false"
    android:process="TU_PROCESO_AQUÍ"
    tools:node="merge" />
```

También puedes llamar a `SentryAndroid.init(...)` en cada proceso en el que quieras inicializar Sentry.
