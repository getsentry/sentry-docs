---
title: Versiones mezcladas
description: "Cómo diagnosticar y resolver versiones mezcladas de dependencias del SDK de Android."
sidebar_order: 1000
---

El uso de varias dependencias del SDK de Android con versiones mezcladas no es compatible, ya que es muy probable que provoque un fallo más adelante. Por ello, preferimos que la aplicación se cierre inesperadamente al inicializar el SDK.

<div id="multiple-dependencies-with-mixed-versions">
  ## Varias dependencias con versiones mezcladas
</div>

El siguiente fragmento muestra un conflicto por versiones mezcladas causado por usar `sentry` en la versión `8.6.0` y `sentry-android` en la versión `8.7.0`. Para solucionarlo, establece la misma versión en todas las dependencias o <PlatformLink to="/configuration/bill-of-materials/">usa `sentry-bom`</PlatformLink>. Esto también puede ocurrir si utilizas una biblioteca interna que tiene definida una versión diferente.

```groovy {tabTitle:Gradle}{filename:build.gradle}
implementation('io.sentry:sentry:8.6.0')
implementation('io.sentry:sentry-android:8.7.0')
```

```xml {tabTitle:Maven}{filename:pom.xml}
<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>io.sentry</groupId>
            <artifactId>sentry</artifactId>
            <version>8.6.0</version>
        </dependency>
        <dependency>
            <groupId>io.sentry</groupId>
            <artifactId>sentry-android</artifactId>
            <version>8.7.0</version>
        </dependency>
    </dependencies>
</dependencyManagement>
```


<div id="build-plugin-and-explicit-dependency">
  ## Plugin de compilación y dependencia explícita
</div>

Cuando usas nuestro plugin de Gradle o Maven y defines manualmente dependencias adicionales del SDK de Sentry para Android, puedes terminar con versiones mezcladas. El siguiente fragmento muestra que el plugin está configurado para usar la versión `8.0.0`, pero hay una dependencia adicional establecida en la versión `8.1.0`. Para solucionarlo, configura la misma versión o <PlatformLink to="/configuration/bill-of-materials/">usa `sentry-bom`</PlatformLink>.

```groovy {tabTitle:Gradle}{filename:build.gradle}
plugins { id "io.sentry.android.gradle" version "5.3.0" }

dependencies {
	implementation 'io.sentry:sentry-okhttp:8.1.0'
}

sentry {
	autoInstallation {
		sentryVersion = "8.0.0"
	}
}
```
