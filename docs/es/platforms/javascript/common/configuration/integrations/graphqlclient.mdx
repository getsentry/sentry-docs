<Alert>

  Esta integración solo funciona en un entorno de navegador.

</Alert>

_Nombre de importación: `Sentry.graphqlClientIntegration`_

Esta integración mejora los datos capturados de las solicitudes GraphQL en tu aplicación.
Extrae información específica de GraphQL de las solicitudes HTTP que coincidan con los endpoints que configures y enriquece
tanto los spans como los breadcrumbs con detalles de operaciones de GraphQL.

Cuando está habilitada, esta integración:

- Actualiza los nombres de los spans con el tipo y el nombre de la operación de GraphQL
- Agrega documentos de consulta de GraphQL a los spans
- Agrega información sobre la operación de GraphQL a los breadcrumbs

<div id="options">
  ## Opciones
</div>

<div id="endpoints">
  ### `endpoints`
</div>

*Tipo: `(string|RegExp)[]`*

Una lista de URL o patrones de URL que se deben tratar como endpoints de GraphQL.
La integración solo procesará las solicitudes dirigidas a estos endpoints. Esta lista puede contener cadenas, expresiones regulares o una combinación de ambas.

Ejemplo:

```javascript
Sentry.init({
  integrations: [
    Sentry.graphqlClientIntegration({
      endpoints: [
        'https://graphql-api.example.com',
        /\/graphql$/
      ]
    })
  ]
});
```

Para que **coincidan todos** los endpoints con `graphqlClientIntegration`, debes configurar la opción endpoints como `[/.*/]` en tu configuración.
Este patrón de regex coincidirá con cualquier URL.
