*Nombre de importación: `Sentry.googleCloudHttpIntegration`*

Esta integración está habilitada de forma predeterminada. Si deseas modificar las integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Añade instrumentación para APIs RESTful en Google Cloud para crear spans automáticamente.

```JavaScript
Sentry.init({
  integrations: [Sentry.googleCloudHttpIntegration()],
});
```
