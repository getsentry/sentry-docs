*Nombre de importación: `Sentry.awsIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Añade instrumentación para aws-sdk v2 y v3 para crear spans automáticamente. Impulsado por [`@opentelemetry/instrumentation-aws-sdk`](https://www.npmjs.com/package/@opentelemetry/instrumentation-aws-sdk).

```JavaScript
import * as Sentry from '@sentry/aws-serverless';

Sentry.init({
  integrations: [Sentry.awsIntegration()],
});
```
