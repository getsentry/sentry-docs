<Alert>
  Esta integración solo funciona en el entorno de ejecución de Node.js.
</Alert>

`nodeProfilingIntegration` debe importarse del paquete `@sentry/profiling-node`.

El [perfilado](/es/product/explore/profiling/) ofrece un nivel más profundo de visibilidad que el trazado tradicional, elimina la necesidad de instrumentación personalizada y habilita una visibilidad precisa a nivel de código en tu aplicación en un entorno de producción.

La integración NodeProfiling configura el perfilado automático del rendimiento para tus aplicaciones de Node.js. Captura perfiles mediante V8 y los envía a Sentry. Para usar esta integración, también necesitas tener habilitado el monitoreo de rendimiento.

Obtén más información sobre [cómo configurar NodeProfiling](./../../../profiling/).

```JavaScript {tabTitle:CJS}
const Sentry = require("@sentry/node");
const { nodeProfilingIntegration } = require("@sentry/profiling-node");

Sentry.init({
  integrations: [nodeProfilingIntegration()],
});
```

```JavaScript {tabTitle:ESM}
import * as Sentry from "@sentry/node";
import { nodeProfilingIntegration } from "@sentry/profiling-node";

Sentry.init({
  integrations: [nodeProfilingIntegration()],
});
```
