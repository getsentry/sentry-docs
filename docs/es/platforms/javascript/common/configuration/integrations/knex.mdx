<Alert>
  Esta integración solo funciona en los entornos de ejecución Node.js y Bun. Requiere la versión `8.38.0` o posterior del SDK.
</Alert>

*Nombre de la importación: `Sentry.knexIntegration`*

La integración `knexIntegration` añade instrumentación para la librería `knex` y captura spans mediante [`@opentelemetry/instrumentation-knex`](https://www.npmjs.com/package/@opentelemetry/instrumentation-knex).

```javascript
Sentry.init({
  integrations: [Sentry.knexIntegration()],
});
```
