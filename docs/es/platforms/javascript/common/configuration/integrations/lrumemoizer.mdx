<Alert>
  Esta integración solo funciona en los entornos de ejecución (runtimes) de Node.js y Bun. Requiere la versión `8.33.0` o posterior del SDK.
</Alert>

*Nombre de la importación: `Sentry.lruMemoizerIntegration`*

Esta integración se habilita de forma predeterminada cuando está activado el monitoreo de rendimiento. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `lruMemoizerIntegration` añade instrumentación para la biblioteca `lru-memoizer` a fin de capturar spans usando [`@opentelemetry/instrumentation-lru-memoizer`](https://www.npmjs.com/package/@opentelemetry/instrumentation-lru-memoizer).

```javascript
Sentry.init({
  integrations: [Sentry.lruMemoizerIntegration()],
});
```
