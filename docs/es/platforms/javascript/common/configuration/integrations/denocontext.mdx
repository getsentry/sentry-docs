---
title: DenoContext
description: "Captura contexto sobre el entorno y el dispositivo en el que se ejecuta el cliente y lo añade a los eventos. (predeterminado)"
supported:
  - javascript.deno
---

<Alert>
  Esta integración solo funciona en el entorno de ejecución de Deno.
</Alert>

*Nombre de importación: `Sentry.denoContextIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Añade contexto relacionado con Deno a los eventos. Esto incluye contextos sobre `app`, `device`, `os`, `v8` y `TypeScript`.

```JavaScript
Sentry.init({
  integrations: [Sentry.denoContextIntegration()],
});
```
