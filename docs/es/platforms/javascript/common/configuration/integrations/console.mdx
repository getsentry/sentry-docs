*Nombre de importación: `Sentry.consoleIntegration`*

Esta integración está habilitada de manera predeterminada. Si deseas modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `consoleIntegration` genera breadcrumbs a partir de los registros de la consola.

```JavaScript
Sentry.init({
  integrations: [Sentry.consoleIntegration()],
});
```
