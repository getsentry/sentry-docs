Para obtener más información sobre cómo configurar la compatibilidad de Sentry con Nest.js, consulta la [documentación de Sentry para Nest.js](/es/platforms/javascript/guides/nestjs/).

<Alert>
  Esta integración solo funciona en los runtimes de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.nestIntegration`*

Esta integración se habilita de forma predeterminada cuando está activado el monitoreo de rendimiento. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `nestIntegration` añade instrumentación para el framework Nest.js a fin de capturar spans mediante [`@opentelemetry/instrumentation-nestjs-core`](https://www.npmjs.com/package/@opentelemetry/instrumentation-nestjs-core).

```JavaScript
Sentry.init({
  integrations: [Sentry.nestIntegration()],
});
```

<div id="supported-versions">
  ## Versiones compatibles
</div>

* Nest.js `>=4.0.0`
