<Alert>
  Esta integración solo funciona en los entornos de ejecución (runtimes) de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.mysqlIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `mysqlIntegration` añade instrumentación para la biblioteca `mysql` y captura spans usando [`@opentelemetry/instrumentation-mysql`](https://www.npmjs.com/package/@opentelemetry/instrumentation-mysql).

```JavaScript
Sentry.init({
  integrations: [Sentry.mysqlIntegration()],
});
```
