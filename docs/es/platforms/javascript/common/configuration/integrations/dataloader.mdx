<Alert>
  Esta integración solo funciona en los runtimes de Node.js y Bun. Requiere la versión `8.31.0` o posterior del SDK.
</Alert>

*Nombre de importación: `Sentry.dataloaderIntegration`*

La `dataloaderIntegration` añade instrumentación para la librería `dataloader` a fin de capturar spans usando [`@opentelemetry/instrumentation-dataloader`](https://www.npmjs.com/package/@opentelemetry/instrumentation-dataloader).

```javascript
Sentry.init({
  integrations: [Sentry.dataloaderIntegration()],
});
```
