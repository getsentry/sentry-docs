Para más información sobre cómo configurar el soporte de Sentry para Hapi, consulta la [documentación de Hapi para Sentry](/es/platforms/javascript/guides/hapi/).

<Alert>
  Esta integración solo funciona en los entornos de ejecución (runtimes) de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.hapiIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `hapiIntegration` añade instrumentación para el framework Hapi para capturar spans usando [`@opentelemetry/instrumentation-hapi`](https://www.npmjs.com/package/@opentelemetry/instrumentation-hapi).

```JavaScript
Sentry.init({
  integrations: [Sentry.hapiIntegration()],
});
```

<div id="supported-versions">
  ## Versiones compatibles
</div>

* Hapi: `>=17.0.0 <22`
