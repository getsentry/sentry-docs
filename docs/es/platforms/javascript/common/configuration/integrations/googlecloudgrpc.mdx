*Nombre de importación: `Sentry.googleCloudGrpcIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar las integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Añade instrumentación para las APIs gRPC en Google Cloud para crear spans automáticamente.

```JavaScript
Sentry.init({
  integrations: [Sentry.googleCloudGrpcIntegration()],
});
```
