<Alert>
  Esta integración solo funciona en un entorno de navegador.
</Alert>

*Nombre de importación: `Sentry.browserSessionIntegration`*

La función [Release Health](/es/product/releases/health/) de Sentry te permite seguir la adopción por parte de los usuarios y la tasa de funcionamiento sin fallos de tu aplicación.
Cuando la integración BrowserSession está habilitada, crea automáticamente una sesión cada vez que un usuario carga tu página o aplicación. Estas sesiones se utilizan para medir la salud de tu versión. Si se captura un error durante una sesión activa, la sesión se marcará como defectuosa.
Obtén más información sobre <PlatformLink to="/configuration/releases/#sessions">sesiones</PlatformLink>.

```JavaScript
Sentry.init({
  integrations: [Sentry.browserSessionIntegration()],
});
```
