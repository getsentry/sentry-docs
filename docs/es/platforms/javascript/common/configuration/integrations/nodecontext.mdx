<Alert>
  Esta integración solo funciona en el entorno de ejecución de Node.js.
</Alert>

*Nombre de importación: `Sentry.nodeContextIntegration`*

Esta integración está habilitada de forma predeterminada. Si deseas modificar las integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `nodeContextIntegration` recopila información sobre el entorno y el dispositivo en el que se ejecuta el SDK y la adjunta a los eventos.

```JavaScript
Sentry.init({
  integrations: [Sentry.nodeContextIntegration()],
});
```


<div id="options">
  ## Opciones
</div>

<div id="app">
  ### `app`
</div>

_Tipo: `boolean`_

Si se establece en false, no se capturará ningún contexto de la aplicación.

<div id="os">
  ### `os`
</div>

_Tipo: `boolean`_

Si se establece en `false`, no se capturará ningún contexto del sistema operativo.

<div id="device">
  ### `device`
</div>

_Tipo: `boolean | { cpu?: boolean; memory?: boolean; }`_

Si se establece en false, no se capturará ningún contexto del dispositivo.

<div id="culture">
  ### `culture`
</div>

_Tipo: `boolean`_

Si se establece en `false`, no se capturará ningún contexto cultural.

<div id="cloudresource">
  ### `cloudResource`
</div>

_Tipo: `boolean`_

Si se establece en `false`, no se capturará ningún contexto de recursos en la nube.
