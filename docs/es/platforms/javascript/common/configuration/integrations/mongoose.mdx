<Alert>
  Esta integración solo funciona en los entornos de ejecución de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.mongooseIntegration`*

Esta integración se activa de forma predeterminada cuando el monitoreo de rendimiento está habilitado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `mongooseIntegration` añade instrumentación para la biblioteca `mongoose` para capturar spans mediante [`@opentelemetry/instrumentation-mongoose`](https://www.npmjs.com/package/@opentelemetry/instrumentation-mongoose).

```JavaScript
Sentry.init({
  integrations: [Sentry.mongooseIntegration()],
});
```


<div id="supported-versions">
  ## Versiones compatibles
</div>

- `mongoose`: `>=5.9.7 <9`
