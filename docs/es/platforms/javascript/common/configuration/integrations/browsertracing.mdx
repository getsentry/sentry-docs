<Alert>
  Esta integración solo funciona en un entorno de navegador.
</Alert>

*Nombre de importación: `Sentry.browserTracingIntegration`*

Con [tracing](/es/product/insights/overview/), Sentry sigue el rendimiento de tu software, midiendo métricas como el throughput y la latencia, y mostrando el impacto de los errores en múltiples sistemas. Sentry captura trazas distribuidas compuestas por transacciones y spans, que miden servicios individuales y las operaciones dentro de esos servicios.

La integración BrowserTracing configura el trazado automático para tus aplicaciones de frontend. Captura transacciones y spans desde el navegador y las envía a Sentry.

Obtén más información sobre [cómo configurar BrowserTracing](./../../../tracing/).

```JavaScript
Sentry.init({
  integrations: [Sentry.browserTracingIntegration()],
});
```

Consulta las <PlatformLink to="/tracing/instrumentation/automatic-instrumentation/#configuration-options">opciones de configuración</PlatformLink> para ver la lista completa de opciones disponibles para `browserTracingIntegration`.
