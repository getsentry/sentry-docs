<Alert level="info">

Esta integración solo funciona en un entorno de navegador. Solo está disponible mediante una instalación basada en paquetes (por ejemplo, `npm` o `yarn`).

</Alert>

La integración de [Unleash](https://www.getunleash.io/) realiza el seguimiento de las evaluaciones de feature flags generadas por el SDK de Unleash. Estas evaluaciones se mantienen en memoria y se envían a Sentry en eventos de error y de transacción. **Por el momento, solo admitimos evaluaciones de flags booleanos del método isEnabled de Unleash.** Esta integración está disponible en **versiones 8.51.0 o posteriores** del SDK de Sentry.

_Nombres de importación: `Sentry.unleashIntegration`_

Antes de usar esta integración, debes instalar e instrumentar Unleash en tu aplicación. Obtén más información en la [referencia del SDK de Unleash](https://docs.getunleash.io/reference/sdks/javascript-browser) y la [guía de inicio rápido](https://docs.getunleash.io/quickstart).
