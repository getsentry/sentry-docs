<Alert level="info">

Esta integración solo funciona en un entorno de navegador. Solo está disponible mediante instalación desde un gestor de paquetes (por ejemplo, `npm` o `yarn`).

</Alert>

La integración de [Statsig](https://www.statsig.com/) rastrea las evaluaciones de flags de funciones generadas por el SDK Statsig JavaScript Client. Estas evaluaciones se mantienen en memoria y se envían a Sentry cuando hay eventos de error y de transacción. **Por el momento, solo admitimos evaluaciones booleanas de flags mediante el método `checkGate` de Statsig**. Más información sobre los [feature gates de Statsig](https://docs.statsig.com/feature-flags/working-with/).

Esta integración está disponible en las **versiones 9.0.0 y posteriores** del SDK de Sentry, o **versiones 8.55.0 y posteriores para v8.**

_Nombre de importación: `Sentry.statsigIntegration`_
