<Alert>

Esta integración solo funciona en un entorno de navegador. Solo está disponible mediante instalación basada en paquetes (por ejemplo, `npm` o `yarn`).

</Alert>

La integración de [OpenFeature](https://openfeature.dev/) realiza el seguimiento de las evaluaciones de flags de funcionalidades generadas por el SDK de OpenFeature. Este SDK es compatible con una amplia variedad de proveedores de feature flagging. Para ver la lista completa, visita la [página del ecosistema de OpenFeature](https://openfeature.dev/ecosystem/?instant_search%5BrefinementList%5D%5Btype%5D%5B0%5D=Provider&instant_search%5BrefinementList%5D%5BallTechnologies%5D%5B0%5D=JavaScript).

Las evaluaciones de flags se mantienen en memoria y se envían a Sentry en eventos de error y de transacción. **Por el momento, solo admitimos evaluaciones de flags booleanos.** Esta integración está disponible en
las **versiones 8.43.0 o posteriores** del SDK de Sentry.

_Nombre de importación: `Sentry.openFeatureIntegration` y `Sentry.OpenFeatureIntegrationHook`_
