<Alert>
  Esta integración solo funciona en el entorno de ejecución de Deno.
</Alert>

*Nombre de importación: `Sentry.denoCronIntegration`*

[Sentry Crons](/es/product/crons/) te permite supervisar la disponibilidad y el rendimiento de cualquier tarea programada y recurrente en tu aplicación.

La integración DenoCron configura la supervisión automática de tus trabajos cron creados con [`Deno.cron`](https://docs.deno.com/deploy/kv/manual/cron). Captura check-ins y los envía a Sentry.

Obtén más información sobre [cómo configurar Cron Monitoring para Deno](./../../../crons/).

```JavaScript
Sentry.init({
  integrations: [Sentry.denoCronIntegration()],
});
```
