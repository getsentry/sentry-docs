  Esta integración solo funciona en el entorno de ejecución de Node.js.
</Alert>

*Nombre de importación: `Sentry.anrIntegration`*

La `anrIntegration` captura errores de Application Not Responding (ANR)/bloqueo del bucle de eventos y los reporta como eventos de Sentry. Para más detalles, consulta la documentación sobre [detección de bloqueo del bucle de eventos](../../event-loop-block).

```JavaScript
Sentry.init({
  integrations: [Sentry.anrIntegration({ captureStackTrace: true })],
});
```
