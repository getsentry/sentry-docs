<Alert>

Esta integración solo funciona en entornos de navegador. Solo está disponible mediante instalación basada en paquetes (por ejemplo, `npm` o `yarn`).

</Alert>

La integración de [LaunchDarkly](https://launchdarkly.com/) realiza el seguimiento de las evaluaciones de flags de funcionalidades producidas por el SDK de LaunchDarkly. Estas evaluaciones se mantienen en memoria y se envían a Sentry en eventos de error y de transacción. **Por el momento, solo admitimos evaluaciones de flags booleanos.** Esta integración está disponible en versiones del SDK de Sentry **8.43.0 o posteriores.**

_Nombres de importación: `Sentry.launchDarklyIntegration` y `Sentry.buildLaunchDarklyFlagUsedHandler`_

<PlatformContent includePath="/configuration/launchdarkly" />

Visita el sitio web de Sentry y confirma que tu evento de error registró el flag de funcionalidad "test-flag" y su valor "false".
