Para obtener más información sobre cómo configurar la compatibilidad con Sentry Connect, consulta la [documentación de Sentry para Connect](/es/platforms/javascript/guides/connect/).

<Alert>
  Esta integración solo funciona en los runtimes de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.connectIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si deseas modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

`connectIntegration` agrega instrumentación para el framework Connect para capturar spans con [`@opentelemetry/instrumentation-connect`](https://www.npmjs.com/package/@opentelemetry/instrumentation-connect).

```JavaScript
Sentry.init({
  integrations: [Sentry.connectIntegration()],
});
```

<div id="supported-versions">
  ## Versiones compatibles
</div>

* `connect`: `^3.0.0`
