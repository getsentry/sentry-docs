_Nombre de importación: `Sentry.contextLinesIntegration`_

Esta integración agrega el código fuente del JavaScript inline del HTML de la página actual (por ejemplo, JS en etiquetas `<script>`) a los stack traces de los errores capturados.
_No_ puede recopilar el código fuente de recursos referenciados por tu HTML (por ejemplo, `<script src="..." />`).

La integración `ContextLines` es útil cuando tienes JS inline en páginas HTML a las que el backend de Sentry no puede acceder, por ejemplo, porque la página está protegida por inicio de sesión.

  ### `frameContextLines`

_Tipo: `number`_

La cantidad de líneas que se recopilan alrededor del número de línea de cada fotograma de la pila. De forma predeterminada, es 7.
