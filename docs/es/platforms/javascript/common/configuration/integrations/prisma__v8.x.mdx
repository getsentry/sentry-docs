<Alert>

Esta integración solo funciona en los entornos de ejecución de Node.js y Bun.

</Alert>

_Nombre de importación: `Sentry.prismaIntegration`_

Sentry admite el rastreo de consultas de [Prisma ORM](https://www.prisma.io/) con la integración de Prisma.

La integración de Prisma crea spans para cada consulta y los reporta a Sentry con detalles relevantes en `description`, si están disponibles.

<div id="prisma-version-6">
  ## Prisma versión 6
</div>

La integración de Sentry con Prisma incluye compatibilidad con Prisma versión 5 de forma predeterminada. Para que sea compatible con Prisma versión 6, debemos pasar una versión específica de la instrumentación de Prisma a la integración de Sentry para Prisma.

Para usar la integración con Prisma versión 6, primero instala el paquete `@prisma/instrumentation` en su versión 6 (idealmente la misma versión exacta que tus paquetes `prisma` y `@prisma/client`).

Luego, agrega `prismaIntegration` a tu inicialización de Sentry de la siguiente manera:

```javascript {1,6-9}
import { PrismaInstrumentation } from "@prisma/instrumentation";

Sentry.init({
  tracesSampleRate: 1.0,
  integrations: [
    Sentry.prismaIntegration({
      // Sobrescribe la instrumentación predeterminada que utiliza Sentry
      prismaInstrumentation: new PrismaInstrumentation(),
    }),
  ],
});
```


<div id="prisma-version-5">
  ## Prisma versión 5
</div>

Para configurar la integración en Prisma versión 5, primero añade la bandera de característica `tracing` al bloque `generator` de tu esquema de Prisma:

```txt {tabTitle: Prisma Schema} {filename: schema.prisma} {3}
generator client {
  provider        = "prisma-client-js"
  previewFeatures = ["tracing"]
}
```

Luego, agrega `prismaIntegration` a la inicialización de Sentry de la siguiente manera:

```javascript {3}
Sentry.init({
  tracesSampleRate: 1.0,
  integrations: [Sentry.prismaIntegration()],
});
```

<div id="prismainstrumentation">
  ### `prismaInstrumentation`
</div>

_Tipo: `Instrumentation`_ (un tipo de OpenTelemetry)

Sustituye la instrumentación que utiliza el SDK de Sentry por la instancia de instrumentación proporcionada.
