<Alert>

Esta integración solo funciona en un entorno de navegador.

</Alert>

_Nombre de importación: `Sentry.moduleMetadataIntegration`_

Los metadatos pueden inyectarse mediante los plugins del empaquetador de Sentry usando la opción de configuración `_experiments.moduleMetadata`.
Una vez que se agrega esta integración, los metadatos que se pasen al plugin del empaquetador se añadirán a los frames de la pila de todos los eventos bajo la propiedad `module_metadata`. Esto puede usarse para ayudar a etiquetar o enrutar eventos de distintos equipos o fuentes.
