<Alert>
  Esta integración solo funciona en entornos de navegador.
</Alert>

*Nombre de importación: `Sentry.replayCanvasIntegration`*

La integración Replay Canvas puede utilizarse para capturar [reproducciones de sesión](/es/product/session-replay/) de elementos canvas en HTML. Requiere que la integración Replay esté habilitada.

<Alert level="warning">
  ¡Actualmente no se realiza limpieza de PII en las grabaciones de canvas!
</Alert>

Obtén más información sobre [cómo configurar Session Replay con la integración de Canvas](./../../../session-replay#canvas-recording).

```JavaScript
Sentry.init({
  integrations: [Sentry.replayCanvasIntegration()],
});
```
