*Nombre de importación: `Sentry.functionToStringIntegration`*

Esta integración está habilitada de forma predeterminada. Si deseas modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración permite que el SDK proporcione los nombres originales de funciones y métodos, incluso cuando dichas funciones o métodos estén envueltos por nuestros manejadores de errores o breadcrumbs.

```JavaScript
Sentry.init({
  integrations: [Sentry.functionToStringIntegration()],
});
```
