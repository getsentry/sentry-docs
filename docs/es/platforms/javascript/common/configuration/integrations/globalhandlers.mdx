<Alert>
  Esta integración solo funciona en los entornos de Browser y Deno.
</Alert>

*Nombre de importación: `Sentry.globalHandlersIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración adjunta controladores globales para capturar excepciones no controladas y rechazos no controlados. Captura errores y rechazos de promesas no controlados de forma predeterminada.

```JavaScript
Sentry.init({
  integrations: [Sentry.globalHandlersIntegration({ onerror: true, onunhandledrejection: true })],
});
```


<div id="options">
  ## Opciones
</div>

<div id="onerror">
  ### `onerror`
</div>

_Tipo: `boolean`_

Captura los errores que se propagan hasta `onerror`.

<div id="onunhandledrejection">
  ### `onunhandledrejection`
</div>

_Tipo: `boolean`_

Captura rechazos de promesas no controlados.
