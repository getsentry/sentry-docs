<Alert>
  Esta integración solo funciona en el runtime de Node.js. Requiere la versión `10.18.0` o superior del SDK.
</Alert>

*Nombre de importación: `Sentry.pinoIntegration`*

La `pinoIntegration` añade instrumentación para la biblioteca `pino` de modo que las llamadas al logger de Pino se capturen como logs. Opcionalmente, puedes capturar las llamadas al logger de Pino como errores.

```JavaScript
Sentry.init({
  integrations: [Sentry.pinoIntegration()],
});
```


<div id="options">
  ## Opciones
</div>

<div id="error">
  ### `error`
</div>

Configura cómo se capturan los logs de pino como errores de Sentry.

<div id="levels">
  #### `levels`
</div>

- Tipo: `Array<'trace' | 'debug' | 'info' | 'warn' | 'error' | 'fatal'>`
- Predeterminado: `[]`

Niveles que activan la captura de eventos. Cuando se registra un mensaje de log de pino en uno de estos niveles, se capturará como un evento de error de Sentry.

<div id="handled">
  #### `handled`
</div>

- Tipo: `boolean`
- Valor predeterminado: `true`

De manera predeterminada, Sentry marcará los errores capturados como gestionados. Configúralo en `false` si quieres marcarlos como no gestionados.

<div id="log">
  ### `log`
</div>

Configura cómo se registran los logs de pino en Sentry.

<div id="levels">
  #### `levels`
</div>

- Tipo: `Array<'trace' | 'debug' | 'info' | 'warn' | 'error' | 'fatal'>`
- Valor predeterminado: `["trace", "debug", "info", "warn", "error", "fatal"]`

Niveles que activan la captura de logs. Los logs solo se capturan si `enableLogs` está activado en tu configuración de Sentry.

<div id="supported-versions">
  ## Versiones compatibles
</div>

- `pino`: `>=8.0.0 <10`

<div id="examples">
  ## Ejemplos
</div>

<div id="send-only-info-error-and-warn-log-levels-to-sentry">
  ### Enviar solo los niveles de log info, error y warn a Sentry
</div>

```js
Sentry.init({
  integrations: [
    Sentry.pinoIntegration({ log: { levels: ["info", "warn", "error"] } }),
  ],
});
```


<div id="send-errors-to-sentry">
  ### Enviar errores a Sentry
</div>

```js
Sentry.init({
  integrations: [
    Sentry.pinoIntegration({ error: { levels: ["warn", "error"] } }),
  ],
});
```
