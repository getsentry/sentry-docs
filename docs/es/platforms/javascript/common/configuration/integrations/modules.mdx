<Alert>
  Esta integración solo funciona en entornos de servidor (Node.js, Bun, Deno).
</Alert>

*Nombre de importación: `Sentry.modulesIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar las integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `modulesIntegration` captura información sobre los módulos/paquetes de Node instalados en el evento.

```JavaScript
Sentry.init({
  integrations: [Sentry.modulesIntegration()],
});
```
