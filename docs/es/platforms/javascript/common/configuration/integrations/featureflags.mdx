---
title: Integración genérica de feature flags
description: "Aprende a adjuntar datos personalizados de feature flags a eventos de error de Sentry."
---

La integración de feature flags te permite realizar un seguimiento manual de las evaluaciones de flags mediante una API.
Estas evaluaciones se mantienen en memoria y se envían a Sentry en eventos de error y de transacción.
**Por el momento, solo admitimos evaluaciones booleanas de flags.**

_Nombres de importación: `Sentry.featureFlagsIntegration` y `type Sentry.FeatureFlagsIntegration`_

<PlatformContent includePath="/configuration/featureflags" />

Visita el sitio web de Sentry y confirma que tu evento de error ha registrado el feature flag "test-flag" y su valor "false".
