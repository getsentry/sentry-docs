Añade instrumentación de errores y spans para aplicaciones Vue.

<div id="manual-setup">
  ## Configuración manual
</div>

Si la aplicación de Vue no está definida desde el principio, puedes añadir el monitoreo de errores específicos de Vue más adelante.
Para añadir manualmente la integración en aplicaciones de Vue definidas más tarde:

```javascript {filename:main.js}
import * as Sentry from "@sentry/vue";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  // Filtrar la integración `Vue` por defecto
  integrations: (integrations) =>
    integrations.filter((integration) => integration.name !== "Vue"),
});

// Más tarde
const app = createApp({
  template: "<div>hola</div>",
});

Sentry.addIntegration(Sentry.vueIntegration({ app }));
```
