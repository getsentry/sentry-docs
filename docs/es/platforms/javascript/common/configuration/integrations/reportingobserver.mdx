<Alert>

Esta integración solo funciona en un entorno de navegador.

</Alert>

_Nombre de importación: `Sentry.reportingObserverIntegration`_

Esta integración se conecta a la [Reporting API](https://developer.mozilla.org/en-US/docs/Web/API/Reporting_API) y envía a Sentry los eventos capturados. Se puede configurar para gestionar únicamente tipos específicos de incidencias.

<PlatformContent includePath="configuration/reporting-observer" />

<div id="options">
  ## Opciones
</div>

<div id="types">
  ### `types`
</div>

_Tipo: `('crash'|'deprecation'|'intervention')[]`_

Solo procesa los tipos de problemas especificados. De forma predeterminada, se procesan todos los tipos de problemas.
