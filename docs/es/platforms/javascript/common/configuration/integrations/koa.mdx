Para obtener más información sobre cómo configurar Sentry para Koa, consulta la [documentación de Koa para Sentry](/es/platforms/javascript/guides/koa/).

<Alert>
  Esta integración solo funciona en los entornos de ejecución (runtimes) Node.js y Bun.
</Alert>

*Nombre de la importación: `Sentry.koaIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La integración `koaIntegration` añade instrumentación para el framework Koa para capturar spans mediante [`@opentelemetry/instrumentation-koa`](https://www.npmjs.com/package/@opentelemetry/instrumentation-koa).

```JavaScript
Sentry.init({
  integrations: [Sentry.koaIntegration()],
});
```

<div id="options">
  ## Opciones
</div>

<div id="ignorelayerstype">
  ### `ignoreLayersType`
</div>

Requiere la versión `9.29.0` o superior del SDK.

*Tipo: `Array<'middleware' | 'router'>`*

Una matriz de tipos de capas de Koa que se ignorarán al crear spans. Esto te permite filtrar tipos específicos de spans de middleware o router para que no se envíen a Sentry.

```javascript
// Para ignorar spans de middleware
const Sentry = require('@sentry/node');

Sentry.init({
  integrations: [
    Sentry.koaIntegration({
      ignoreLayersType: ['middleware']
    })
  ],
})
```

<div id="supported-versions">
  ## Versiones compatibles
</div>

* `koa`: `^2.0.0`
* `@koa/router`: `>=8`
