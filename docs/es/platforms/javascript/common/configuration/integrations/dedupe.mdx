_Nombre de importación: `Sentry.dedupeIntegration`_

Esta integración está habilitada de forma predeterminada. Si deseas modificar las integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración deduplica ciertos eventos y puede ser útil si estás recibiendo muchos errores duplicados. Ten en cuenta que Sentry solo compara las trazas de pila y las huellas digitales.
