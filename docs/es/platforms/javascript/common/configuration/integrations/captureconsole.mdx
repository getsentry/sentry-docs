_Nombre de importación: `Sentry.captureConsoleIntegration`_

Esta integración captura todas las llamadas a la API de Console y las redirige a Sentry usando las llamadas del SDK `captureMessage` o `captureException`, según el nivel de log. Luego las vuelve a emitir para preservar el comportamiento nativo predeterminado:

  ## Opciones
</div>

<div id="levels">
  ### `levels`
</div>

_Tipo: `string[]`_

Arreglo de métodos que deben capturarse.
Por defecto: `['log', 'info', 'warn', 'error', 'debug', 'assert']`
