<Alert>
  Esta integración solo funciona en los entornos de ejecución de Node.js y Bun. Requiere la versión `8.30.0` o superior del SDK.
</Alert>

*Nombre de importación: `Sentry.kafkaIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `kafkaIntegration` agrega instrumentación para la biblioteca `kafkajs` con el fin de capturar spans usando [`@opentelemetry/instrumentation-kafkajs`](https://www.npmjs.com/package/@opentelemetry/instrumentation-kafkajs).

```JavaScript
Sentry.init({
  integrations: [Sentry.kafkaIntegration()],
});
```
