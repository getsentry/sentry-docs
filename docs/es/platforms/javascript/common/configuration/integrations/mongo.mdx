<Alert>
  Esta integración solo funciona en los entornos de ejecución (runtimes) de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.mongoIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `mongoIntegration` agrega instrumentación a la biblioteca `mongodb` para capturar spans usando [`@opentelemetry/instrumentation-mongodb`](https://www.npmjs.com/package/@opentelemetry/instrumentation-mongodb).

```JavaScript
Sentry.init({
  integrations: [Sentry.mongoIntegration()],
});
```


<div id="supported-versions">
  ## Versiones compatibles
</div>

- `mongodb`: `>=3.3 <7`
