<Alert>

Esta integración solo funciona en los entornos de ejecución de Node.js y Bun.

</Alert>

_Nombre de importación: `Sentry.prismaIntegration`_

Sentry admite el trazado de consultas de [Prisma ORM](https://www.prisma.io/) con la integración de Prisma.

La integración de Prisma crea spans para cada consulta y los envía a Sentry con los detalles relevantes en la `description`, si están disponibles.

Esta integración está habilitada de forma predeterminada y es compatible con Prisma 5 y 6. En Prisma v5, debes seguir las instrucciones a continuación para habilitar el trazado.

Si quieres aprender a modificar tus integraciones predeterminadas, visita la documentación en <PlatformLink to="/configuration/integrations/#modifying-default-integrations">Modificar integraciones predeterminadas</PlatformLink>.

## Versión de Prisma &gt;=6

Para usar la integración con Prisma &gt;=6, no se necesita configuración: el tracing en Prisma v6 está habilitado de forma predeterminada.

```javascript {3}
Sentry.init({
  tracesSampleRate: 1.0,
  integrations: [Sentry.prismaIntegration()],
});
```


<div id="prisma-version-5">
  ## Prisma versión 5
</div>

Para configurar la integración en Prisma versión 5, primero agrega el indicador de característica `tracing` al bloque `generator` de tu esquema de Prisma:

```txt {tabTitle: Prisma Schema} {filename: schema.prisma} {3}
generator client {
  provider        = "prisma-client-js"
  previewFeatures = ["tracing"]
}
```

Luego, `prismaIntegration` capturará automáticamente spans de las consultas de Prisma.

```javascript {3}
Sentry.init({
  tracesSampleRate: 1.0,
  integrations: [Sentry.prismaIntegration()],
});
```
