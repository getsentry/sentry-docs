*Nombre de la importación: `Sentry.awsLambdaIntegration`*

Esta integración viene habilitada de forma predeterminada. Si deseas modificar las integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Añade instrumentación para AWS Lambda para crear spans automáticamente. Con tecnología de [`@opentelemetry/instrumentation-aws-lambda`](https://www.npmjs.com/package/@opentelemetry/instrumentation-aws-lambda).

```JavaScript
import * as Sentry from '@sentry/aws-serverless';

Sentry.init({
  integrations: [Sentry.awsIntegration()],
});
```
