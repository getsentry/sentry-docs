<Alert>
  Esta integración solo funciona en un entorno de navegador.
</Alert>

*Nombre de importación: `Sentry.httpContextIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración adjunta información de la solicitud HTTP, como la URL, el user-agent, el referrer y otros headers, al evento.
Permite catalogar y etiquetar correctamente los eventos con información específica del sistema operativo, el navegador y la versión.

```JavaScript
Sentry.init({
  integrations: [Sentry.httpContextIntegration()],
});
```
