<Alert>
  Esta integración solo funciona en los runtimes de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.mysql2Integration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `mysql2Integration` añade instrumentación para la biblioteca `mysql2` a fin de capturar spans mediante [`@opentelemetry/instrumentation-mysql2`](https://www.npmjs.com/package/@opentelemetry/instrumentation-mysql2).

```JavaScript
Sentry.init({
  integrations: [Sentry.mysql2Integration()],
});
```


<div id="supported-versions">
  ## Versiones compatibles
</div>

- `mysql2`: `>= 1.4.2, < 4.0`
