<Alert>
  Esta integración solo funciona en los entornos de ejecución de Node.js y Bun. Requiere la versión `8.32.0` o posterior del SDK.
</Alert>

*Nombre de importación: `Sentry.amqplibIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `amqplibIntegration` añade instrumentación para la biblioteca `amqplib` y captura spans usando [`@opentelemetry/instrumentation-amqplib`](https://www.npmjs.com/package/@opentelemetry/instrumentation-amqplib).

```JavaScript
Sentry.init({
  integrations: [Sentry.amqplibIntegration()],
});
```
