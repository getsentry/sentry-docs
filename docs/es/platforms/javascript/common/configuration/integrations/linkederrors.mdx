*Nombre de importación: `Sentry.linkedErrorsIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar las integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración te permite configurar errores vinculados. Se leerán de forma recursiva hasta un límite especificado y luego se buscarán mediante una clave concreta. De forma predeterminada, el SDK de Sentry establece el límite en cinco y la clave usada es `"cause"`.

```JavaScript
Sentry.init({
  integrations: [Sentry.linkedErrorsIntegration()],
});
```
