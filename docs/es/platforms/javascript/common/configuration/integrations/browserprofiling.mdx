<Alert>
  Esta integración solo funciona en un entorno de navegador.
</Alert>

*Nombre de importación: `Sentry.browserProfilingIntegration`*

[Profiling](/es/product/explore/profiling/) ofrece un nivel más profundo de visibilidad sobre el trazado tradicional, elimina la necesidad de instrumentación personalizada y permite una visibilidad precisa a nivel de código en tu aplicación en producción.

La integración BrowserProfiling configura el perfilado de rendimiento automático para tus aplicaciones de frontend. Captura perfiles mediante la [JS Self-Profiling API](https://wicg.github.io/js-self-profiling/) desde el navegador y los envía a Sentry. Para usar esta integración, también necesitas tener habilitada la [integración BrowserTracing](../browsertracing).

Obtén más información sobre [cómo configurar BrowserProfiling](./../../../profiling/).

```JavaScript
Sentry.init({
  integrations: [Sentry.browserProfilingIntegration()],
});
```
