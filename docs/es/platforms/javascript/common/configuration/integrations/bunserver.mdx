supported:
  - javascript.bun
---

<Alert>
  Esta integración solo funciona en el entorno de ejecución de Bun.
</Alert>

*Nombre de la importación: `Sentry.bunServerIntegration`*

Esta integración está habilitada de forma predeterminada. Si deseas modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `bunServerIntegration` instrumenta la [API `Bun.serve`](https://bun.sh/docs/api/http) para crear transacciones automáticamente y capturar errores.

```JavaScript
import * as Sentry from '@sentry/bun';

Sentry.init({
  integrations: [Sentry.bunServerIntegration()],
});
```
