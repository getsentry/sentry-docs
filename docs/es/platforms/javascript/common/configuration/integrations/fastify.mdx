Para obtener más información sobre cómo configurar el soporte de Sentry para Fastify, consulta la [documentación de Sentry para Fastify](/es/platforms/javascript/guides/fastify/).

<Alert>
  Esta integración solo funciona en los entornos de ejecución (runtimes) de Node.js y Bun.
</Alert>

*Nombre de importación: `Sentry.fastifyIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `fastifyIntegration` añade instrumentación para el framework Fastify para capturar spans usando [`@opentelemetry/instrumentation-fastify`](https://www.npmjs.com/package/@opentelemetry/instrumentation-fastify).

```JavaScript
Sentry.init({
  integrations: [Sentry.fastifyIntegration()],
});
```

<div id="supported-versions">
  ## Versiones compatibles
</div>

* `fastify`: `^3.0.0 || ^4.0.0`
