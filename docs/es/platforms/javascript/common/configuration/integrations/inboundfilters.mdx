*Nombre de importación: `Sentry.inboundFiltersIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración te permite ignorar errores específicos según el tipo, el mensaje o las URL de una excepción.

De forma predeterminada, omitirá los errores que comiencen con `Script error` o `JavaScript error: Script error`.

Para configurar esta integración, usa directamente las opciones del SDK `ignoreErrors`, `ignoreTransactions`, `denyUrls` y `allowUrls`. Por ejemplo:

```javascript
Sentry.init({
  ignoreErrors: ["ignore-this-error"],
});
```
