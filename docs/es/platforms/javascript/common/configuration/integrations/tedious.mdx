<Alert>
  Esta integración solo funciona en los entornos de ejecución de Node.js y Bun. Requiere la versión `8.38.0` o posterior del SDK.
</Alert>

*Nombre de la importación: `Sentry.tediousIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, consulta [esto](./../#modifying-default-integrations).

La `tediousIntegration` añade instrumentación para la biblioteca `tedious` para capturar spans usando [`@opentelemetry/instrumentation-tedious`](https://www.npmjs.com/package/@opentelemetry/instrumentation-tedious).

```javascript
Sentry.init({
  integrations: [Sentry.tediousIntegration()],
});
```
