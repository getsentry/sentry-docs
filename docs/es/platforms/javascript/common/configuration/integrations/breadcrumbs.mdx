<Alert>
  Esta integración solo funciona en un entorno de navegador.
</Alert>

*Nombre de importación: `Sentry.breadcrumbsIntegration`*

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `breadcrumbsIntegration` envuelve las API nativas para capturar breadcrumbs.

De forma predeterminada, el SDK de Sentry envuelve las API del navegador `console`, `dom`, `fetch`, `history` y `xhr` para añadir breadcrumbs. Puedes excluir la captura de breadcrumbs para partes específicas de tu aplicación (por ejemplo, puedes indicar que no se capturen las llamadas a `console.log` como breadcrumbs) mediante las opciones a continuación.

```JavaScript
Sentry.init({
  integrations: [
    Sentry.breadcrumbsIntegration({
      console: true,
      dom: true,
      fetch: true,
      history: true,
      xhr: true,
    }),
  ],
});
```


<div id="options">
  ## Opciones
</div>

<div id="console">
  ### `console`
</div>

_Tipo: `boolean`_

Registra las llamadas a `console.log`, `console.debug` y otros.

<div id="dom">
  ### `dom`
</div>

_Tipo: `boolean` | `{ serializeAttribute: string | string[] }`_

Registra todos los eventos de clic y de pulsación de teclas.

Cuando se proporciona un objeto con la clave `serializeAttribute`, la integración de Breadcrumbs buscará los atributos indicados en los elementos del DOM al generar las rutas de breadcrumbs. Los elementos coincidentes se identificarán por sus atributos personalizados, en lugar de por sus `id` o nombres de `class`.

<div id="fetch">
  ### `fetch`
</div>

_Tipo: `boolean`_

Registra las solicitudes HTTP realizadas con la API Fetch.

<div id="history">
  ### `history`
</div>

_Tipo: `boolean`_

Registra las llamadas a `history.pushState` y a las APIs relacionadas.

<div id="sentry">
  ### `sentry`
</div>

_Tipo: `boolean`_

Registra cada vez que enviamos un evento al servidor.

<div id="xhr">
  ### `xhr`
</div>

_Tipo: `boolean`_

Registra las solicitudes HTTP realizadas con la API de XHR.
