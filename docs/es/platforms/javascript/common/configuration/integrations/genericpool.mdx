<Alert>
  Esta integración solo funciona en los entornos de ejecución de Node.js y Bun. Requiere la versión `8.29.0` o superior del SDK.
</Alert>

*Nombre de importación: `Sentry.genericPoolIntegration`*

Esta integración se habilita de forma predeterminada cuando el monitoreo de rendimiento está activado. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

La `genericPoolIntegration` añade instrumentación para la biblioteca `generic-pool` para capturar spans usando [`@opentelemetry/instrumentation-generic-pool`](https://www.npmjs.com/package/@opentelemetry/instrumentation-generic-pool).

```JavaScript
Sentry.init({
  integrations: [Sentry.genericPoolIntegration()],
});
```
