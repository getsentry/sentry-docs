Para solucionar la mayoría de los elementos obsoletos en `7.x`, puedes usar el codemod [`@sentry/migr8`](https://www.npmjs.com/package/@sentry/migr8) para actualizar automáticamente cómo usas el SDK. `@sentry/migr8` requiere Node 18 o superior.

```bash
npx @sentry/migr8@latest
```

Nuestra herramienta de migración te permitirá elegir qué actualizaciones ejecutar y actualizará tu código automáticamente. En algunos casos, no podemos modificar el código por ti de forma automática. En esos casos, se marcarán con un comentario `TODO(sentry)`. ¡Asegúrate de revisar todos los cambios en el código después de ejecutar `@sentry/migr8`!
