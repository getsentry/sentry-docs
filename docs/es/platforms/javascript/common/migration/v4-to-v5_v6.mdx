Recomendamos actualizar directamente de `4.x` a `6.x`. La migración de `5.x` a `6.x` no introduce cambios incompatibles en la API del SDK.

Consulta nuestra [guía de migración detallada](https://github.com/getsentry/sentry-javascript/blob/develop/docs/migration/v4-to-v5_v6.md) para obtener más información sobre cómo migrar de `4.x` a `5.x`/`6.x`.
