La mayoría de las aplicaciones de Node hoy en día están escritas en CommonJS (CJS) o se compilan a CJS antes de ejecutarse.
CommonJS usa `require()` para cargar módulos. Nuestro método de instalación recomendado para CommonJS es requerir el archivo `instrument.js` al inicio de tu aplicación.

Debes crear un archivo llamado `instrument.js` que importe e inicialice Sentry:
