El SDK de Sentry utiliza [OpenTelemetry](https://opentelemetry.io/) bajo el capó. Esto significa que cualquier instrumentación de OpenTelemetry que emita spans será detectada automáticamente por Sentry sin configuración adicional.

Para empezar a capturar trazas y spans, configura <PlatformLink to='/tracing'>Tracing and Performance Monitoring con tu SDK de Sentry</PlatformLink>. Si no usas tracing, Sentry igual se conecta a OpenTelemetry bajo el capó para garantizar que el aislamiento de contexto y la propagación de trazas funcionen correctamente.

De forma predeterminada, Sentry configurará OpenTelemetry automáticamente por ti, pero también puedes usar tu propia configuración de OpenTelemetry. Lee las guías a continuación para aprender a usar una configuración personalizada de OpenTelemetry o a sacarle el máximo partido a la integración entre Sentry y OpenTelemetry.
