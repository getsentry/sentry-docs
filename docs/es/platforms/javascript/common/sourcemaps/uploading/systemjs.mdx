En esta guía, aprenderás a cargar correctamente mapas de código fuente para SystemJS con nuestra herramienta `sentry-cli`.

<Include name="debug-id-requirement.mdx" />

<div id="1-generate-source-maps">
  ### 1. Generar mapas de código fuente
</div>

Primero, configura SystemJS para generar mapas de código fuente:

```
builder.bundle("src/app.js", "dist/app.min.js", {
  minify: true,
  sourceMaps: true,
  sourceMapContents: true,
});
```

<Alert>
  Generar sourcemaps puede exponerlos públicamente y provocar la filtración de tu código fuente. Puedes evitarlo configurando el servidor para denegar el acceso a los archivos `.js.map` o eliminando los sourcemaps antes de implementar tu aplicación.
</Alert>
