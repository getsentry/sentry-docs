En esta guía, aprenderás a cargar correctamente los mapas de origen para SystemJS usando nuestra herramienta `sentry-cli`.

<Include name="debug-id-requirement.mdx" />

<div id="1-generate-source-maps">
  ### 1. Generar mapas de origen
</div>

Primero, configura UglifyJS para que emita mapas de origen:

```
uglifyjs app.js \
  -o app.min.js \
  --source-map url=app.min.js.map,includeSources
```

<Alert>
  Generar sourcemaps puede exponerlos públicamente y provocar la filtración de tu código fuente. Puedes evitarlo configurando el servidor para bloquear el acceso a los archivos `.js.map` o eliminando los sourcemaps antes de implementar tu aplicación.
</Alert>
