Si usas `AngularJS 1.x`, puedes usar la integración de AngularJS de Sentry.

<Alert level="warning" title="Fin del soporte para AngularJS 1.x">
  A partir de la versión 7, el SDK de JavaScript de Sentry no será compatible con AngularJS 1.x. Usa la versión 6.x del SDK si quieres usar Sentry junto con AngularJS 1.x.
</Alert>

<div id="features">
  ## Funciones
</div>

Además de capturar errores, puedes supervisar las interacciones entre varios servicios o aplicaciones al [habilitar el tracing](/es/concepts/key-terms/tracing/). También puedes llegar más rápido a la causa raíz de un error o problema de rendimiento viendo una reproducción tipo video de una sesión de usuario con [session replay](/es/product/explore/session-replay/web/getting-started/).

Selecciona qué funciones de Sentry te gustaría instalar, además de Error Monitoring, para obtener a continuación las instrucciones correspondientes de instalación y configuración.

<div id="install">
  ### Instalación
</div>

Instala `@sentry/browser` y `@sentry/integrations` con `yarn` o `npm`:

```bash {tabTitle:npm}
npm install @sentry/browser@6 @sentry/integrations@6 --save
```

```bash {tabTitle:yarn}
yarn add @sentry/browser@6 @sentry/integrations@6
```

```bash {tabTitle:pnpm}
pnpm add @sentry/browser@6 @sentry/integrations@6
```

y luego usarlo así:

```javascript
import angular from "angular";
import * as Sentry from "@sentry/browser";
import { Angular as AngularIntegration } from "@sentry/integrations";

// Asegúrate de llamar a Sentry.init después de importar AngularJS.
// También puedes pasar {angular: AngularInstance} al constructor de Integrations.Angular().
Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [new AngularIntegration()],
});

// Por último, añade ngSentry como dependencia en el módulo de tu aplicación.
angular.module("yourApplicationModule", ["ngSentry"]);
```

Si usas la versión del SDK por CDN, Sentry ofrece un archivo independiente para cada integración:

```html
<script
  src="https://browser.sentry-cdn.com/6.19.7/bundle.min.js"
  integrity="sha384-KXjn4K+AYjp1cparCXazrB+5HKdi69IUYz8glD3ySH3fnDgMX3Wg6VTMvXUGr4KB"
  crossorigin="anonymous"
></script>

<!-- Si incluyes la integración, estará disponible como Sentry.Integrations.Angular -->
<script
  src="https://browser.sentry-cdn.com/6.19.7/angular.min.js"
  crossorigin="anonymous"
></script>

<script>
  Sentry.init({
    dsn: "___PUBLIC_DSN___",
    integrations: [new Sentry.Integrations.Angular()],
  });
</script>
```
