_Nombre de importación: `Sentry.contextLinesIntegration`_

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración añade código fuente a tus marcos de pila y te permite ver en la interfaz de Sentry el código que provocó el error.

<PlatformContent includePath="configuration/contextlines" />

<div id="options">
  ## Opciones
</div>

<div id="framecontextlines">
  ### `frameContextLines`
</div>

_Tipo: `number`_

El número de líneas que se recopilan alrededor del número de línea de cada marco de la pila. De forma predeterminada es 7.
