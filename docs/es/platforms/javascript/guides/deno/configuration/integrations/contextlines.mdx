_Nombre de importación: `Sentry.contextLinesIntegration`_

Esta integración está habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración agrega código fuente a tus fotogramas de la pila. Te permite ver el código que causó el error directamente en la interfaz de Sentry.

<PlatformContent includePath="configuration/contextlines" />

<div id="options">
  ## Opciones
</div>

<div id="framecontextlines">
  ### `frameContextLines`
</div>

_Tipo: `number`_

La cantidad de líneas que se recopilan alrededor del número de línea de cada fotograma de la pila. El valor predeterminado es 7.
