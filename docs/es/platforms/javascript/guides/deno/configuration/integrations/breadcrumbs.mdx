_Nombre de importación: `Sentry.breadcrumbsIntegration`_

Esta integración está habilitada de forma predeterminada. También puedes [modificar las integraciones predeterminadas](./../#modifying-default-integrations).

La `breadcrumbsIntegration` envuelve APIs nativas para capturar rastreos.
De forma predeterminada, el SDK de Sentry envuelve todas las API. Puedes excluir la captura de rastreos para partes específicas de tu aplicación (por ejemplo, puedes indicar que no se capturen las llamadas a `console.log` como rastreos) mediante las opciones a continuación.

<div id="options">
  ## Opciones
</div>

<div id="console">
  ### `console`
</div>

_Tipo: `boolean`_

Registra las llamadas a `console.log`, `console.debug` y similares.

<div id="fetch">
  ### `fetch`
</div>

_Tipo: `boolean`_

Registra las solicitudes HTTP realizadas con la API Fetch.

<div id="sentry">
  ### `sentry`
</div>

_Tipo: `boolean`_

Registra cada vez que enviamos un evento al servidor.
