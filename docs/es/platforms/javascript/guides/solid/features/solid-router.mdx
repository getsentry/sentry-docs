---
title: Solid Router
description: "Conoce la integración de Solid Router de Sentry."
---

<Alert>
  La instrumentación de enrutamiento es compatible con [Solid Router](https://docs.solidjs.com/solid-router) a partir de la versión 0.13.4.
</Alert>

El SDK de Sentry proporciona instrumentación de enrutamiento para Solid Router que crea spans de navegación para garantizar
que recopiles datos de rendimiento relevantes sobre el estado de las cargas de página y las solicitudes asociadas.

Para empezar, importa `solidRouterBrowserTracingIntegration` desde `@sentry/solid/solidrouter` y agrégalo a `Sentry.init`
en lugar de `Sentry.browserTracingIntegration` para habilitar el rastreo de rendimiento.

Importa `withSentryRouterRouting` desde `@sentry/solid/solidrouter` y úsalo para envolver `Router`, `MemoryRouter` o `HashRouter` de `@solidjs/router`.
Esto crea un componente de orden superior que permitirá a Sentry acceder al contexto de tu router.

```jsx
import * as Sentry from "@sentry/solid";
import { solidRouterBrowserTracingIntegration, withSentryRouterRouting } from '@sentry/solid/solidrouter';
import { Route, Router } from "@solidjs/router";
import { render } from "solid-js/web";
import App from "./app";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [solidRouterBrowserTracingIntegration()],
  tracesSampleRate: 1.0, //  Captura el 100% de las transacciones
});

// Envuelve Solid Router para recopilar datos de rendimiento relevantes en los cambios de ruta
const SentryRouter = withSentryRouterRouting(Router);

render(
  () => (
    <SentryRouter>
      <Route path="/" component={App} />
      ...
    </SentryRouter>
  ),
  document.getElementById("root"),
);
```
