---
title: Límite de errores de Solid
description: "Aprende a envolver límites de errores de Solid para capturar errores automáticamente."
---

El SDK de Sentry exporta una función para envolver el componente nativo de límite de errores de Solid y capturar automáticamente las excepciones dentro de un árbol de componentes, además de renderizar un componente alternativo.

Envuelve el componente nativo `ErrorBoundary` de Solid con `Sentry.withSentryErrorBoundary`.

```jsx
import * as Sentry from "@sentry/solid";
import { ErrorBoundary } from "solid-js";

// Envuelve el ErrorBoundary de Solid para capturar excepciones automáticamente
const SentryErrorBoundary = Sentry.withSentryErrorBoundary(ErrorBoundary);

export default function SomeComponent() {
  return (
    <SentryErrorBoundary fallback={err => <div>Error: {err.message}</div>}>
        <div>Componente de ejemplo</div>
    </SentryErrorBoundary>
  );
}
```
