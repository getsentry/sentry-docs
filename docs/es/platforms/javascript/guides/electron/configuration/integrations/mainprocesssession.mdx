Esta integración captura sesiones vinculadas a la vida útil del proceso principal de Electron. Capturar sesiones permite mostrar estadísticas de [salud de la versión](/es/product/releases/health/) en Sentry.

Si prefieres capturar sesiones vinculadas al foco de `BrowserWindow`,
usa la integración <PlatformLink to="/configuration/integrations/browserwindowsession">BrowserWindowSession</PlatformLink>
en los procesos del renderer.

Para desactivar el envío de sesiones, excluye `MainProcessSession` de las integraciones predeterminadas:

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: (defaults) => defaults.filter((i) => i.name !== "MainProcessSession"),
});
```

El comportamiento predeterminado es enviar sesiones solo cuando ocurre un error o cuando finaliza el proceso principal. Para enviar sesiones cuando se crea el proceso principal, habilita la opción `sendOnCreate`.

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [Sentry.mainProcessSessionIntegration({ sendOnCreate: true })],
});
```
