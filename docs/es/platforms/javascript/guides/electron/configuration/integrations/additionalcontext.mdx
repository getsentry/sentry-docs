Añade contexto adicional del dispositivo a los eventos.

```typescript
interface AdditionalContextOptions {
  /**
   * Captura las dimensiones y la resolución de la pantalla principal
   * @default true
   */
  screen: boolean;
  /**
   * Captura el modelo y fabricante del dispositivo.
   * No compatible con Linux.
   * @default false
   */
  deviceModelManufacturer: boolean;
}
```

Para desactivar elementos de contexto específicos, establécelos en `false`:

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [
    Sentry.additionalContextIntegration({
      screen: false,
    }),
  ],
});
```
