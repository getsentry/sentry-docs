Esta integración inyecta un script de preload mediante la API de Electron
[`session.setPreloads(preloads)`](https://www.electronjs.org/docs/latest/api/session#sessetpreloadspreloads).

De forma predeterminada, las preloads solo se establecen en `defaultSession`. Si usas otras sesiones, puedes pasar una función como `getSessions` a `init`.

```javascript
import { session } from "electron";
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  getSessions: () => [
    session.defaultSession,
    session.fromPartition("persist:my-session"),
  ],
});
```
