Captura y envía minidumps mediante el cargador integrado `crashReporter` de Electron. Los bloqueos del proceso de renderizado y de la GPU solo incluirán un contexto mínimo debido a las limitaciones del cargador `crashReporter`. Para obtener el contexto completo, es preferible usar la integración predeterminada `SentryMinidump`.

Si agregas la integración `ElectronMinidump`, la integración `SentryMinidump` se eliminará automáticamente para evitar que se informen bloqueos duplicados.

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [Sentry.electronMinidumpIntegration()],
});
```
