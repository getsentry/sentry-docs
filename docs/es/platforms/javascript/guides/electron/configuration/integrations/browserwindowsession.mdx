Requiere `@sentry/electron` versión `4.11.0` o posterior.

Captura sesiones vinculadas al enfoque de las `BrowserWindow` de Electron. Capturar
sesiones permite mostrar estadísticas de [salud de la versión](/es/product/releases/health/)
en Sentry.

Al agregar esta integración, se desactiva la integración de sesión <PlatformLink to="/configuration/integrations/mainprocesssession">MainProcessSession</PlatformLink>.

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [Sentry.browserWindowSessionIntegration()],
});
```

De forma predeterminada, las sesiones terminan cuando ninguna ventana ha tenido el foco durante 30 segundos. Puedes ajustar este tiempo de espera con la opción `backgroundTimeoutSeconds`:

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integraciones: [
    Sentry.browserWindowSessionIntegration({
      backgroundTimeoutSeconds: 120,
    }),
  ],
});
```
