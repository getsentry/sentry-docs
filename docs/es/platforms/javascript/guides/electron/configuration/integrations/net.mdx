Esta integración captura breadcrumbs y spans de trazado para el módulo `net` de Electron. Los breadcrumbs y los spans de trazado están habilitados de forma predeterminada, y las cabeceras `sentry-trace` se añaden a las solicitudes para todos los orígenes.

La integración se puede configurar con las siguientes opciones:

```typescript
interface NetOptions {
  /**
   * Si se deben capturar breadcrumbs para las solicitudes de red
   *
   * Valor predeterminado: true
   */
  breadcrumbs: boolean;
  /**
   * Si se deben capturar spans de transacción para las solicitudes de red
   *
   * Valor predeterminado: true
   */
  tracing: boolean | (method: string, url: string) => boolean;
}
```

Aquí tienes un ejemplo de código para desactivar la captura de breadcrumbs y añadir los encabezados `sentry-trace` a fin de capturar spans solo en las solicitudes `POST`:

```javascript
import * as Sentry from "@sentry/electron/main";

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [
    Sentry.electronNetIntegration({
      breadcrumbs: false,
      tracing: (method) => method == "POST",
    }),
  ],
});
```
