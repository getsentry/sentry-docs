title: OnUncaughtException
excerpt: ""

sidebar_order: 10
---

Permite capturar eventos de `uncaughtException` manteniendo el comportamiento predeterminado de Electron de mostrar un diálogo de error si no hay otros listeners.

Para desactivar el diálogo de error predeterminado de Electron, agrega otro listener:

```javascript
process.on("uncaughtException", () => {});
```
