_Nombre de importación: `Sentry.contextLinesIntegration`_

Esta integración viene habilitada de forma predeterminada. Si quieres modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración agrega código fuente a tus stack frames. Te permite ver directamente en la interfaz de Sentry el código que provocó el error.

<PlatformContent includePath="configuration/contextlines" />

<div id="options">
  ## Opciones
</div>

<div id="framecontextlines">
  ### `frameContextLines`
</div>

_Tipo: `number`_

La cantidad de líneas que se recopilan alrededor del número de línea de cada frame de la pila. De forma predeterminada, es 7.
