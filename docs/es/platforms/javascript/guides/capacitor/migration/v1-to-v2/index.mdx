title: Migrar de la 1.x a la 2.x

description: "Aprende a migrar del Sentry Capacitor SDK 1.x al 2.x"
---

La versión 2 del Sentry Capacitor SDK introduce principalmente una depuración del API y cambios en la compatibilidad de versiones.

Esta actualización incluye cambios de comportamiento que no serán detectados por comprobadores de tipos, linters ni pruebas, por lo que recomendamos leer detenidamente toda la guía de migración en lugar de depender de herramientas automáticas.

La versión 2 del SDK es compatible con las versiones self-hosted de Sentry 24.4.2 o superiores (sin cambios respecto a la v1). Las versiones anteriores pueden seguir funcionando, pero es posible que no admitan todas las funciones.

<div id="typescript-version-policy">
  ### Política de versiones de TypeScript
</div>

En preparación para la v2 del SDK de OpenTelemetry, la versión mínima requerida de TypeScript se actualiza a la `5.0.4`.

<div id="dropping-support-for-capacitor-5-and-earlier">
  ### Fin del soporte para Capacitor 5 y versiones anteriores
</div>

Para garantizar la mejor experiencia con las funciones más recientes de Capacitor, el soporte para Capacitor 5 y versiones anteriores se retirará a partir de la versión 2.0. Recomendamos actualizar a Capacitor 6 o posterior. Si necesitas seguir usando versiones anteriores de Capacitor, mantente en Sentry Capacitor SDK v1.

<div id="self-hosted-sentry-compatibility">
  ### Compatibilidad con Sentry autoalojado
</div>

A partir de la versión 2.0.0, el SDK de Sentry para Capacitor solo será compatible con instancias de Sentry autoalojadas a partir de la versión 25.2.0. Este cambio no afecta a los usuarios de `sentry.io`.

<div id="how-to-upgrade-to-sdk-version-2">
  ## Cómo actualizar a la versión 2 del SDK:
</div>

* Capacitor versión 2 depende de Sentry JavaScript SDK versión 9. Recomendamos leer la siguiente [guía de migración](#sibling-migration-guide) para saber cómo abordar cualquier cambio incompatible al actualizar los SDK correspondientes.

<Alert level="warning" title="Importante">
  Para evitar problemas, usa versiones fijas de Sentry Capacitor y de los SDK correspondientes. Instala siempre la versión del SDK correspondiente que coincida con Sentry Capacitor.
</Alert>
