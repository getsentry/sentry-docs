<div id="how-to-upgrade-to-sdk-version-1">
  ## Cómo actualizar a la versión 1 del SDK:
</div>

Al actualizar a la versión 1 del SDK, ten en cuenta los principales cambios incompatibles con Sentry Capacitor. Puedes encontrar información más detallada sobre los cambios incompatibles en los SDK relacionados en la <PlatformLink to="/migration/v0-to-v1/v7-to-v8/">guía de migración de los SDK relacionados</PlatformLink>.

- La versión mínima de Angular admitida pasó a ser la 14; actualiza a Angular 14 o superior para usar el SDK más reciente.

- Si usas el paquete `@sentry/angular-ivy`, debes reemplazarlo por `@sentry/angular`.

- Se eliminó `enableOutOfMemoryTracking` y se reemplazó por `enableWatchdogTerminationTracking`.

- Capacitor V1 requiere el SDK de Sentry para JavaScript en la versión 8.x. Te recomendamos leer la guía de migración a continuación para saber cómo abordar cualquier cambio incompatible al actualizar las dependencias relacionadas a la versión 8.
