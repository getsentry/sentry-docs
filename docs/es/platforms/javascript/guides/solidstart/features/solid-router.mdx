---
title: Solid Router
description: "Conoce la integración de Sentry con Solid Router."
---

<Alert>
  La instrumentación de enrutamiento es compatible con [Solid Router](https://docs.solidjs.com/solid-router) a partir de la versión 0.13.4.
</Alert>

El SDK de Sentry proporciona instrumentación de enrutamiento para Solid Router con el fin de crear spans de navegación y asegurarte de
recopilar datos de rendimiento relevantes sobre el estado de las cargas de página y las solicitudes asociadas.

Para empezar, importa `solidRouterBrowserTracingIntegration` de `@sentry/solidstart/solidrouter` y añádelo a `Sentry.init`
en lugar del `Sentry.browserTracingIntegration` habitual para habilitar el rastreo de rendimiento.

Importa `withSentryRouterRouting` de `@sentry/solidstart/solidrouter` y úsalo para envolver `Router`, `MemoryRouter` o `HashRouter` de `@solidjs/router`.
Esto crea un componente de orden superior, lo que permitirá que Sentry acceda al contexto de tu router.

```jsx
import { withSentryRouterRouting } from '@sentry/solidstart/solidrouter';
import { Router } from "@solidjs/router";
import { FileRoutes } from "@solidjs/start/router";

// Envuelve Solid Router para recopilar datos de rendimiento útiles al cambiar de rutas
const SentryRouter = withSentryRouterRouting(Router);

export default function App() {
  return (
    <SentryRouter>
      <FileRoutes />
    </SentryRouter>
  )
}
```
