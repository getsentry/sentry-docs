---
title: Instrumentación automática
description: "Descubre qué transacciones se capturan después de habilitar el rastreo."
sidebar_order: 10
---

Cuando el rendimiento está habilitado mediante `tracesSampleRate` o una función `tracesSampler`, el SDK de Sentry capturará automáticamente spans para solicitudes entrantes y solicitudes HTTP realizadas con:

* `http`
* `https`
* `fetch`

La instrumentación de bases de datos se puede habilitar manualmente agregando las integraciones correspondientes al array `integrations` en el método `init`. Se admiten los siguientes paquetes:

* `mysql`
* `mysql2`
* `pg`
* `graphql` (incluido Apollo Server)
* `mongo`
* `mongoose`
* `ioredis`
* `prisma` (requiere configuración adicional; consulta <PlatformLink to="/configuration/integrations/prisma">Integración de Prisma</PlatformLink>)

Puedes agregar las integraciones de la siguiente manera:

```javascript
const Sentry = require("@sentry/aws-serverless");

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  tracesSampleRate: 1.0,
  integrations: [
    Sentry.mysqlIntegration(),
    Sentry.mysql2Integration(),
    Sentry.pgIntegration(),
    Sentry.graphqlIntegration(),
    Sentry.mongoIntegration(),
    Sentry.mongooseIntegration(),
    Sentry.ioredisIntegration(),
  ],
});
```
