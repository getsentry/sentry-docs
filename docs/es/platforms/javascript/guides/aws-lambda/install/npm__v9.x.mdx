Según si tu función Lambda se ejecuta en CommonJS o en ESM, debes seguir las instrucciones del sistema de módulos correspondiente.

<div id="my-lambda-function-is-written-in-typescript">
  ## Mi función Lambda está escrita en TypeScript
</div>

Si usas TypeScript, es probable que tu función Lambda se transpile a CommonJS antes de ejecutarse. En ese caso, sigue las [instrucciones de CommonJS](./cjs-npm__v9.x).
Ten en cuenta que TypeScript también puede configurarse para generar ESM; en ese caso, deberías seguir las [instrucciones de ESM](./esm-npm__v9.x).

<div id="my-lambda-function-uses-require">
  ## Mi función Lambda usa `require`
</div>

Si estás usando `require()` en tu función, sigue las [instrucciones de CommonJS](./cjs-npm__v9.x).

<div id="my-lambda-function-uses-import">
  ## Mi función Lambda usa `import`
</div>

Si estás usando la sintaxis `import` en tu función y _no_ transpilas el código a CommonJS, sigue las [instrucciones de ESM](./esm-npm__v9.x).
