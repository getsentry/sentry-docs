<div id="should-i-use-the-lambda-layer-or-the-npm-package">
  ### ¿Debería usar la capa de Lambda o el paquete de NPM?
</div>

Por lo general, recomendamos usar la capa de Lambda, ya que no requiere desplegar ninguna dependencia de Sentry junto con tu función.
Con la capa, puedes conseguir el mismo nivel de personalización que con el paquete de NPM.
Aun así, hay dos motivos por los que podrías preferir usar el paquete de NPM:

1. Quieres minimizar el tamaño de la función Lambda y aplicar tree-shaking a las partes del código del SDK que no necesitas. Un motivo relacionado podría ser que estás transpilando tu código y quieres transpilar también tus dependencias.
2. Ya usas paquetes de NPM y despliegas `node_modules` con tu función, y no quieres añadir una (u otra) capa de Lambda a tus funciones.
