Vue 3 te permite usar varias aplicaciones con la misma instancia del SDK de Sentry y también agregar más aplicaciones dinámicamente después de haber inicializado el SDK.

<div id="multiple-apps">
  ### Varias aplicaciones
</div>

Pasa todas las aplicaciones que creaste inicialmente a la opción `app` en `Sentry.init`:

```javascript {filename:main.js}
const appOne = Vue.createApp(App);
const appTwo = Vue.createApp(App);
const appThree = Vue.createApp(App);

Sentry.init({
  app: [appOne, appTwo, appThree],
});
```


<div id="dynamic-initialization">
  ### Inicialización dinámica
</div>

Si algunas de tus aplicaciones se inicializan más tarde, puedes agregar Sentry a ellas de la siguiente manera:

```javascript {filename:main.js}
// ...
const myLazyApp = createApp(MiscApp);

myLazyApp.mixin(Sentry.createTracingMixins({ trackComponents: true }));
Sentry.attachErrorHandler(myLazyApp);
```
