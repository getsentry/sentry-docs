<div id="install-the-sentry-sdk">
  ### Instala el SDK de Sentry
</div>

Ejecuta el comando del gestor de paquetes que prefieras para añadir el SDK de Sentry a tu aplicación:


```bash {tabTitle:yarn}
yarn add @sentry/vue
```

```bash {tabTitle:pnpm}
pnpm add @sentry/vue
```

<div id="step-2-configure">
  ## Paso 2: Configurar
</div>

Para inicializar Sentry en tu aplicación de Vue, añade el siguiente fragmento de código a tu `main.js`:

```javascript {tabTitle:Vue 3} {filename:main.js} {3, 12-41}
import { createApp } from "vue";
import { createRouter } from "vue-router";
import * as Sentry from "@sentry/vue";

const app = createApp({
  // ...
});
const router = createRouter({
  // ...
});

Sentry.init({
  app,
  dsn: "___PUBLIC_DSN___",

  // Agrega encabezados de solicitud e IP para usuarios, para más información visita:
  // https://docs.sentry.io/platforms/javascript/guides/vue/configuration/options/#sendDefaultPii
  sendDefaultPii: true,

  integrations: [
    // ___PRODUCT_OPTION_START___ performance
    Sentry.browserTracingIntegration({ router }),
    // ___PRODUCT_OPTION_END___ performance
    // ___PRODUCT_OPTION_START___ session-replay
    Sentry.replayIntegration(),
    // ___PRODUCT_OPTION_END___ session-replay
    // ___PRODUCT_OPTION_START___ user-feedback
    Sentry.feedbackIntegration({
      // La configuración adicional del SDK va aquí, por ejemplo:
      colorScheme: "system",
    }),
    // ___PRODUCT_OPTION_END___ user-feedback
  ],
  // ___PRODUCT_OPTION_START___ logs

  // Habilita el envío de logs a Sentry
  enableLogs: true,
  // ___PRODUCT_OPTION_END___ logs

  // ___PRODUCT_OPTION_START___ performance
  // Establece tracesSampleRate en 1.0 para capturar el 100%
  // de las transacciones para el rastreo.
  // Recomendamos ajustar este valor en producción
  // Más información en
  // https://docs.sentry.io/platforms/javascript/configuration/options/#traces-sample-rate
  tracesSampleRate: 1.0,

  // Establece `tracePropagationTargets` para controlar en qué URLs debe habilitarse la propagación del rastreo
  tracePropagationTargets: ["localhost", /^https:\/\/yourserver\.io\/api/],
  // ___PRODUCT_OPTION_END___ performance
  // ___PRODUCT_OPTION_START___ session-replay

  // Captura Replay para el 10% de todas las sesiones,
  // más el 100% de las sesiones con error
  // Más información en
  // https://docs.sentry.io/platforms/javascript/session-replay/configuration/#general-integration-configuration
  replaysSessionSampleRate: 0.1,
  replaysOnErrorSampleRate: 1.0,
  // ___PRODUCT_OPTION_END___ session-replay
});

app.use(router);
app.mount("#app");
```

<div id="configuration-for-late-defined-vue-apps">
  ### Configuración para aplicaciones de Vue definidas posteriormente
</div>

Si tu aplicación de Vue no está definida desde el principio, puedes añadir más adelante el monitoreo de errores específicos de Vue.
Para añadir manualmente la integración en aplicaciones de Vue definidas posteriormente, actualiza tu archivo `main.js`:

```javascript {filename:main.js}
import * as Sentry from "@sentry/vue";

Sentry.init({
  dsn: "...",
  // Filtrar la integración `Vue` por defecto
  integrations: (integrations) =>
    integrations.filter((integration) => integration.name !== "Vue"),
});

// Más tarde
const app = createApp({
  template: "<div>hola</div>",
});

Sentry.addIntegration(Sentry.vueIntegration({ app }));
```

<div id="configuration-for-pinia">
  ### Configuración de Pinia
</div>

Para capturar el estado de Pinia, usa `Sentry.createSentryPiniaPlugin()` y agrégalo a la instancia de tu store de Pinia:

```javascript
import { createPinia } from "pinia";
import { createSentryPiniaPlugin } from "@sentry/vue";

const pinia = createPinia();

pinia.use(createSentryPiniaPlugin());
```

Más información sobre el [plugin de Pinia](./features/pinia) y sus opciones.

<div id="issues">
  ### Incidencias
</div>

Para comprobar que Sentry está capturando errores y creando incidencias en tu proyecto de Sentry, añade el siguiente botón de prueba a una de tus páginas; al hacer clic, provocará un error que Sentry registrará:

```javascript {filename:App.vue}
// ...
<button @click="throwError">Lanzar error</button>
// ...

export default {
  // ...
  methods: {
    throwError() {
      throw new Error('Error de Sentry');
    }
  }
};
```
