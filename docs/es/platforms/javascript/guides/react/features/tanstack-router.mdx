---

La integración de TanStack Router está incluida en el paquete `@sentry/react` y es compatible con la versión `1.64.0` de `@tanstack/react-router` y superiores.

<Alert title="Nota">

La integración de TanStack Router está diseñada para funcionar con Sentry Tracing. Consulta <PlatformLink to="/tracing/#enable-tracing">Introducción al rendimiento en React</PlatformLink> para obtener más detalles sobre cómo configurar e instalar el SDK.

</Alert>

La instrumentación de TanStack Router utiliza tus rutas de TanStack Router para crear transacciones de `pageload/navigation` y ayudarte a recopilar datos de rendimiento relevantes sobre el estado de las cargas de página y las solicitudes asociadas.

<div id="usage">
  ## Uso
</div>

Para utilizar la integración de TanStack Router, incluye `Sentry.tanstackRouterBrowserTracingIntegration` en tu opción `integrations`:

```javascript
import * as Sentry from "@sentry/react";
import { createRouter } from "@tanstack/react-router";

const router = createRouter({
  // Opciones del enrutador...
});

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  integrations: [Sentry.tanstackRouterBrowserTracingIntegration(router)],

  // Es necesario definir una tasa de muestreo para enviar datos de rendimiento.
  // Recomendamos ajustar este valor en producción o usar tracesSampler
  // para un control más preciso.
  tracesSampleRate: 1.0,
});
```
