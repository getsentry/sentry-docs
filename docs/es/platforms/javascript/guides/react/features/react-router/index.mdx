*(Disponible a partir de la versión 5.21.0)*

La compatibilidad con React Router está incluida en el paquete `@sentry/react` desde la versión `5.21.0`.

<Alert title="Nota">
  La integración de React Router está diseñada para funcionar con Sentry Tracing. Consulta [Configurar el trazado con React](/es/platforms/javascript/guides/react/tracing/) para obtener más detalles sobre cómo configurar e instalar el SDK.
</Alert>

La instrumentación de React Router utiliza la biblioteca React Router para crear transacciones de `pageload/navigation`, de modo que puedas recopilar datos de rendimiento relevantes sobre el estado de las cargas de página y las solicitudes asociadas.

Ofrecemos integraciones para React Router 3, 4, 5, 6 y 7.
