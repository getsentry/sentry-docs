La mayoría de las aplicaciones de Node hoy en día están escritas en, o se compilan a, CommonJS (CJS), que utiliza `require()` para cargar módulos.
Nest.js, de forma predeterminada, transpila tu código TypeScript a CommonJS.

Puedes seguir las instrucciones de instalación en la <PlatformLink to="/">Guía de inicio rápido</PlatformLink> del SDK.
