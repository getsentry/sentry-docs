---
title: Decorador SentryTraced
description: "Obtén información sobre el decorador SentryTraced de Sentry."
---

<Alert>
  El decorador `@SentryTraced` está disponible a partir de la versión `@sentry/nestjs` 8.15.0 y posteriores.
</Alert>

El SDK de NestJS incluye un decorador `@SentryTraced` que creará spans cada vez que se invoque el método que decora.
Esta función habilita un monitoreo del rendimiento más granular y te ayuda a obtener información detallada sobre el rendimiento de funciones individuales dentro de tu aplicación.

Para empezar, importa `SentryTraced` desde `@sentry/nestjs` y úsalo para decorar la función que prefieras:

```typescript
import { SentryTraced } from '@sentry/nestjs';

export class MyService {
  @SentryTraced()
  myFunction() {
    // Lógica de tu función aquí
  }
}
```
