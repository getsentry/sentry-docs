---
title: Decorador SentryCron
description: "Conoce el decorador SentryCron de Sentry."
---

<Alert>
  El decorador `@SentryCron` está disponible a partir de la versión `@sentry/nestjs` 8.16.0.
</Alert>

El SDK de NestJS incluye un decorador `@SentryCron` que puede utilizarse para ampliar el decorador nativo `@Cron` de NestJS y enviar check-ins a Sentry antes y después de cada ejecución de un trabajo cron.

Para empezar, importa `SentryCron` de `@sentry/nestjs` y úsalo para decorar una función de trabajo cron en tu aplicación NestJS.

<Alert>
  El decorador `@SentryCron` debe aplicarse después del decorador `@Cron`; de lo contrario, la instrumentación no funciona.
</Alert>

```typescript
import { Cron } from '@nestjs/schedule';
import { SentryCron } from '@sentry/nestjs';

export class MyCronService {
  @Cron('* * * * *')
  @SentryCron('my-monitor-slug', {
    schedule: {
      type: "crontab",
      value: "* * * * *",
    },
    checkinMargin: 2, // En minutos. Opcional.
    maxRuntime: 10, // En minutos. Opcional.
    timezone: "America/Los_Angeles", // Opcional.
  })
  handleCron() {
    // Tu lógica de tarea cron aquí
  }
}
```
