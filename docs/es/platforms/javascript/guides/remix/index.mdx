<div id="step-1-install">
  ## Paso 1: Instalación
</div>

Para instalar Sentry con el asistente de instalación, ejecuta el siguiente comando en tu proyecto:

```bash
npx @sentry/wizard@latest -i remix
```

El asistente luego te guía por el proceso de configuración y te pide habilitar funciones adicionales (opcionales) de Sentry para tu aplicación, más allá del monitoreo de errores.

<PlatformContent includePath="getting-started-features-expandable" />

Esta guía asume que habilitas todas las funciones y permites que el asistente cree una página de ejemplo. Puedes agregar o quitar funciones en cualquier momento, pero configurarlas ahora te ahorrará el trabajo de hacerlo manualmente más adelante.

<Expandable title="¿Qué cambia el asistente de instalación dentro de tu aplicación?">
  * Crea o actualiza `entry.(client|server).tsx` para inicializar el SDK
  * Crea `.env.sentry-build-plugin` con un token de autenticación para cargar mapas de código fuente
    (este archivo se agrega automáticamente a `.gitignore`)
  * Actualiza tu script de compilación o la configuración de Vite para cargar automáticamente los mapas de código fuente a Sentry
  * Instrumenta tu aplicación para admitir funciones de Remix v2 si usas [Remix future flags](https://remix.run/docs/en/main/pages/api-development-strategy#current-future-flags)
  * Agrega una página de ejemplo a tu aplicación para ayudarte a verificar tu configuración de Sentry
</Expandable>

<div id="step-2-server-side-performance-monitoring">
  ## Paso 2: Supervisión del rendimiento en el lado del servidor
</div>

Para capturar datos de rendimiento en el lado del servidor, envuelve tu componente raíz de Remix (`root.tsx`) con `withSentry`:

```tsx {filename: root.tsx}
import {
  Links,
  LiveReload,
  Meta,
  Outlet,
  Scripts,
  ScrollRestoration,
} from "@remix-run/react";

import { withSentry } from "@sentry/remix";

function App() {
  return (
    <html>
      <head>
        <Meta />
        <Links />
      </head>
      <body>
        <Outlet />
        <ScrollRestoration />
        <Scripts />
        <LiveReload />
      </body>
    </html>
  );
}

export default withSentry(App);
```

<div id="step-4-verify-your-setup">
  ## Paso 4: Verificar la configuración
</div>

Si aún no has probado tu configuración de Sentry, hagámoslo ahora. Puedes confirmar que Sentry funciona correctamente y envía datos a tu proyecto de Sentry usando la página de ejemplo creada por el asistente de instalación:

1. Abre la página de ejemplo `/sentry-example-page` en tu navegador. En la mayoría de las aplicaciones Remix, estará en localhost.
2. Haz clic en el botón &quot;Throw Sample Error&quot; para generar un error y una traza en el frontend.

<Alert level="success" title="Consejo">
  No olvides explorar el código de la página de ejemplo en tu proyecto para entender qué ocurre después de hacer clic en el botón.
</Alert>
