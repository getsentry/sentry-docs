_Nombre de importación: `Sentry.contextLinesIntegration`_

Esta integración está habilitada de forma predeterminada. Si deseas modificar tus integraciones predeterminadas, lee [esto](./../#modifying-default-integrations).

Esta integración agrega código fuente a tus marcos de pila. Te permite ver, directamente en la interfaz de Sentry, el código que provocó el error.

<PlatformContent includePath="configuration/contextlines" />

<div id="options">
  ## Opciones
</div>

<div id="framecontextlines">
  ### `frameContextLines`
</div>

_Tipo: `number`_

Número de líneas que se recopilan alrededor del número de línea de cada marco de la pila. Valor predeterminado: 7.
