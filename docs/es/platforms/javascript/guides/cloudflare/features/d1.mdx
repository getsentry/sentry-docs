---
title: Instrumentación de Cloudflare D1
description: "Aprende a añadir instrumentación de spans para Cloudflare D1."
---

Puedes usar el método `instrumentD1WithSentry` para instrumentar [Cloudflare D1](https://developers.cloudflare.com/d1/),
la base de datos SQL sin servidor de Cloudflare, con Sentry.

```javascript
import * as Sentry from "@sentry/cloudflare";

// env.DB es el binding de la base de datos D1 configurado en tu `wrangler.jsonc`/`wrangler.toml`
const db = Sentry.instrumentD1WithSentry(env.DB);
// Ahora puedes usar la base de datos con normalidad
await db.prepare("SELECT * FROM table WHERE id = ?").bind(1).run();
```
