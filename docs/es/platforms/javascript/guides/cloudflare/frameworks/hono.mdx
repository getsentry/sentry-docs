<div id="migration-from-community-middleware">
  ### Migración desde el middleware de la comunidad
</div>

Si actualmente utilizas el middleware `@hono/sentry`, migra al middleware oficial `@sentry/cloudflare`:

```javascript
// Nuevo enfoque con el SDK oficial de Sentry
import { Hono } from 'hono';
import * as Sentry from '@sentry/cloudflare';

const app = new Hono();

// Envuelve tu aplicación con Sentry
export default Sentry.withSentry(
  (env: Env) => ({
    dsn: '___PUBLIC_DSN___',
    tracesSampleRate: 1.0,
  }),
  app
);
```
