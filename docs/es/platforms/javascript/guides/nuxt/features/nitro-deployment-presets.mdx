El SDK de Sentry para Nuxt ofrece soporte para varios perfiles de implementación de Nitro, la mayoría de los cuales **funcionan sin configuración adicional**.

<div id="cloudflare-pages">
  ## Cloudflare Pages
</div>

Usar el plugin Cloudflare Nitro de Sentry habilita el monitoreo de Sentry en Cloudflare Workers y Pages. Consulta la [documentación del plugin Cloudflare Nitro](/es/platforms/javascript/guides/cloudflare/frameworks/nuxt).
