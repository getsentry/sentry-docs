---
title: Vue Router
description: "Conoce la integración de enrutamiento de Vue de Sentry."
---

Nuxt utiliza `vue-router` internamente, y el SDK de Sentry para Nuxt lo instrumenta automáticamente para crear nombres de transacción parametrizados para las cargas de página y las navegaciones. Por ejemplo, una navegación a `/users/123` se registrará como una transacción llamada `/users/:id`.

Esta funcionalidad está habilitada de forma predeterminada y no requiere configuración adicional.
