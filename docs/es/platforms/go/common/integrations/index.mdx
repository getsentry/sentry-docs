<div id="default-integrations">
  ## Integraciones predeterminadas
</div>

Estas integraciones están habilitadas de forma predeterminada y se integran en la biblioteca estándar o en el propio runtime de Go. Están documentadas para que sepas qué hacen y puedas deshabilitarlas si causan problemas.

Para deshabilitar las integraciones predeterminadas, puedes proporcionar una lista vacía de integraciones al llamar a `sentry.Init()`.

```go
sentry.Init(sentry.ClientOptions{
    Dsn: "___PUBLIC_DSN___",
	Integrations: func(i []sentry.Integration) []sentry.Integration {
		return []sentry.Integration{}
	},
})
```


<div id="modulesintegration">
  ### ModulesIntegration
</div>

Esta integración registra todos los módulos de Go utilizados en tu aplicación, incluida su versión, como parte de los detalles del evento.

<div id="environmentintegration">
  ### EnvironmentIntegration
</div>

Esta integración enriquece los datos del evento con información específica del entorno de ejecución de Go y del sistema operativo, como la cantidad de núcleos/hilos de CPU y la versión del entorno de ejecución.

<div id="ignoreerrorsintegration">
  ### IgnoreErrorsIntegration
</div>

Esta integración te permite ignorar determinados patrones de error mediante expresiones regulares. Se asegura de que los errores que coincidan con los patrones especificados no se envíen a Sentry.

<div id="ignoretransactionsintegration">
  ### IgnoreTransactionsIntegration
</div>

Esta integración te permite ignorar transacciones específicas que coincidan con los patrones indicados. Es útil cuando ciertos nombres de transacción no deberían registrarse.

<div id="contextifyframesintegration">
  ### ContextifyFramesIntegration
</div>

Esta integración captura el contexto de las líneas de código que provocaron un error. Proporciona una instantánea del código fuente, incluidas las líneas anteriores y posteriores al error, para mejorar la depuración.

<div id="globaltagsintegration">
  ### GlobalTagsIntegration
</div>

Esta integración añade etiquetas globales a todos los eventos. Puedes definirlas como variables de entorno con el prefijo `SENTRY_TAGS_` o en las opciones del cliente.

<div id="disabling-certain-integrations">
  ### Deshabilitar ciertas integraciones
</div>

Puedes personalizar la lista de integraciones sin deshabilitar todas las predeterminadas usando la opción Integrations. En el siguiente ejemplo, todas las integraciones están habilitadas excepto la integración ContextifyFrames:

```go
sentry.Init(sentry.ClientOptions{
	Integrations: func(integrations []sentry.Integration) []sentry.Integration {
		var filteredIntegrations []sentry.Integration
		for _, integration := range integrations {
			if integration.Name() == "ContextifyFrames" {
				continue
			}
			filteredIntegrations = append(filteredIntegrations, integration)
		}
		return filteredIntegrations
	},
})
```
