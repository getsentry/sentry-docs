---
title: Apagado y vaciado
sidebar_order: 70
description: "Obtén más información sobre el comportamiento predeterminado de nuestro SDK si la aplicación se apaga de forma inesperada."
---

De forma predeterminada, el SDK utiliza un transporte asíncrono que se ejecuta en su propia goroutine independiente para enviar eventos por la red. Como resultado, algunos eventos pueden perderse si la aplicación se apaga inesperadamente. El SDK ofrece mecanismos para manejar este escenario.
