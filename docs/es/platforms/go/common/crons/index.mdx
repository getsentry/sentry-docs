<div id="alerts">
  ## Alertas
</div>

Cuando tu trabajo recurrente no realiza el check-in (perdido), supera el tiempo máximo de ejecución configurado (fallido) o informa manualmente un error, Sentry creará un evento de error con una etiqueta para tu monitor.

Para recibir alertas sobre estos eventos:

1. Ve a **Alerts** en la barra lateral.
2. Crea una nueva alerta y selecciona &quot;Issues&quot; bajo &quot;Errors&quot; como tipo de alerta.
3. Configura tu alerta y define un filtro para usar: `The event's tags match {key} {match} {value}`.

Ejemplo: `The event's tags match monitor.slug equals my-monitor-slug-here`
