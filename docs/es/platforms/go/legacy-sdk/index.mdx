Raven-Go proporciona una implementación del cliente de Sentry para el lenguaje de programación Go.

<div id="getting-started">
  ## Primeros pasos
</div>

Comenzar con Sentry es un proceso de tres pasos:

1. [Crea una cuenta](https://sentry.io/signup/)
2. [Instala tu SDK](#install)
3. [Configúralo](#configure)

<div id="installation">
  ## Instalación
</div>

Puedes instalar Raven-Go como cualquier otra biblioteca de Go con `go get`:

```bash
go get github.com/getsentry/raven-go
```

<div id="configuration">
  ## Configuración
</div>

Para usar `raven-go`, debes importar el paquete `raven` y luego inicializar tu DSN de forma global. Si defines la variable de entorno `SENTRY_DSN`, esto se hará automáticamente. La versión (release) y el entorno también pueden definirse mediante las variables de entorno `SENTRY_RELEASE` y `SENTRY_ENVIRONMENT`, respectivamente.
Más información en la sección [Configuración](/es/platforms/go/legacy-sdk/config/).

```go
package main

import "github.com/getsentry/raven-go"

func init() {
	raven.SetDSN("___PUBLIC_DSN___")
}
```
