Cuando un usuario se encuentra con un error, Sentry permite recopilar comentarios adicionales. Puedes recopilar comentarios según el método que admita el SDK.

<div id="crash-report-modal">
  ## Modal de informe de fallos
</div>

Nuestro modal de informe de fallos, incrustable y basado en JavaScript, es útil cuando normalmente mostrarías una página de error simple (la clásica `500.html`) en tu sitio web.

Para recopilar comentarios, el modal de informe de fallos solicita y recopila el nombre del usuario, su dirección de correo electrónico y una descripción de lo ocurrido. Cuando se envían comentarios, Sentry los asocia con el evento original, brindándote información adicional sobre los problemas.
