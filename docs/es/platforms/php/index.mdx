</div>

* Necesitas una [cuenta de Sentry](https://sentry.io/signup/) y un proyecto
* Tu aplicación debe ejecutarse en PHP 7.2 o posterior
* Lee una de estas guías si usas <PlatformLink to="/guides/laravel">Laravel</PlatformLink> o <PlatformLink to="/guides/symfony">Symfony</PlatformLink>

<div id="install">
  ## Instalación
</div>

Instala el SDK de Sentry con [Composer](https://getcomposer.org/):

```bash
composer require sentry/sentry
```

<OnboardingOption optionId="profiling">
  Para usar Profiling, también necesitas instalar la extensión Excimer mediante PECL:

  ```bash
  pecl install excimer
  ```

  La extensión de PHP Excimer es compatible con PHP 7.2 y posteriores. Excimer requiere Linux o macOS y no es compatible con Windows. Para otras formas de instalar Excimer, [consulta la documentación](/es/platforms/php/profiling/#installation).
</OnboardingOption>

<div id="configure">
  ## Configurar
</div>

Para capturar todos los errores, incluidos los que se producen durante el arranque de tu aplicación, debes inicializar el SDK de Sentry para PHP lo antes posible.

```php
\Sentry\init([
  'dsn' => '___PUBLIC_DSN___',
  // Agrega encabezados de la solicitud, cookies y la dirección IP,
  // consulta https://docs.sentry.io/platforms/php/data-management/data-collected/ para obtener más información
  'send_default_pii' => true,
  // ___PRODUCT_OPTION_START___ performance
  // Especifica una tasa de muestreo fija
  'traces_sample_rate' => 1.0,
  // ___PRODUCT_OPTION_END___ performance
  // ___PRODUCT_OPTION_START___ profiling
  // Establece una tasa de muestreo para el perfilado; esto es relativo a traces_sample_rate
  'profiles_sample_rate' => 1.0,
  // ___PRODUCT_OPTION_END___ profiling
  // ___PRODUCT_OPTION_START___ logs

  // Habilita el envío de logs a Sentry
  'enable_logs' => true,
  // ___PRODUCT_OPTION_END___ logs
]);
```

<Alert level="warning">
  Para incluir los argumentos del stack trace en tus errores, asegúrate de establecer `zend.exception_ignore_args: Off` en tu php.ini
</Alert>

<div id="verify">
  ## Verificar
</div>

En PHP puedes capturar una excepción ya atrapada o capturar el último error con captureLastError.

```php
try {
  $this->functionFailsForSure();
} catch (\Throwable $exception) {
  \Sentry\captureException($exception);
}
```
