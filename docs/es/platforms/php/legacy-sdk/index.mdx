<div id="installation">
  ## Instalación
</div>

Hay varias maneras de instalar la integración de PHP para Sentry. La forma recomendada es usar [Composer](https://getcomposer.org/):

```bash
composer require sentry/sentry "^1.0"
```

Alternativamente, puedes instalarlo manualmente:

1. Descarga y extrae el archivo [sentry-php](https://github.com/getsentry/sentry-php/archive/master.zip) más reciente en tu proyecto PHP.
2. Incluye el autoloader en tu aplicación:

   ```php
   require_once '/path/to/Raven/library/Raven/Autoloader.php';
   Raven_Autoloader::register();
   ```

<div id="configuration">
  ## Configuración
</div>

Lo más importante es crear el cliente de Raven. Créalo una vez y haz referencia a él desde cualquier lugar donde quieras interactuar con Sentry:

```php
$client = new Raven_Client('___PUBLIC_DSN___');
```

Una vez que tengas el cliente, puedes usarlo manualmente o habilitar la captura automática de errores y excepciones, que es lo recomendado:

```php
$error_handler = new Raven_ErrorHandler($client);
$error_handler->registerExceptionHandler();
$error_handler->registerErrorHandler();
$error_handler->registerShutdownFunction();
```

<div id="adding-context">
  ## Agregar contexto
</div>

Gran parte de la utilidad de Sentry proviene de los datos de contexto adicionales que acompañan a los eventos. El cliente de PHP lo hace muy sencillo al ofrecer métodos para establecer datos de contexto locales al hilo, que luego se envían automáticamente con todos los eventos. Por ejemplo, puedes usar el método `user_context` para agregar información sobre el usuario actual:

```php
$client->user_context(array(
    'email' => $USER->getEmail()
));
```

Para más información, consulta [Providing Request Context](/es/platforms/php/legacy-sdk/config/#sentry-php-request-context).
