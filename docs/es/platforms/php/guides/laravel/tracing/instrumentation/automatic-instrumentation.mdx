---
title: Instrumentación automática
sidebar_order: 10
description: "Descubre qué instrumentación captura transacciones automáticamente."
---

El SDK de Laravel instrumenta automáticamente tu aplicación para capturar transacciones y añade spans para las siguientes funcionalidades:

* Vistas
* Trabajos en cola
* Cachés (cuando se usa Laravel v11.11.0 o superior)
* Notificaciones
* Consultas a la base de datos
* Operaciones de Redis
* Solicitudes del cliente HTTP de Laravel (cuando se usa Laravel v8.45.0 o superior)

La instrumentación automática también está disponible con los siguientes paquetes:

* Operaciones GraphQL de [Lighthouse](https://lighthouse-php.com/)
* Rutas basadas en páginas de [Laravel Folio](https://laravel.com/docs/11.x/folio)
* Componentes de [Laravel Livewire](https://livewire.laravel.com/)

El SDK de Laravel también puede crear spans para operaciones de acceso al sistema de archivos.
Puedes habilitar esta función envolviendo la configuración de todos los discos
con una llamada a `Sentry\Laravel\Features\Storage\Integration::configureDisks()`:

```php {filename:config/filesystems.php}
'disks' => Sentry\Laravel\Features\Storage\Integration::configureDisks([
    'local' => [
        'driver' => 'local',
        'root' => storage_path('app'),
        'throw' => false,
    ],

    // ...
], /* enableSpans: */ true, /* enableBreadcrumbs: */ true),
```

Como alternativa, puedes habilitar esta función solo para discos específicos:

```php {filename:config/filesystems.php}
'disks' => [
    'local' => [
        'driver' => 'local',
        'root' => storage_path('app'),
        'throw' => false,
    ],

    's3' => Sentry\Laravel\Features\Storage\Integration::configureDisk('s3', [
        // ...
    ], /* enableSpans: */ true, /* enableBreadcrumbs: */ true),
],
```

De forma predeterminada, tanto los spans como las migas de pan (breadcrumbs) están habilitados. Puedes deshabilitarlos pasando el segundo argumento, `$enableSpans`, o el tercer argumento, `$enableBreadcrumbs`.
