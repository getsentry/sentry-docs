<Alert>
  Laravel 4.x es compatible hasta la versión 0.8.x del paquete `sentry/sentry-laravel`.
</Alert>

Laravel es compatible mediante un paquete nativo: [sentry-laravel](https://github.com/getsentry/sentry-laravel).

Esta guía es para Laravel 4.x. También ofrecemos instrucciones para [la última versión de Laravel](/es/platforms/php/guides/laravel/) así como [instrucciones específicas para Lumen](/es/platforms/php/guides/laravel/other-versions/lumen/).

<div id="install">
  ## Instalación
</div>

Instala el paquete `sentry/sentry-laravel`:

```shell
composer require "sentry/sentry-laravel:^0.8"
```

Agrega el proveedor de servicios y la fachada de Sentry en `config/app.php`:

```php {filename:config/app.php}
'providers' => array(
    // ...
    'Sentry\SentryLaravel\SentryLaravelServiceProvider',
),
'aliases' => array(
    // ...
    'Sentry' => 'Sentry\SentryLaravel\SentryFacade',
)
```

<div id="configure">
  ## Configuración
</div>

Publica el archivo de configuración de Sentry (`config/sentry.php`):

```bash
php artisan config:publish sentry/sentry-laravel
```

Añade tu DSN a `config/sentry.php`:

```php {filename:config/sentry.php}
<?php

return array(
    'dsn' => '___PUBLIC_DSN___',

    // ...
);
```

<div id="usage">
  ## Uso
</div>

Si quieres integrar Sentry en cualquier lugar fuera de los controladores de errores estándar, o si necesitas configurar opciones adicionales, puedes acceder a la instancia de Sentry a través de `$app`:

```php
$app['sentry']->setRelease(Git::sha());
```
