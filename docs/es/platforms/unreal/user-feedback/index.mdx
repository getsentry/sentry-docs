Cuando un usuario se encuentra con un error, Sentry permite recopilar comentarios adicionales. Puedes recopilar comentarios según el método que admita el SDK.

<div id="user-feedback-api">
  ## API de comentarios de usuario
</div>

La API de comentarios de usuario te permite recopilar comentarios usando tu propia interfaz. Puedes usar el mismo lenguaje de programación que ya empleas en tu app para enviar comentarios de usuario. En este caso, el SDK crea la solicitud HTTP para que no tengas que ocuparte de enviar datos por HTTP.
