Sentry es compatible con [PlayStation](/es/platforms/playstation/), [Xbox](/es/platforms/xbox/) y [Nintendo Switch](/es/platforms/nintendo-switch/) mediante las extensiones del SDK de Unreal Engine. Una vez que completes el proceso de verificación del middleware, nos pondremos en contacto contigo y te enviaremos una invitación a nuestros repositorios privados de GitHub con código específico para consolas.
Esto permite que tu configuración y los datos personalizados definidos mediante C++ o Blueprints aparezcan tanto en eventos no fatales como en volcados de fallos en estas plataformas de consola.

<Alert>
  El acceso a los informes de errores y fallos de Sentry para consolas es una función de pago. Revisaremos contigo el precio y los detalles durante el proceso de verificación.
</Alert>
