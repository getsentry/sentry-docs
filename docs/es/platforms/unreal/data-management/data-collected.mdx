---
title: Datos recopilados
description: "Consulta qué datos recopila el SDK de Sentry."
sidebar_order: 1
---

Sentry se toma muy en serio la privacidad de los datos y cuenta con configuraciones predeterminadas que priorizan su seguridad, especialmente cuando se trata de información de identificación personal (PII). Al añadir el SDK de Sentry a tu aplicación, permites que recopile datos y los envíe a Sentry durante la ejecución de la aplicación.

Los tipos de categorías y la cantidad de datos recopilados varían según las integraciones que tengas habilitadas en el SDK de Sentry. A continuación, encontrarás una lista de las categorías de datos que recopila el SDK de Sentry para Unreal Engine:

<div id="users-ip-addresses">
  ## Direcciones IP de los usuarios
</div>

De forma predeterminada, el SDK de Sentry no envía la dirección IP del usuario. Una vez habilitado, los servicios de backend de Sentry inferirán la dirección IP del usuario a partir de la solicitud entrante, a menos que ciertas integraciones que puedes habilitar sobrescriban este comportamiento.

Para habilitar el envío de la dirección IP del usuario, establece <PlatformLink to="/configuration/options/#SendDefaultPii">`sendDefaultPii=true`</PlatformLink>.

<div id="screenshots">
  ## Capturas de pantalla
</div>

La <PlatformLink to="/enriching-events/screenshots">función de capturas de pantalla</PlatformLink> está desactivada de forma predeterminada, pero al habilitarla, las capturas pueden contener datos personales (PII).

<div id="device-information">
  ## Información del dispositivo
</div>

El SDK de Sentry recopila información sobre el dispositivo, como el nombre, la versión y la compilación de tu sistema operativo o distribución de Linux. Esta información se envía a Sentry de forma predeterminada.

<div id="thread-stack-information">
  ## Información de la pila de hilos
</div>

En el momento del bloqueo, la pila de cada hilo se recopila y se envía a Sentry como parte de la instantánea de Minidump para los backends `crashpad` y `breakpad`. Esta información se envía a Sentry de forma predeterminada, pero se descarta después de procesar el evento en el backend.

Estos archivos no se almacenan por defecto, pero puedes <PlatformLink to="/data-management/store-minidumps-as-attachments/">habilitar el almacenamiento de Minidumps</PlatformLink> en la configuración de la organización o del proyecto en Sentry.
