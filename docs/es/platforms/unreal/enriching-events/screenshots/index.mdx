Sentry permite tomar automáticamente una captura de pantalla e incluirla como un <PlatformLink to="/enriching-events/attachments/">archivo adjunto</PlatformLink> cuando un usuario se encuentra con un error, una excepción o un fallo.

Esta función solo está disponible para SDK con interfaz de usuario, como los de aplicaciones móviles y de escritorio, y además depende de si es posible tomar una captura. Por ejemplo, en algunos entornos, como iOS nativo, realizar una captura requiere el hilo de la interfaz, que a menudo no está disponible en caso de fallo. Otro caso en el que puede no haber una captura disponible es cuando el evento ocurre antes de que la pantalla empiece a cargarse. Por lo tanto, esta función es, por naturaleza, una solución de “mejor esfuerzo”.
