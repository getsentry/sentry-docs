<div id="performance-considerations">
  ## Consideraciones de rendimiento
</div>

- Los registros se envían de forma asíncrona para no afectar el rendimiento del juego
- Considera desactivar los registros de nivel Debug en producción para evitar un volumen excesivo
- Cada nivel de gravedad puede controlarse de forma individual para afinar qué se envía a Sentry
- Usa categorías para organizar los registros y facilitar su búsqueda y filtrado
- Los manejadores previos al registro se ejecutan de forma síncrona, así que mantén el procesamiento ligero
