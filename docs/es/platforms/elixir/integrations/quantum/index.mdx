La integración de [Quantum](https://github.com/quantum-elixir/quantum-core) admite la supervisión de cron para trabajos de Quantum.

Esta integración está integrada en el SDK de Sentry.

<div id="configure">
  ## Configurar
</div>

Puedes configurar la integración de Quantum en la configuración de Sentry, en la clave `:integrations`:

```elixir {filename:config/config.exs} {diff}
config :sentry,
  # ...,
  integrations: [
+    quantum: [
+      cron: [enabled: true]
+    ]
  ]
```
