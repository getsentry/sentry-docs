---

Esta integración conecta Sentry con la plataforma de revisión de accesos de Elba para ofrecer a las organizaciones una solución integral para gestionar y revisar el acceso de usuarios en Sentry. 
Ayuda a mejorar la seguridad, garantizar el cumplimiento y aumentar la eficiencia operativa al proporcionar:

- Reducción de riesgos: Las revisiones periódicas de acceso ayudan a identificar y eliminar accesos innecesarios o no autorizados, reduciendo el riesgo de filtraciones de datos y amenazas internas.
- Gestión proactiva: La detección temprana de anomalías en permisos permite aplicar acciones correctivas de forma oportuna.

Se conecta a la API de Sentry para obtener información detallada sobre los miembros de la organización, incluidos sus roles, pertenencia a equipos y niveles de permiso. Los datos obtenidos se presentan en la plataforma de Elba, donde los administradores pueden revisar de forma sistemática los derechos de acceso de cada usuario. Además, la plataforma analiza los permisos de los usuarios para identificar posibles riesgos de seguridad, como privilegios excesivos o accesos obsoletos, y proporciona recomendaciones para su mitigación.

Esta integración es mantenida y respaldada por Elba. Para más detalles o preguntas, no dudes en escribir a support@elba.security.

<div id="install-and-configure">
  ## Instalar y configurar
</div>

<Alert>

Para instalar esta integración se requieren permisos de Sentry Owner, Manager o Admin.

</Alert>

1. Ve a **Settings > Integrations > Elba**

2. Sigue las [instrucciones de instalación completas de Elba](https://sentry.elba.security/install).
