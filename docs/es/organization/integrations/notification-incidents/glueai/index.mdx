La integración de Glue.ai permite recibir notificaciones de alertas de métricas a través de Glue para que puedas debatir y priorizar de forma eficiente.

Esta integración es mantenida y respaldada por Glue.ai. Para más detalles, consultas o soporte, no dudes en escribir a support@glue.ai.

<div id="install-and-configure">
  ## Instalación y configuración
</div>

<Alert>

Necesitas permisos de Owner, Manager o Admin de Sentry para instalar esta integración.

</Alert>

1. Ve a **Settings > Integrations > Glue.ai**

2. Sigue las [instrucciones completas de instalación de Glue.ai](https://docs.glue.ai/integrations/sentry-coming-soon).
