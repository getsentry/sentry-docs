La integración de Cased es una completa herramienta de monitoreo DevOps que se conecta a la perfección con Sentry. Está diseñada para leer y consultar datos de Sentry y brindar a nuestros usuarios información contextual detallada sobre el estado de la aplicación y sus incidencias. 
Las funciones clave incluyen:

- Consultas de datos en tiempo real desde Sentry para obtener información accionable sobre el rendimiento de la aplicación y el seguimiento de errores.
- Contexto de incidencias mejorado para facilitar una respuesta a incidentes y una depuración rápidas y efectivas.
- Recepción de alertas de webhook de Sentry para activar nuestro flujo de trabajo de respuesta a incidentes, incluyendo la agregación de alertas, la desduplicación y el enrutamiento inteligente de notificaciones.
- Paneles personalizables que ofrecen una vista general de los eventos de Sentry, lo que permite una evaluación rápida y la priorización de respuestas.
- Integración con herramientas de comunicación para agilizar la gestión de incidentes y la coordinación del equipo durante interrupciones o eventos de alta severidad.

Esta integración es mantenida y soportada por Cased. Para más detalles o preguntas, no dudes en escribir a support@cased.com.
