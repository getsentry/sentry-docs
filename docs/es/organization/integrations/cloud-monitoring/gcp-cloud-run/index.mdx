Cloud Run ayuda a los desarrolladores a ahorrar tiempo al crear y desplegar sus aplicaciones. Sentry ayuda a ahorrar tiempo al resolver problemas en producción al ofrecer funciones detalladas de depuración.

<div id="install-and-configure">
  ## Instalación y configuración
</div>

<Alert>
  Se necesitan permisos de propietario, gestor o administrador de Sentry para instalar esta integración.
</Alert>

1. [Crea una cuenta](https://sentry.io/signup/) (o [inicia sesión](https://sentry.io/auth/login/)) en Sentry

2. Elige la plataforma de tu aplicación

   ![Sentry Choose Platform](./img/sentry-choose-platform.png)

3. Sigue las instrucciones para instrumentar tu aplicación de Cloud Run. Consulta la documentación detallada de la plataforma [aquí](/es/platforms/).

4. Despliega tu aplicación de Cloud Run en GCP.
