<div id="sentry-integration-with-expo-dashboard">
  ## Integración de Sentry con el panel de Expo
</div>

<Alert level="warning">
  Si cambias el slug de tu organización, deberás actualizar la configuración de esta integración. Más información en nuestra [guía de solución de problemas](/es/organization/integrations/troubleshooting).
</Alert>

La integración de Sentry con Expo te permite ver informes de fallos y [reproducciones de sesión](/es/platforms/react-native/session-replay/) de los lanzamientos de tu app de Expo directamente en el panel de Expo Application Services (EAS). Esta integración proporciona un enlace directo a las trazas de pila de Sentry con contexto completo, reproducciones de sesión y herramientas de depuración.

<div id="install">
  ### Instalar
</div>

<Alert>
  Se requieren permisos de propietario, gestor o administrador de Sentry para instalar esta integración.
</Alert>

1. Inicia sesión en tu panel de Expo y abre **Account Settings &gt; Overview** (`https://expo.dev/accounts/[your-account]/settings`).
2. Ve a la sección **Connections** y haz clic en **Connect** junto a Sentry.
3. Inicia sesión en Sentry y acepta la integración en tu organización. Serás redirigido de vuelta a Account Settings &gt; Overview.

<div id="configure">
  ### Configuración
</div>

<div id="link-your-projects">
  #### Vincula tus proyectos
</div>

Después de conectar tus cuentas, debes vincular tu proyecto del panel de Expo con tu proyecto de Sentry:

1. Abre **Projects &gt; [Your Project] &gt; Configuration &gt; Project settings** en el panel de Expo.
2. Haz clic en **Link** y selecciona tu proyecto de Sentry en la lista desplegable.

<div id="usage">
  ### Uso
</div>

Para ver tus Issues y Replays de Sentry en el panel de control de Expo, primero debes asegurarte de haber creado un [canal del panel de Expo](https://docs.expo.dev/eas-update/how-it-works/) y de haber asignado compilaciones a ese canal para crear un [despliegue de EAS](https://docs.expo.dev/eas-update/deployment/). Una vez hecho esto, puedes ver tus datos de Sentry de la siguiente manera:

1. Abre **Projects &gt; [Your Project] &gt; Updates &gt; Deployments &gt; [Deployment]** para ver los datos de Sentry de una Release.

<Alert level="info">
  Para aprovechar esta integración, necesitas agregar el SDK de Sentry para React Native a tu aplicación de Expo. Puedes configurar el SDK fácilmente ejecutando el asistente de Sentry:

  ```bash
  npx @sentry/wizard@latest -i reactNative
  ```

  Para más detalles sobre el uso de Sentry con Expo, consulta la [documentación de Sentry](/es/platforms/react-native/) para aplicaciones de React Native.
</Alert>
