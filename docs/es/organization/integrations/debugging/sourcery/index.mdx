description: "Conoce la integración Sourcery de Sentry, que puede usarse para generar una pull request para corregir problemas y explicar cómo se solucionarán."
---

Sourcery realiza revisiones de código en las pull requests para ofrecer comentarios inmediatos sobre tu código. Esta integración es mantenida y soportada por Sourcery. Para más detalles, preguntas o soporte, no dudes en escribir a hello@sourcery.ai.

<div id="install-and-configure">
  ## Instalación y configuración
</div>

<Alert>

Se requieren permisos de Owner, Manager o Admin de Sentry para instalar esta integración.

</Alert>

1. Ve a **Settings > Integrations > Sourcery**

2. Sigue las [instrucciones completas de instalación de Sourcery](https://docs.sourcery.ai/Integrations/sentry/).
