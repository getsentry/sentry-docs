---

CodeComet desarrolló un agente de depuración basado en LLM que analiza los detalles de los errores e inspecciona los repositorios de código correspondientes para generar correcciones. El resultado es una explicación junto con código listo para que un desarrollador lo revise.
Esta integración es mantenida y respaldada por CodeComet. Para más detalles, preguntas o soporte, no dudes en escribir a support@codecomet.io.

<div id="install-and-configure">
  ## Instalación y configuración
</div>

<Alert>

Se requieren permisos de Sentry Owner, Manager o Admin para instalar esta integración.

</Alert>

1. Ve a **Settings > Integrations > CodeComet**

2. Sigue las [instrucciones completas de instalación de CodeComet](https://www.codecomet.io/docs/integrations#section4).
