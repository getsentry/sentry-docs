---

Tembo lleva el monitoreo de Postgres a los PR. Identifica de forma autónoma los cuellos de botella de rendimiento en la base de datos, escribe correcciones de código optimizadas y envía PR listos para desplegar directamente a tus repositorios. Esta integración es mantenida y respaldada por Tembo. Para más detalles, consultas o soporte, no dudes en escribir a support@tembo.io.

<div id="install-and-configure">
  ## Instalación y configuración
</div>

<Alert>

Se requieren permisos de Sentry Owner, Manager o Admin para instalar esta integración.

</Alert>

1. Ve a **Settings > Integrations > Tembo**

2. Sigue las [instrucciones completas de instalación de Tembo](https://docs.tembo.io/integrations/sentry).
