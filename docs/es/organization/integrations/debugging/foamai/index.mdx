<div id="install-and-configure">
  ## Instalación y configuración
</div>

<Alert>
  Para instalar esta integración, necesitas permisos de propietario, manager o administrador de Sentry.
</Alert>

1. Ve a **Settings &gt; Integrations &gt; Foam**

2. Sigue la [guía interactiva de configuración de Foam](https://foam.ai/docs)
