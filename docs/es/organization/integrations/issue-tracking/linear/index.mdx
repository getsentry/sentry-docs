Linear es una herramienta de seguimiento de incidencias diseñada para la rapidez, que optimiza la gestión de proyectos de software, tareas y el seguimiento de errores. Crea incidencias nuevas de Linear desde Sentry sin esfuerzo y vincula incidencias de Sentry con incidencias existentes de Linear.

Esta integración es mantenida y respaldada por la empresa que la creó. Para más detalles, consulta nuestra [documentación de la Plataforma de Integraciones](/es/organization/integrations/integration-platform/).

<div id="install-and-configure">
  ## Instalar y configurar
</div>

<Alert>
  Se requieren permisos de propietario, manager o administrador de Sentry para instalar esta integración.

  Linear **no** funciona con Sentry autogestionado.
</Alert>

1. Ve a **Settings &gt; Integrations &gt; Linear**

   ![Instalar la integración de Linear](./img/linear-install-pg.png)

2. Sigue las [instrucciones completas de instalación de Linear](https://linear.app/docs/sentry).

<div id="issue-management">
  ## Gestión de incidencias
</div>

El seguimiento de incidencias te permite crear incidencias de Linear directamente desde Sentry y vincular incidencias de Sentry con incidencias existentes de Linear. La gestión de incidencias se puede configurar de dos maneras: automáticamente o manualmente.

<div id="automatically">
  ### Automáticamente
</div>

Para configurar automáticamente la gestión de incidencias, crea una [alerta de incidencia](/es/product/alerts/alert-types/#issue-alerts) o una [alerta de métrica](/es/product/alerts/alert-types/#metric-alerts). Al seleccionar la [acción](/es/product/alerts/create-alerts/issue-alert-config/#then-conditions-actions), elige “Notify Integration” y luego “Linear” para las alertas de incidencia, o selecciona “Linear” en el menú desplegable de acciones para las alertas de métrica:

![Create Linear issue as metric alert rule action](./img/linear-rule.png)

Se creará automáticamente una incidencia en Linear cuando se active la alerta.

<div id="manually">
  ### Manualmente
</div>

Para configurar la gestión de incidencias manualmente, ve a una incidencia específica de Sentry y busca la sección &quot;Linked Issues&quot; en el panel derecho:

![Vincular incidencia de Linear desde una incidencia de Sentry](./img/linear-link-issue.png)

Aquí podrás crear o vincular incidencias de Linear.

![Formulario para crear incidencia de Linear](./img/linear-create-issue.png)
