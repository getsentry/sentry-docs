<Alert title="Beta">
  **Agente de Sentry para Linear** está en fase beta. Actualmente, solo está disponible para organizaciones de Sentry.io con sede en EE. UU. Se esperan cambios en la experiencia de usuario y en la funcionalidad del agente.
</Alert>

El Agente de Sentry para Linear permite iniciar ejecuciones de Seer Issue Fix directamente desde Linear. Estos resultados pueden utilizarse después con otros agentes en Linear para activar acciones adicionales.

Esta integración se beneficia especialmente de tener [la automatización configurada en Seer](/es/product/ai-in-sentry/seer/#automation), para crear automáticamente análisis de causa raíz y soluciones basadas en las puntuaciones de capacidad de corrección de Seer.

<div id="install-and-configure">
  ## Instalar y configurar
</div>

<Alert>
  Se requieren permisos de propietario, responsable o administrador de Sentry para instalar esta integración.

  Linear **no** funciona con Sentry autogestionado.
</Alert>

Ve a **Settings &gt; Integrations &gt; Linear (Sentry Agent)**, acepta e instala.

![Instalar Sentry Agent para Linear](./img/Linear-Sentry-Agent-Install.png)

<div id="interaction-model">
  ## Modelo de interacción
</div>

El Sentry Agent se puede usar de 2 formas:

1. Delegar el issue de Linear a Sentry mediante el flujo de asignación de Linear.
2. Comentar `@sentry` dentro del issue de Linear y hacer tu solicitud en lenguaje natural.

Cuando el issue se delega al agente, este verificará automáticamente si existe una ejecución completa de Issue Fix y mostrará el resultado. Si no existe, el agente solicitará ejecutar una.

Para las interacciones basadas en comentarios, la integración acepta lenguaje natural. Intentamos interpretar la intención de la pregunta y activar la respuesta adecuada. Si no lo logramos, mostraremos el comando de ayuda, que te guiará por las opciones.

Actualmente, la integración permite al usuario ejecutar las siguientes acciones:

* Issue Fix - Ejecuta un Seer Issue Fix para la aplicación.
* Status - Actualiza el estado del issue.
* Help - Muestra el comando de ayuda que te guiará por las opciones.

<div id="assigning-sentry-agent-to-an-issue">
  ## Asignar Sentry Agent a un issue
</div>

1. Ve a un issue en Linear y selecciona &quot;Assign&quot; en el menú desplegable de la barra lateral derecha.

   ![Seleccionar Sentry Agent para Linear](./img/Linear-Agent-Delegation.png)

2. Selecciona &quot;Sentry&quot; en el menú desplegable.

3. La tarea pasará a in-progress y se creará la sesión de Sentry Agent. Si es la primera vez que usas el Agent, se te pedirá que te autentiques. Usa el Button &quot;Link&quot; para vincular tu usuario de Linear con tu usuario de Sentry.

   ![Vincular Sentry Agent para Linear](./img/Linear-Linking-Auth.png)

4. Una vez vinculado, se te preguntará si quieres ejecutar un Seer Issue Fix para la aplicación. Puedes responder &quot;Yes&quot;, o pedir &quot;Help&quot; para ver más opciones.

   ![Ejecutar Seer en Linear](./img/Linear-Help.png)

5. Si respondes &quot;Yes&quot;, el Agent ejecutará un Seer Issue Fix para la aplicación. **Nota**: Puede que necesites pedir &quot;Status&quot; para ver los resultados.

   ![RCA de Sentry Seer en Linear](./img/Linear-Sentry-RCA-2.png)

<div id="uninstalling-the-integration">
  ## Desinstalar la integración
</div>

La integración de Sentry instala una aplicación basada en Linear dentro de tu entorno de Linear y también configura la integración en Sentry. Para eliminarla por completo, debes quitar ambas.

<div id="removing-from-sentry">
  ### Desinstalar desde Sentry
</div>

1. Ve a Settings &gt; Integrations &gt; Linear (Sentry Agent)

2. Selecciona &quot;Uninstall&quot;

   ![Desinstalar Sentry de las integraciones](./img/Linear-Agent-Uninstall.png)

<div id="removing-from-linear">
  ### Eliminar en Linear
</div>

1. Ve a Settings &gt; Applications &gt; Sentry

   ![Desinstalar Sentry de las aplicaciones de Linear](./img/Linear-App-Uninstall.png)

2. Selecciona &quot;...&quot; y luego &quot;Revoke Access&quot;

   ![Revocar acceso a Sentry desde las aplicaciones de Linear](./img/Linear-App-Revoke.png)
