<div id="install-and-configure">
  ## Instalar y configurar
</div>

<Alert>

Para instalar esta integración, necesitas permisos de propietario, manager o administrador de Sentry.

</Alert>

1. Ve a **Settings > Integrations > Goast**

2. Sigue las [instrucciones de instalación completas de Goast](https://support.goast.ai/integrations/sentry).
