La integración de Height con Sentry permite a tu equipo registrar y resolver rápidamente errores críticos al vincular tareas nuevas o existentes de Height con incidencias de Sentry. Al vincular tareas, tus compañeros obtienen visibilidad de las actualizaciones desde Sentry y pueden marcar automáticamente las incidencias de Sentry como *Resolved* a medida que cambia el estado de la tarea en Height. Además, configura alertas personalizadas para crear nuevas tareas de Height cuando los disparadores clave que te interesan —como errores reportados o bloqueos de sesión— superen cierto umbral.

Esta integración es mantenida y respaldada por la empresa que la creó. Para más detalles, consulta nuestra [documentación de la Plataforma de Integraciones](/es/organization/integrations/integration-platform/).

<div id="install-and-configure">
  ## Instalar y configurar
</div>

<Alert>
  Se requieren permisos de propietario, responsable o administrador de Sentry para instalar esta integración.

  Height **no** funciona con Sentry autogestionado/autohospedado.
</Alert>

Para añadir la integración de Height, ve a **Settings &gt; Integrations &gt; Height** y haz clic en &quot;Install&quot;.

![Installed Height integration](./img/height-install.png)

<div id="issue-management">
  ## Gestión de incidencias
</div>

El seguimiento de incidencias te permite crear tareas de Height desde [sentry.io](https://sentry.io) y vincular incidencias de Sentry a tareas de Height existentes. La gestión de incidencias puede configurarse de dos formas: automáticamente o manualmente.

<div id="automatically">
  ### Automáticamente
</div>

Para configurar la gestión de incidencias de forma automática, crea una [alerta de incidencia](/es/product/alerts/alert-types/#issue-alerts) o una [alerta de métrica](/es/product/alerts/alert-types/#metric-alerts). Al seleccionar la [acción](/es/product/alerts/create-alerts/issue-alert-config/#then-conditions-actions), elige “Notify Integration” y luego “Height” para las alertas de incidencia, o selecciona “Height” en el menú desplegable de acciones para las alertas de métrica:

![Configuración de la acción de regla de alerta de métrica de Height](./img/height-rule.png)

Se creará automáticamente una tarea en Height cuando se active la alerta.

<div id="manually">
  ### Manualmente
</div>

Para configurar la gestión de incidencias manualmente, ve a una incidencia específica de Sentry y busca la sección &quot;Linked Issues&quot; en el panel derecho:

![Vincular incidencia de Height en Sentry](./img/height-link-issue.png)

Aquí podrás crear o vincular tareas de Height:

![Formulario para crear incidencia de Height](./img/height-create-issue.png)

<div id="issue-sync">
  ## Sincronización de issues
</div>

Height mantendrá sincronizados los estados de tus issues. Hay una asignación predeterminada que mapea el estado de las tareas en Height con el estado de los issues en Sentry. Si resuelves una tarea en Height, el issue de Sentry vinculado manualmente también se marcará como resuelto.
