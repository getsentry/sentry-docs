Factory se integra con el código de los usuarios y extrae Issues para ayudar a automatizar la detección y solución de problemas.

Esta integración es mantenida y respaldada por Factory. Para más detalles, preguntas o soporte, no dudes en escribir a shashank@factory.ai.

<div id="install-and-configure">
  ## Instalar y configurar
</div>

<Alert>

Para instalar esta integración, necesitas permisos de propietario, manager o administrador de Sentry.

</Alert>

1. Ve a **Settings > Integrations > Factory**

2. Sigue las [instrucciones completas de instalación de Factory](https://docs.factory.ai/onboarding/integrating-with-your-engineering-system/sentry).
