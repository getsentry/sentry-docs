description: "Conoce la integración de Sentry con Plane."
---

La integración de Makeplane (Plane) permite crear automáticamente incidencias de Plane a partir de alertas de Sentry, admite vincular incidencias de Sentry con incidencias de Plane y mantiene sus estados sincronizados para optimizar el seguimiento y la resolución de errores. Esta integración es mantenida y soportada por Plane; para más información, escribe a support@plane.so.

<div id="install-and-configure">
  ## Instalación y configuración
</div>

<Alert>

Para instalar esta integración, necesitas permisos de propietario, administrador o manager de Sentry.

</Alert>

1. Ve a **Settings > Integrations > Makeplane**

2. Sigue las [instrucciones completas de instalación de Plane](https://docs.plane.so/integrations/sentry).
