Los errores, las funcionalidades y todo lo que rastrees en Shortcut ahora pueden estar juntos en un mismo lugar. La nueva integración de Shortcut tiene paridad de funciones con el plugin de Shortcut. Si estás eligiendo entre ambos, te recomendamos instalar la integración de Shortcut.

Esta integración es mantenida y respaldada por la empresa que la creó. Para más detalles, consulta nuestra [documentación de la plataforma de integraciones](/es/organization/integrations/integration-platform/).

<div id="install-and-configure">
  ## Instalar y configurar
</div>

<Alert>
  Para instalar esta integración se requieren permisos de propietario, manager o administrador de Sentry.

  Shortcut **no** funciona con Sentry autogestionado.
</Alert>

1. Ve a **Settings &gt; Integrations &gt; Shortcut**

![Instalar la integración de Shortcut](./img/shortcut.png)

2. Sigue las [instrucciones completas de instalación de Shortcut](https://help.shortcut.com/hc/en-us/articles/360026979272-Setting-up-the-Sentry-Integration).
