Las integraciones de Sentry que están disponibles como [acciones de reglas de alerta](/es/organization/integrations/integration-platform/#alerts) pueden recibir webhooks de alertas de métricas.

La demostración interactiva a continuación muestra cómo configurar una integración interna que reciba alertas de Sentry.

<Arcade src="https://demo.arcade.software/v7uhzmdV6Q5PDzoVPAE6?embed&show_copy_link=true" />

<div id="sentry-hook-resource-header">
  ## Encabezado Sentry-Hook-Resource
</div>

`'Sentry-Hook-Resource': 'metric_alert'`

<div id="attributes">
  ## Atributos
</div>

<div id="action">
  ### action
</div>

* type: string
* description: será uno de (`critical`, `warning`, `resolved`)

### data[&#39;metric&#95;alert&#39;]

* type: objeto
* description: el incidente que activó la regla de alerta

### data[&#39;metric&#95;alert&#39;][&#39;alert&#95;rule&#39;]

* type: object
* description: regla de alerta

### data[&#39;metric&#95;alert&#39;][&#39;alert&#95;rule&#39;][&#39;triggers&#39;]

* type: matriz de objetos
* description: los umbrales que activan la alerta

### data[&#39;metric&#95;alert&#39;][&#39;alert&#95;rule&#39;][&#39;triggers&#39;][0][&#39;actions&#39;]

* type: arreglo de objetos
* description: acciones que se ejecutarán cuando se alcance el umbral de un disparador

### data[&#39;description&#95;text&#39;]

* type: string
* description: una descripción legible por humanos de la alerta

### data[&#39;description&#95;title&#39;]

* type: string
* description: un título legible por humanos para la alerta

### data[&#39;web&#95;url&#39;]

* type: string
* description: la URL de la API del incidente

<Alert>
  Este campo es únicamente para [componentes de la interfaz de usuario de acciones de reglas de alerta](/es/organization/integrations/integration-platform/ui-components/alert-rule-action).
</Alert>

### data[&#39;metric&#95;alert&#39;][&#39;alert&#95;rule&#39;][&#39;triggers&#39;][0][&#39;actions&#39;][0][&#39;settings&#39;]

* type: object
* description: la configuración guardada para dirigir la alerta dentro del servicio externo

<div id="payload">

</div>

<Alert>

</Alert>
