OpenReplay es una herramienta de reproducción de sesiones para desarrolladores. Reproduce cada sesión de usuario junto con tus registros del front y del backend y otros datos distribuidos en tu stack para que puedas encontrar, reproducir y corregir errores más rápido.

Puedes encontrar las [instrucciones de instalación aquí](https://docs.openreplay.com/integrations/sentry).
