Para ciertas tareas, puedes usar el ejecutable de línea de comandos `sentry-cli`. Puede conectarse a la API de Sentry y administrar algunos datos de tus proyectos. Se utiliza principalmente para gestionar archivos de información de depuración en iOS y Android, así como la administración de versiones y source maps en otras plataformas.
