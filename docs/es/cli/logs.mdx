La herramienta `sentry-cli` se puede usar para ver y transmitir registros de tus proyectos de Sentry. Esto te permite supervisar los registros de tu aplicación directamente desde la línea de comandos.

<div id="requirements">
  ## Requisitos
</div>

Para usar el comando logs, necesitas:

* [Instalar](/es/cli/installation) la Sentry CLI
* [Autenticarte](/es/cli/configuration) en Sentry usando un token de autenticación
* Tener habilitados los registros para tu proyecto

<div id="configuration">
  ## Configuración
</div>

El comando logs utiliza tu token de autenticación de Sentry. Puedes configurarlo de las siguientes maneras:

```bash
sentry-cli login
```

O bien configurando la variable de entorno `SENTRY_AUTH_TOKEN`:

```bash
export SENTRY_AUTH_TOKEN=___ORG_AUTH_TOKEN___
```

Más información sobre el [archivo de configuración](/es/cli/configuration#configuration-file) de la CLI.

<div id="basic-usage">
  ## Uso básico
</div>

El comando logs admite dos operaciones principales: listar registros y transmitirlos en tiempo real.

<div id="list-logs">
  ### Listar registros
</div>

Para recuperar y mostrar registros:

```bash
sentry-cli logs list --project=my-project --org=my-org
```

<div id="stream-logs">
  ### Transmisión de registros
</div>

Para transmitir registros en tiempo real de forma continua:

```bash
sentry-cli logs list --project=my-project --org=my-org --live
```

El modo de transmisión en directo consultará continuamente si hay nuevos registros y los mostrará a medida que vayan llegando.

<div id="command-options">
  ## Opciones de comandos
</div>

<div id="project-and-organization">
  ### Proyecto y organización
</div>

Especifica explícitamente el proyecto y la organización:

```bash
sentry-cli logs list --project=my-project --org=my-org
```

O usa los valores predeterminados de tu archivo de configuración.

<div id="examples">
  ## Ejemplos
</div>

<div id="monitor-application-logs">
  ### Monitoriza los registros de la aplicación
</div>

Transmite los registros de tu aplicación en producción:

```bash
sentry-cli logs list --project=my-app --org=my-org --live
```

<div id="view-recent-logs">
  ### Ver registros recientes
</div>

Obtén y muestra los registros de tu proyecto:

```bash
sentry-cli logs list --project=my-app --org=my-org
```

<div id="output-format">
  ## Formato de salida
</div>

El comando logs muestra las entradas de registro con la siguiente información:

* Marca de tiempo
* Nivel de registro
* Mensaje
* ID de traza (si corresponde)

Las entradas de registro se muestran en un formato legible, lo que facilita supervisar la salida de registro de tu aplicación directamente desde la línea de comandos.
