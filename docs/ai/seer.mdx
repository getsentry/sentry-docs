Seer is Sentry's AI-powered debugging assistant that helps you understand and fix issues faster. It provides intelligent root cause analysis, fix suggestions, and automated issue triage directly within your Sentry workflow.

## What Seer Can Do

- **Root Cause Analysis** — Understand why errors occur with AI-powered explanations
- **Fix Suggestions** — Get actionable code suggestions to resolve issues
- **Issue Triage** — Automatically prioritize and categorize issues
- **Code Context** — Seer understands your codebase when connected to GitHub

## Getting Started with Seer

To get the most out of Seer, connect your GitHub repository:

1. Go to **Settings > Integrations > GitHub**
2. Install the Sentry GitHub App
3. Configure repository access

Once connected, Seer can analyze your source code to provide more accurate root cause analysis and fix suggestions.
