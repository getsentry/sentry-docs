Agent skills are instruction sets that teach AI coding assistants how to perform specific tasks. Sentry's official agent skills give your AI assistant the knowledge to set up and configure Sentry in your projectsâ€”no manual configuration required.

<StepComponent>

## Available Skills

Sentry provides skills for common integration tasks:

| Skill                        | Description                                                             |
| ---------------------------- | ----------------------------------------------------------------------- |
| `sentry-code-review`         | Analyze and resolve Sentry comments on GitHub Pull Requests             |
| `sentry-nextjs-wizard`       | Setup Sentry in Next.js projects using the Sentry Wizard CLI            |
| `sentry-setup-ai-monitoring` | Setup Sentry AI Agent Monitoring for OpenAI, Anthropic, LangChain, etc. |
| `sentry-setup-logging`       | Setup Sentry Logging for JavaScript, Python, and Ruby projects          |
| `sentry-setup-metrics`       | Setup Sentry Metrics (counters, gauges, distributions)                  |
| `sentry-setup-tracing`       | Setup Sentry Tracing and Performance Monitoring                         |

## Install

Choose your AI coding assistant and run the installation command. Skills can be installed at the user level (applies to all projects) or project level (single repository).

<SplitSection>
<SplitSectionText>

### Claude Code

Install skills for Claude Code (Anthropic's official CLI).

**User-level** installs to `~/.claude/skills/` and applies to all projects.

**Project-level** installs to `.claude/skills/` in your repository.

</SplitSectionText>
<SplitSectionCode>

```bash {tabTitle:User-level}
git clone https://github.com/getsentry/sentry-agent-skills.git /tmp/sentry-skills && \
  mkdir -p ~/.claude/skills && \
  cp -r /tmp/sentry-skills/skills/* ~/.claude/skills/ && \
  rm -rf /tmp/sentry-skills
```


## Usage

Once installed, your AI assistant will automatically discover the skills. Simply ask it to perform Sentry-related tasks:

```text
"Set up Sentry in my Next.js app"
```

```text
"Set up Sentry tracing in my project"
```

```text
"Add Sentry logging to my app"
```

```text
"Configure Sentry AI monitoring for my OpenAI integration"
```

```text
"Set up Sentry metrics"
```

```text
"Review the Sentry comments on PR #123"
```

The assistant will load the appropriate skill and guide you through the setup process automatically.

## Skill Format

These skills follow the [Agent Skills specification](https://agentskills.io/specification). Each skill contains a `SKILL.md` file with YAML frontmatter and markdown instructions:

```
skill-name/
  SKILL.md        # Required: YAML frontmatter + markdown instructions
```

**SKILL.md structure:**

```markdown
---
name: skill-name
description: Description of what this skill does and when to use it
---

# Skill Title

Instructions for the AI assistant...
```

## Contributing

Want to contribute a new skill? Skills must:

1. Follow the [Agent Skills specification](https://agentskills.io/specification)
2. Have a valid `name` (lowercase, hyphens, 1-64 characters)
3. Include a clear `description` (1-1024 characters)
4. Provide comprehensive instructions in the markdown body

Submit contributions to the [sentry-agent-skills repository](https://github.com/getsentry/sentry-agent-skills).
