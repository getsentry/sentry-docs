- Kotlin plugin is upgraded to `1.6.10`.

This version is still compatible with Kotlin `1.4`.

<Alert>

This only affects integrations that are written in Kotlin, such as `timber`, `fragment`, `okhttp`, `navigation` and `compose`.

</Alert>
