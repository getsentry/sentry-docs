Node.js integrations support tracking database queries as spans.

## Supported Platforms

Auto-discovered:

- `pg` (Postgres)
- `pg-native` (Postgres)
- `mongodb` (Mongo)
- `mongoose` (Mongo)
- `mysql` (MySQL)
- `graphql` (GraphQL)
- `apollo-server-core` (Apollo)
- `@nestjs/graphql` (Apollo)

Opt-in:

- [Prisma ORM](https://www.prisma.io/)
- [Express](https://expressjs.com/)
