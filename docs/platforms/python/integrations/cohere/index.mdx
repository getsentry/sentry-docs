This integration connects Sentry with the [Cohere Python SDK](https://github.com/cohere-ai/cohere-python).

Once you've installed this SDK, you can use Sentry AI Agents Monitoring, a Sentry dashboard that helps you understand what's going on with your AI requests.

Sentry AI Monitoring will automatically collect information about prompts, tools, tokens, and models. Learn more about the [AI Agents Dashboard](/ai/monitoring/agents).

<Alert level="info" title="V1 API Support">

The Cohere integration currently instruments the **V1 API** (`Client.chat`, `Client.chat_stream`, and `Client.embed`). The V2 API (`ClientV2`) is not yet supported.

</Alert>

## Install

Install `sentry-sdk` from PyPI:

```bash {tabTitle:pip}
pip install sentry-sdk
```

```bash {tabTitle:uv}
uv add sentry-sdk
```

## Configure

If you have the `cohere` package in your dependencies, the Cohere integration will be enabled automatically when you initialize the Sentry SDK.

## Verify

Verify that the integration works by making a chat request to Cohere.

```python
import sentry_sdk
import cohere

sentry_sdk.init(...)  # same as above

client = cohere.Client(api_key="(your Cohere key)")

def my_llm_stuff():
    with sentry_sdk.start_transaction(name="The result of the AI inference"):
        response = client.chat(
            model="command-a-03-2025",
            message="say hello",
        )
        print(response.text)
```

After running this script, the resulting data should show up in the `AI Spans` tab on the `Explore > Traces > Trace` page on Sentry.io.

If you manually created an <PlatformLink to="/tracing/instrumentation/custom-instrumentation/ai-agents-module/#invoke-agent-span">Invoke Agent Span</PlatformLink> (not done in the example above) the data will also show up in the [AI Agents Dashboard](/ai/monitoring/agents).

It may take a couple of moments for the data to appear in [sentry.io](https://sentry.io).

## Behavior

- The Cohere integration will connect Sentry with all supported Cohere methods automatically.

- The supported methods are `chat` (non-streaming), `chat_stream` (streaming), and `embed`.

- All exceptions raised during Cohere API calls are reported.

- Sentry considers LLM inputs/outputs as PII (Personally identifiable information) and doesn't include PII data by default. If you want to include the data, set `send_default_pii=True` in the `sentry_sdk.init()` call. To explicitly exclude prompts and outputs despite `send_default_pii=True`, configure the integration with `include_prompts=False` as shown in the [Options section](#options) below.

## Options

By adding `CohereIntegration` to your `sentry_sdk.init()` call explicitly, you can set options for `CohereIntegration` to change its behavior:

```python
import sentry_sdk
from sentry_sdk.integrations.cohere import CohereIntegration

sentry_sdk.init(
    # ...
    # Add data like inputs and responses;
    # see https://docs.sentry.io/platforms/python/data-management/data-collected/ for more info
    send_default_pii=True,
    integrations=[
        CohereIntegration(
            include_prompts=False,  # LLM inputs/outputs will be not sent to Sentry, despite send_default_pii=True
        ),
    ],
)
```

You can pass the following keyword arguments to `CohereIntegration()`:

- `include_prompts`:

  Whether LLM inputs and outputs should be sent to Sentry. Sentry considers this data personal identifiable data (PII) by default. If you want to include the data, set `send_default_pii=True` in the `sentry_sdk.init()` call. To explicitly exclude prompts and outputs despite `send_default_pii=True`, configure the integration with `include_prompts=False`.

  The default is `True`.

## Supported Versions

- Cohere: 5.4.0+
- Python: 3.9+
