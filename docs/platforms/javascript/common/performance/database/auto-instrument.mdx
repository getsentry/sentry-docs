Node.js integrations support tracking database queries as spans. `@sentry/node`
will auto-detect supported database drivers or ORMs being used in your project.

<SignInNote />

Supported packages and their integration name:

- `mysql`
- `mysql2`
- `pg` (Postgres)
- `graphql` (including Apollo Server)
- `mongo`
- `mongoose`
- `ioredis`
