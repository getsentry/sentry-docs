To integrate Sentry logs in your NestJS application, follow the general [logs setup guide](/platforms/javascript/guides/nestjs/logs/).

## Using the Default NestJS Logger

The default NestJS logger uses `console.log` under the hood. If you're using the default logger, you can use the [`consoleIntegration`](/platforms/javascript/guides/nestjs/configuration/integrations/console/) to capture console logs.

Note that this only works if you set the `forceConsole` option in the logger:

```typescript
const app = await NestFactory.create(AppModule, {
  logger: new ConsoleLogger({
    forceConsole: true,
  })
});
```
