### Install the Sentry SDK

Run the command for your preferred package manager to add the Sentry SDK to your application:

```bash {tabTitle:npm}
npm install @sentry/remix --save
```

```bash {tabTitle:yarn}
yarn add @sentry/remix
```

```bash {tabTitle:pnpm}
pnpm add @sentry/remix
```
