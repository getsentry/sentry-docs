## Enable Tracing

Profiling depends on Sentryâ€™s Tracing product being enabled beforehand. To enable tracing in the SDK:

```javascript
Sentry.init({
  dsn: "___DSN___",
  tracesSampleRate: 1.0,
});
```

Check out the <PlatformLink to="/performance/">tracing setup documentation</PlatformLink> for more detailed information on how to configure sampling. Setting the sample rate to 1.0 means all transactions will be captured.

## Enable Profiling

<Note>

Profiling for React Native is available in beta in SDK versions `5.8.0` and above.

</Note>

To enable profiling, set the `profilesSampleRate`:

```javascript
Sentry.init({
  dsn: "___DSN___",
  tracesSampleRate: 1.0,
  _experiments: {
    // profilesSampleRate is relative to tracesSampleRate.
    // Here, we'll capture profiles for 100% of transactions.
    profilesSampleRate: 1.0,
  },
});
```
