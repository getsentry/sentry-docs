---
title: Sentry Android Gradle Plugin
description: "Learn how to configure the Sentry Android Gradle Plugin to automatically upload source maps and debug symbols for your React Native project."
---

To enable the plugin add the `enableAndroidGradlePlugin` in the `experimental_android` section of the `@sentry/react-native/expo` plugin.

```javascript {tabTitle:app.json}
"plugins": [
  [
    "@sentry/react-native/expo",
    {
      "experimental_android": {
        "enableAndroidGradlePlugin": true,
      }
    }
  ]
```

## Prerequisities

The supported [Sentry Android Gradle Plugin options](/platforms/android/configuration/gradle/) are the following:
- `autoUploadProguardMapping`
- `includeProguardMapping`
- `dexguardEnabled`
- `uploadNativeSymbols`
- `autoUploadNativeSymbols`
- `includeNativeSources`
- `includeSourceContext`

and should be added in the `experimental_android` section like:

```javascript {tabTitle:app.json}
"experimental_android": {
  "enableAndroidGradlePlugin": true,
  "autoUploadProguardMapping": true,
  "uploadNativeSymbols": true,
  "includeNativeSources": true,
  "includeSourceContext": true,
}
```
