This integration is maintained and supported by the company that created it. For more details, or questions, feel free to contact support@truto.one.

## Install and Configure

<Note>

Sentry Owner, Manager, or Admin permissions are required to install this integration.

</Note>

1. Navigate to **Settings > Integrations > Truto**

2. Follow the full [Truto installation instructions](https://truto.one/docs/integration-guides/sentry#sentry).
