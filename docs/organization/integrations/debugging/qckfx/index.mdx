The Qckfx integration connects Sentry with an AI bug investigator that uses Sentry errors and Issues as context to help root cause and assign bug reports from Slack.

This integration is maintained and supported by Qckfx. For more details, questions, or support feel free to contact chris.wood@qckfx.com.
## Install and Configure

<Alert>

Sentry owner, manager, or admin permissions are required to install this integration.

</Alert>

1. Navigate to **Settings > Integrations > Qckfx**

2. Follow the full [Qckfx installation instructions](https://qckfx.com/).
