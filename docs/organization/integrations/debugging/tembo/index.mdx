Tembo: Background agents that plan, code, and review

This is the official Tembo integration for Sentry that allows Tembo to monitor all your sentry errors and submit PRs to the correct repositories for error fixes. For more details, questions, or support feel free to contact support@tembo.io.

## Install and Configure

<Alert>

Sentry Owner, Manager, or Admin permissions are required to install this integration.

</Alert>

1. Navigate to **Settings > Integrations > Tembo**

2. Follow the full [Tembo installation instructions](https://docs.tembo.io/integrations/sentry).
