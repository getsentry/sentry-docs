The Makeplane (Plane <> Sentry) integration enables automatic creation of Plane issues from Sentry alerts, supports linking Sentry issues to Plane issues, and keeps their status in sync for streamlined error tracking and resolution. This integration is maintained and supported by the company that created it. For more details, or questions, please feel free to reach out to support@plane.so!

## Install and Configure

<Alert>

Sentry owner, manager, or admin permissions are required to install this integration.

</Alert>

1. Navigate to **Settings > Integrations > Makeplane**

2. Follow the full [Makeplane installation instructions](https://docs.plane.so/integrations/sentry).
