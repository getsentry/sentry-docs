---
title: Relay
description: Un servicio para filtrar, aplicar límites de tasa y procesar grandes volúmenes de eventos de Sentry.
---

La ingesta la realiza [Sentry Relay](https://github.com/getsentry/relay), un servicio de filtrado, limitación de tasa y procesamiento de eventos.

Puede actuar como:

* el endpoint de almacenamiento de tu instalación de Sentry. Consulta la [documentación para desarrolladores de Relay](https://getsentry.github.io/relay/relay/).
* una capa intermedia adicional entre tu aplicación y Sentry. Consulta la [documentación del producto Relay](https://docs.sentry.io/product/relay/).
