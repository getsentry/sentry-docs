Este documento define el formato Sample que usa Sentry para incorporar perfiles. Está dirigido a desarrolladores del SDK de Sentry y a quienes mantienen la canalización de ingesta de perfiles.

Este formato está pensado para ser multiplataforma y permitir representar muestras
recopiladas por un generador de perfiles.
