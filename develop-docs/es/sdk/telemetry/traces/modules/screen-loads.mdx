---
title: Módulo de carga de pantallas
---

El módulo de carga de pantallas registra métricas a partir de spans en transacciones `ui.load` utilizando operaciones de span específicas.

<div id="transaction-conventions">
  # Convenciones de transacciones
</div>

<div id="transaction-measurements">
  ## Mediciones de la transacción
</div>

| Clave | Unidad | Descripción | Condiciones |
|:--|:--|:--|:--|
| `time_to_initial_display` | milisegundos | Tiempo hasta la primera visualización | requerido |
| `time_to_full_display` | milisegundos | Tiempo hasta la visualización completa de la pantalla | opcional, requiere instrumentación manual |

<div id="span-conventions">
  # Convenciones de span
</div>

<div id="span-operations">
  ## Operaciones de span
</div>

| Operación de span | Descripción |
|:--|:--|
| `db.sql.query` | Una consulta a la base de datos |
| `db.sql.room` | Una consulta a la base de datos con la biblioteca Room en Android |
| `db.sql.transaction` | - |
| `db` | Una operación en la base de datos |
| `file.read` | Lectura de un archivo del sistema de archivos |
| `file.write` | Escritura de un archivo en el sistema de archivos |
| `http.client` | Una solicitud de red saliente |
| `ui.load` | Una operación en una interfaz móvil |
| `ui.load.full_display` | Cuando la pantalla ha terminado de cargarse por completo |
| `ui.load.initial_display` | Cuando la pantalla ha renderizado su primer fotograma |
