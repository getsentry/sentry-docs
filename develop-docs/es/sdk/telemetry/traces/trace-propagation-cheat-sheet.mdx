<div id="faq">
  ## Preguntas frecuentes
</div>

**¿Por qué a veces hacemos muestreo incluso si `traces_sample_rate` es 0? ¿Por qué a veces no muestreamos si `traces_sample_rate` es 1?**

Un `traces_sample_rate` entre 0 y 1 solo se tiene en cuenta si no hay una traza entrante y el SDK debe tomar su propia decisión de muestreo. Si hay una traza entrante, la decisión de muestreo del elemento padre siempre tiene prioridad sobre `traces_sample_rate`.
