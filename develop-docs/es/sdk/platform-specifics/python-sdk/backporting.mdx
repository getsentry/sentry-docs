---
title: Backport
description: Cómo hacer backport a una versión principal anterior.
sidebar_order: 10
---

La última versión normal 1.x fue la [1.45.0](https://github.com/getsentry/sentry-python/releases/tag/1.45.0) en abril de 2024. El desarrollo de 1.x se ha detenido: no estamos haciendo backport de nuevas funciones ni de correcciones de errores, salvo las de seguridad.

Si hay un problema de seguridad en el SDK lo suficientemente grave (usa tu criterio) como para requerir un backport a 1.x (consulta, por ejemplo, la [1.45.1](https://github.com/getsentry/sentry-python/releases/tag/1.45.1)):

- Cambia a la rama `1.x`
- Porta la corrección a la rama `1.x`
- Lanza una versión. Asegúrate de configurar tanto “Use workflow from” como “Target branch to merge into” en `1.x`
- Continúa con el proceso de lanzamiento como de costumbre
