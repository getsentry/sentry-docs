Esta sección contiene información específica del SDK de JavaScript que resulta útil para desarrollar nuevos SDK de JS o ampliar los existentes, así como para que los equipos de ingesta y de producto comprendan cómo funciona el SDK de JS y qué datos envía.

Si quieres contribuir nuevas funciones al SDK o deseas entender qué datos puedes consumir del SDK de JS, este es el lugar indicado para empezar.
El equipo del SDK de JS fomenta las contribuciones de la comunidad o entre equipos al SDK y estará encantado de ayudarte a empezar.

Comunícate con el equipo del SDK (en [GitHub](https://github.com/getsentry/sentry-javascript) o internamente en Slack `#discuss-javascript`) si tienes preguntas, inquietudes o planeas desarrollar una funcionalidad más amplia.

<Alert level="warning">

Este documento **no** reemplaza las especificaciones generales del SDK, como la Unified API o la Guía de desarrollo de SDK.
Tampoco explica las implementaciones en detalle ni enlaza al código del SDK.
Es un complemento de otros documentos y contiene información específica del SDK de JavaScript.

</Alert>
