`timestamp`

: *Obligatorio*. Una marca de tiempo que indica cuándo terminó la medición. El
formato puede ser una cadena según la [RFC
3339](https://tools.ietf.org/html/rfc3339) o un valor numérico (entero o flotante)
que representa el número de segundos transcurridos desde la [época de Unix](https://en.wikipedia.org/wiki/Unix_time).

```json
  {
    "timestamp": "2011-05-02T17:41:36.955Z"
  }
```

o:

```json
  {
    "timestamp": 1304358096.955
  }
```
