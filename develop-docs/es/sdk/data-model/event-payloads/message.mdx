---
title: Interfaz de mensajes
sidebar_order: 3
---

La interfaz de mensajes contiene un mensaje de registro que describe un evento o error.
Opcionalmente, puede incluir una cadena de formato y parámetros estructurados. Esto puede
ayudar a agrupar mensajes similares en el mismo issue.

<div id="attributes">
  ## Atributos
</div>

`formatted`

: **Obligatorio**. El mensaje completamente formateado. Si falta, Sentry intentará
  interpolar el mensaje.

En Relay se aplica un límite de 8192 caracteres y los mensajes más largos se truncarán. 
  Los SDK no deberían imponer este límite.
  Sentry también acepta un mensaje sin este valor para admitir SDK heredados.

`message`

: _Opcional_. La cadena de mensaje sin interpolación.

En Relay se aplica un límite de 8192 caracteres y los mensajes más largos se truncarán.
  Los SDK no deberían imponer este límite.

`params`

: _Opcional_. Una lista de parámetros de formato, preferentemente cadenas. Los valores que no sean cadenas
  se convertirán a cadenas.

<div id="examples">
  ## Ejemplos
</div>

```json
{
  "logentry": {
    "message": "Mi mensaje en bruto con cadenas interpretadas como %s",
    "params": ["esto"]
  }
}
```
