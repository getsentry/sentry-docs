<div id="create-a-gitlab-app">
  ## Crea una aplicación de GitLab
</div>

Para usar la integración de GitLab, debes crear una aplicación OAuth en GitLab. Para empezar, visita [Applications](https://gitlab.com/oauth/applications/).

Al configurar la aplicación, usa los siguientes valores:

| Setting                         | Value                                         |
| ------------------------------- | --------------------------------------------- |
| Name                            | Sentry                                        |
| Redirect URI                    | `{YOUR_DOMAIN}/extensions/gitlab/setup/`      |
| Scopes                          | api                                           |

Asegúrate de marcar "Confidential".

Toma nota del ID y del secreto de tu aplicación, ya que los necesitarás en el siguiente paso.

Sigue nuestra [documentación sobre cómo instalar y configurar la integración de GitLab](https://docs.sentry.io/workflow/integrations/gitlab/) para finalizar la instalación y usar la integración.
