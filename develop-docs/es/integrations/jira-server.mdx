Sigue nuestra [documentación para configurar la integración con Jira Server](https://docs.sentry.io/product/integrations/issue-tracking/jira/#jira-server) para preparar e instalar la integración, reemplazando cualquier mención de sentry.io por tu dominio. Por ejemplo, la URL de la aplicación será `{YOUR_DOMAIN}/extensions/jira-server/setup/` en lugar de `https://sentry.io/extensions/jira-server/setup/`.
