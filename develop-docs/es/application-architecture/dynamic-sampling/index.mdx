---
title: Muestreo dinámico
description: >-
  El muestreo dinámico es una función del pipeline de ingesta que permite a Sentry
  ajustar automáticamente la cantidad de datos conservados según el valor de los
  datos.
sidebar_order: 50
og_image: /og-images/application-architecture-dynamic-sampling.png
---

De todos los datos recibidos por los SDK, Sentry puede extraer información de baja granularidad mediante métricas, mientras que el muestreo dinámico decide si conservar o descartar datos.

Los objetivos del muestreo dinámico son:

- **Valor a cualquier escala**: Sentry tiene una visión completa de tu aplicación y puede mostrar con precisión métricas de rendimiento, detectar tendencias y problemas de rendimiento a partir del 100% de tus eventos.
- **Valor inmediato desde el primer momento**: Sentry ajusta automáticamente la retención de datos para almacenar muestras suficientes de los eventos de mayor valor con todo detalle.
- **Priorizar según tus necesidades**: Sentry ofrece un mecanismo para expresar prioridades como un conjunto de reglas y luego ajusta la retención de datos todo lo posible para cumplirlas.
