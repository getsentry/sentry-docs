---
title: Resolución de problemas de Redis
sidebar_title: Redis
sidebar_order: 5
---

En la configuración autohospedada, Redis se utiliza tanto como almacén de datos transaccional como como cola de trabajo de Celery. Por ello, puede verse sobrecargado durante picos de eventos. Hemos introducido mejoras importantes al respecto a partir de la versión `20.10.1`. Si aún tienes problemas, puedes considerar escalar Redis en sí o cambiar a un broker de Celery diferente, como [RabbitMQ](https://www.rabbitmq.com/).
