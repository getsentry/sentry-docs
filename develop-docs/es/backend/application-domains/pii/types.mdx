---
title: Tipos de reglas de PII
---

`pattern`

: Expresión regular personalizada al estilo Perl (PCRE).

```json
  {
    "rules": {
      "hash_device_id": {
        "type": "pattern",
        "pattern": "d/[a-f0-9]{12}",
        "redaction": {
          "method": "hash"
        }
      }
    },
    "applications": {
      "$string": ["hash_device_id"]
    }
  }
```

`imei`

: Coincide con un IMEI o un IMEISV.

```json
  {
    "rules": {
      "hash_imei": {
        "type": "imei",
        "redaction": {
          "method": "hash"
        }
      }
    },
    "applications": {
      "$string": ["hash_imei"]
    }
  }
```


`anything`

: Coincide con cualquier valor. Es básicamente equivalente a una expresión regular con comodín.

Por ejemplo, para eliminar todas las cadenas:

```json
  {
    "rules": {
      "remove_everything": {
        "type": "anything",
        "redaction": {
          "method": "remove"
        }
      }
    },
    "applications": {
      "$string": ["remove_everything"]
    }
  }
```

`multiple`

: Combina varias reglas en una sola. Esto es una disyunción (OR): el campo en
cuestión solo tiene que cumplir una de las reglas para cumplir la regla combinada, no
todas.

```javascript
  {
    "rules": {
      "remove_ips_and_macs": {
        "type": "multiple",
        "rules": [
          "@ip",
          "@mac"
        ],
        "hide_rule": false,  // Oculta las reglas internas al mostrar qué reglas se aplicaron. El valor predeterminado es false.
        "redaction": {
          "method": "remove"
        }
      }
    },
    "applications": {
      "$string": ["remove_ips_and_macs"]
    }
  }
```

`alias`

: Asigna un alias de una regla a otra. Es equivalente a `multiple`, salvo que solo puedes envolver una única regla.

```javascript
  {
    "rules": {
      "remove_ips": {
        "type": "multiple",
        "rule": "@ip",
        "hide_rule": false,  // Oculta la regla interna al mostrar las reglas aplicadas. El valor predeterminado es false.
        "redaction": {
          "method": "remove"
        }
      }
    },
    "applications": {
      "$string": ["remove_ips"]
    }
  }
```
