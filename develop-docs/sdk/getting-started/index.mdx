## Tips

- **Consult existing SDKs.** When in doubt, refer to existing Sentry SDKs for implementation patterns.

- **Join us on Discord.** Reach out to our SDK maintainers and open source contributors on the [Sentry Discord server](https://discord.gg/sentry).
