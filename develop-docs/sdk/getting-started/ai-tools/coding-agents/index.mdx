All of the listed tools are available for every engineer working at Sentry. For general information on AI assisted development at Sentry, see [AI-Assisted Development](/engineering-practices/ai-assisted-development/). For an up-to-date list of approved AI tools, see [Sentry AI Usage Policy](https://www.notion.so/sentry/Sentry-AI-Usage-Policy-16f8b10e4b5d801d862ce27b0951333b).

## Coding Agents

These tools are designed for hands-on coding work — writing, editing, and reviewing code directly in your codebase.

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
- [Cursor](https://www.cursor.com/)
- [GitHub Copilot](https://github.com/features/copilot)

## General-Purpose Assistants

These are conversational AI tools useful for non-coding related tasks and not intended for direct codebase manipulation.

- [Claude](https://claude.ai/)
- [Gemini](https://gemini.google.com/)
