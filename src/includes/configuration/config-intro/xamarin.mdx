Options can be set by passing a callback to the `Init()` method which will
pass the option object along for modifications:

```csharp
using Sentry;

SentryXamarin.Init(o =>
{
    o.Dsn = "___PUBLIC_DSN___";
	o.AddXamarinFormsIntegration();
    o.MaxBreadcrumbs = 50;
    o.Debug = true;
});
    // app code here
```
