A `Func<SentryEvent, SentryEvent>` can be used to mutate, discard (return null), or return a completely new event.

```csharp
using Sentry;

SentryXamarin.Init(o =>
{
	o.AddXamarinFormsIntegration();
    o.BeforeSend = sentryEvent =>
    {
        // Modify the event here:
        sentryEvent.UnsetTag("company.id"); // Unset the tag called 'company.id'.
        return sentryEvent;
    };
});
```
