 `Func<SentryEvent, SentryEvent>` can be used to mutate, discard (return null), or return a completely new event.

```csharp
using Sentry;

// Add this to the SDK initialization callback
options.BeforeSend = sentryEvent =>
{
    if (sentryEvent.Exception != null
      && sentryEvent.Exception.Message.Contains("Noisy Exception"))
    {
        return null; // Don't send this event to Sentry
    }

    sentryEvent.ServerName = null; // Never send Server Name to Sentry
    return sentryEvent;
};
```
