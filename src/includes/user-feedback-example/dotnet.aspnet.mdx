Make sure you've got the JavaScript SDK available:

```html
<script
  src="https://browser.sentry-cdn.com/{{ packages.version('sentry.javascript.browser') }}/bundle.min.js"
  integrity="sha384-{{ packages.checksum('sentry.javascript.browser', 'bundle.min.js', 'sha384-base64') }}"
  crossorigin="anonymous"
></script>
```

If you are rendering the page from the server, for example on ASP.NET MVC, the `Error.cshtml` razor page can be:

```cshtml
@if (SentrySdk.LastEventId != Sentry.Protocol.SentryId.Empty) {
    <script>
    Sentry.init({ dsn: "___PUBLIC_KEY___" });
    Sentry.showReportDialog({ eventId: "@SentrySdk.LastEventId" });
  </script>
}
```

In case you prefer to build your own form to collect user feedback, you can capture it via the .NET SDK, on the backend.
All you need is the user information you've collected, and the eventId which the feedback is related to.

```csharp {tabTitle:C#}
using Sentry;
var eventId = SentrySdk.CaptureMessage("An event with a user feedback.");
SentrySdk.CaptureUserFeedback(eventId,
  "user@example.com",
  "It broke.",
  "The User");
```

```fsharp {tabTitle:F#}
open Sentry
let eventId = SentrySdk.CaptureMessage("An event with a user feedback.")
let userFeedback = UserFeedback(eventId, "user@example.com", "It broke.", "The User")
SentrySdk.CaptureUserFeedback(userFeedback)
```
