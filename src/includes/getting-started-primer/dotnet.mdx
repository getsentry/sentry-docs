The main [Sentry NuGet package](https://www.nuget.org/packages/Sentry) targets .NET Standard 2.0. Read more about [compatibility and unit testing](./unit-testing.mdx).

**Features:**


- We provide code samples in our documentation for .NET apps and frameworks in both C# and F#.
- Samples provided for [C#](https://github.com/getsentry/sentry-dotnet/tree/main/samples), [F# Console](https://github.com/sentry-demos/fsharp), and[F# Giraffe](https://github.com/sentry-demos/giraffe)
