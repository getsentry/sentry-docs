The `sentry-android-okhttp` library provides [OkHttp](https://github.com/square/okhttp) support for Sentry via the [OkHttp Interceptor](https://github.com/square/okhttp/blob/a2059dedc0b1d4a977480834ae4ed9ea576a3eb8/okhttp/src/main/kotlin/okhttp3/Interceptor.kt) that sends spans and breadcrumbs to Sentry. The source can be found [on GitHub](https://github.com/getsentry/sentry-java/tree/main/sentry-android-okhttp/src/main/java/io/sentry/android/okhttp).
