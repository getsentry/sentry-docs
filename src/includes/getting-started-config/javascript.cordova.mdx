You should `init` the SDK in the `deviceReady` function, to make sure the native integrations runs. For more details about Cordova <Link to="/platforms/javascript/cordova/">click here</Link>

```javascript
onDeviceReady: function() {
  var Sentry = cordova.require("sentry-cordova.Sentry");
  Sentry.init({ dsn: '___PUBLIC_DSN___' });
}
```
