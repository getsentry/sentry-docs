Initialize the SDK within your `config/application.rb`:

```ruby {filename:config/application.rb}
Raven.configure do |config|
  config.dsn = '___PUBLIC_DSN___'
end
```

If you have added items to [Railsâ€™ log filtering](https://guides.rubyonrails.org/action_controller_overview.html#parameters-filtering), you can also make sure that those items are not sent to Sentry:

```ruby
# in your application.rb:
config.filter_parameters << :password

# in an initializer, like sentry.rb
Raven.configure do |config|
  config.dsn = '___PUBLIC_DSN___'

  config.sanitize_fields = Rails.application.config.filter_parameters.map(&:to_s)
end
```
