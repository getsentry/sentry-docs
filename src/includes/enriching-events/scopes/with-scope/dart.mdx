This feature is available as a Beta. Features available in Beta are still in-progress and may have bugs. We recognize the irony.

This is available in 5.1.0.beta.1 version of the Sentry Dart/Flutter SDK.

</Note>

```dart
import 'package:sentry/sentry.dart';

Sentry.captureMessage(
    'This event has an extra tag',
    withScope: (scope) {
        // will be tagged with my-tag="my value"
        scope.setTag('my-tag', 'my value');
    },
);

// will not be tagged with my-tag
Sentry.captureMessage('This event has no extra tag');
```
