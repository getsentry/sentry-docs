Install the package via the [Unity Package Manager](https://docs.unity3d.com/Manual/upm-ui.html) using a Git URL to Sentry's SDK repository:

```
https://github.com/getsentry/unity.git#{{ packages.version('sentry.dotnet.unity', '0.0.5') }}
```
<Note>

Confirm there is no trailing whitespace at the end of the URL. The Unity Package Manager will fail to find the package otherwise.

</Note>
