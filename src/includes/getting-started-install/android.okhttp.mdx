This is a preview API, available in `4.4.0-alpha.1`. Features available in a preview API are still in-progress and may have bugs. We recognize the irony. The API may be renamed, changed, or removed in a future version.

</Note>

To add the OkHttp integration, initialize the [Android SDK](/platforms/android/), then add the `sentry-android-okhttp` dependency. Using Gradle:

```groovy
implementation 'io.sentry:sentry-android:{{ packages.version('sentry.java.android', '4.4.0') }}'
implementation 'io.sentry:sentry-android-okhttp:{{ packages.version('sentry.java.android.okhttp', '4.4.0') }}'
```
