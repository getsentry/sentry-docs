<Note>

Supported in Sentry's Android SDK version `4.4.0` and above.

</Note>

To add the OkHttp integration, Initialize the <Link to="/platforms/android/">Android's SDK</Link> and Add the `sentry-android-okhttp` dependency. Using Gradle:

```groovy
implementation 'io.sentry:sentry-android:{{ packages.version('sentry.java.android', '4.4.0') }}'
implementation 'io.sentry:sentry-android-okhttp:{{ packages.version('sentry.java.android.okhttp', '4.4.0') }}'
```
