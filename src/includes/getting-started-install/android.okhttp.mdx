To add the OkHttp integration, Add the `sentry-android-okhttp` dependency. Using Gradle:

```groovy
implementation 'io.sentry:sentry-android:{{ packages.version('sentry.java.android', '4.2.0') }}'
implementation 'io.sentry:sentry-android-okhttp:{{ packages.version('sentry.java.android.okhttp', '4.4.0') }}'
```
