```csharp {tabTitle:C#}
using Sentry;

// Add this to the SDK initialization callback
options.Release = "my-project-name@2.3.12";
```

```fsharp {tabTitle:F#}
open Sentry

// Add this to the SDK initialization callback
options.Release <- "my-project-name@2.3.12"
```

The SDK will set the release by default from `Application.productName` and `Application.version`.
```csharp
options.Release = $"{Application.productName}@{Application.version}";
```
