Make sure you've got the JavaScript SDK available:

```html
<script
  src="https://browser.sentry-cdn.com/{{ packages.version('sentry.javascript.browser') }}/bundle.min.js"
  integrity="sha384-{{ packages.checksum('sentry.javascript.browser', 'bundle.min.js', 'sha384-base64') }}"
  crossorigin="anonymous"
></script>
```

If you are rendering the page from the server, for example on ASP.NET MVC, the `Error.cshtml` razor page can be:

```cshtml
@if (SentrySdk.LastEventId != Sentry.Protocol.SentryId.Empty) {
  <script>
    Sentry.init({ dsn: "___PUBLIC_KEY___" });
    Sentry.showReportDialog({ eventId: "@SentrySdk.LastEventId" });
  </script>
}
```
