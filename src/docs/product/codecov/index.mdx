Codecov, the leading provider of test code coverage for developers, is now part of the Sentry family. It makes coverage actionable for over two dozen test frameworks and helps improve testing, coverage, and code reliability. With Codecov, you'll be able to get test coverage insights in Sentry directly, as you monitor your errors. Read more about Codecov [here](https://docs.codecov.com/docs/about-code-coverage).
