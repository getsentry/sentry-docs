## Integration Platform

Sentryâ€™s Integration Platform provides a way for external services to interact with Sentry. Integrations utilizing this platform are first-class actors within Sentry, and you can build them for [public](/product/integrations/integration-platform/#public-integrations) as well as [internal](/product/integrations/integration-platform/#internal-integrations) use cases.
