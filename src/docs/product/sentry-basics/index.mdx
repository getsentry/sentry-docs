Welcome to Sentry Basics, our primer on using the web UI. While much of this is self-documented, it's useful to know about...

- [Environments](/product/sentry-basics/environments/): Learn more about how environments can help you better filter issues, releases, and user feedback in the Issue Details page of the web UI. 

- [Search](/product/sentry-basics/search/): Learn more about Search, which is available on several major Sentry views: Issues, Events, and Releases.

- [Best Practices](/product/sentry-basics/guides/): Learn more about our best practice guides and follow our tutorials that walk you through the flow of setting up and configuring your Sentry account to monitor errors in both a frontend and backend application.
