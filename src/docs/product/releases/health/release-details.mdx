The **Release Details** page focuses on an individual release. Elements of the release are presented in visual trends for crashes and sessions, specifics regarding each issue, adoption graphs, and commit author breakdowns.

![View of the release details page showing a release comparison graph, issues, adoption, and other details.](release_details.png)

## Overview
### Graphs
After selecting the time range (the default is the entire release period), the top chart provide insight into:

- [Crash free session](/product/releases/health/#crash-free-sessionsusers) rate
- [Healthy](/product/releases/health/#healthy), [abnormal](/product/releases/health/#abnormal), [errored](/product/releases/health/#errored) session rate
- [Crashed session](/product/releases/health/#crashed-users) rate
- [Failure rate](/product/performance/metrics/#failure-rate)
- [Session](/product/releases/health/#session) count
- Error count
- Transaction count

Percentage based metrics of this release are compared to average of all releases to easily spot the difference in trend.

[Errored](/product/releases/health/#errored) sessions most likely result in handled issues and [crashed](/product/releases/health/#crash) sessions in unhandled issues. Click on the handled/unhandled issues link to see them in [Issues](/product/issues/) page.

Click on any metric to display it in the top chart.

Click on any series in the chart legend to hide it (hide Healthy series to see Crashes in better scale).

Hover over specific dates and times in the overview graph for a quick perspective of events that crashed, behaved abnormally, errored, or are healthy, as defined for [Release Health](/product/releases/health/).

### Issues
Use Release Details to view issues associated with this release and to access them for triage. You can navigate to the [Issues](/product/issues/) page by clicking Open in Issues button located at the top of the issues table. The issues in the table are sorted by the number of events in the selected time range. Click the filter buttons to see:
- All Issues
- New Issues
- Unhandled Issues
- Resolved Issues 

### Sessions Adopted
See the release adoption percentage over time. This graph compares sessions on this release with session on all releases. You can change main Date Range to see adoption in the different timeframe.

### Commit Author Breakdown
See the list of people that authored commits in this release together with the absolute and relative counts of commits per person. [Learn how to set up commits](/product/releases/suspect-commits/).

### Deploys
See when and to what environments this release has been deployed to. Click on the deploy badge to see issues from that environment. [Learn how to set up deploys](/product/releases/setup/#notify-sentry).

### Other Projects in This Release
Releases are global per organization. See here the list of projects with the same release version. Click on project to go to [Project Details](/product/projects/project-details/), click on View to see details of that project's release.
