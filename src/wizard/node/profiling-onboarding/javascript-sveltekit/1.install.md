#### Install

The Profiling integration requires the `@sentry/sveltekit` SDK (minimum version 7.47.0).

```bash
# Using yarn
yarn add @sentry/sveltekit @sentry/profiling-node

# Using npm
npm install --save @sentry/sveltekit @sentry/profiling-node
```
