If you are using `yarn` or `npm` you can add our package as a dependency:

```bash
# Using yarn
$ yarn add @sentry/browser@{% sdk_version sentry.javascript.browser %}

# Using npm
$ npm install @sentry/browser@{% sdk_version sentry.javascript.browser %}
```
