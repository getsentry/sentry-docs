On your client-side SvelteKit app, add:

```javascript
import * as Sentry from "@sentry/sveltekit";

Sentry.init({
  dsn: "___PUBLIC_DSN___",

  integrations: [
    new Sentry.Feedback({
      // Additional SDK configuration goes in here, for example:
      colorScheme: "light",
    }),
  ],
});
```
