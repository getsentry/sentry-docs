USentrySubsystem* SentrySubsystem = ...;

The same result can be achieved by calling corresponding function in blueprint:

![Set tag](unreal_set_tag.png)

Alternatively, this configuration cab be provided to the crash reporter [during initialization](/platforms/unreal/setup-crashreporter/#configure-attributes).

Commonly used values like `Engine version`, `Configuration`, `Target Type`, etc. can be promoted to tags by using the Sentry configuration window. Navigate to the editor's menu **Project Settings > Plugins > Sentry > Misc > Promote values to tags** and select which values should be automatically added to captured events as tags.
