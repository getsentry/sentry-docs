## Configuration

All of Sentry's tracing-related integrations (`BrowserTracing`, `Http`, and `Express`), as well as the [Next.JS SDK](/platforms/javascript/guides/nextjs/), support distributed tracing out of the box.

<Alert title="Note" level="note">

**You need to have "Performance Monitoring" enabled!** See <PlatformLink to="/performance/">Set Up Performance</PlatformLink> on how to enable performance monitoring.

</Alert>
