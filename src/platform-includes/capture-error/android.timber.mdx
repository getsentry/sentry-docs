The following example adds a breadcrumb and captures an error using Timberâ€™s static API; this breadcrumb and error will be forwarded to Sentry.

```java
int division(int x, int y) {
  Timber.i("x = " + x + ", y = " + y)

  try {
    return x / y
  } catch (ArithmeticException e) {
    Timber.e(e, "Division error")
  }
  return 0
}
```

```kotlin
fun division(x: Int = 5, y: Int = 0): Int {
  Timber.i("x = $x, y = $y")

  try {
    return x / y
  } catch (e: ArithmeticException) {
    Timber.e(e, "Division error")
  }
  return 0
}
```
