According to the [compatibility table](https://docs.microsoft.com/en-us/dotnet/standard/net-standard), the package is compatible with the following versions or newer:

- .NET Framework 4.6.1
- .NET Core 2.0
- Mono 5.4
- Xamarin.Android 8.0
- Xamarin.Mac 3.8
- Xamarin.iOS 10.14
- Universal Windows Platform 10.0.16299

<Note>

If you're using an older version of .NET Framework or Mono, [our legacy SDK](/clients/csharp/) supports .NET Framework as early as 3.5.

</Note>

## Unit Testing

Sentry runs a validation test suite for SDK functionality in the following platforms:

- .NET 5 - Windows, macOS, and Linux
- .NET Core 3.1 - Windows, macOS, and Linux
- .NET Core 2.1 - Windows, macOS, and Linux
- .NET Framework 4.8 - Windows
- Mono 5.12 - macOS and Linux
